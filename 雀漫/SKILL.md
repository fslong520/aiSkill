# Que Man - 雀漫动态漫画分镜生成器

## 核心规格（Seedance 2.0 标准）

**时长要求**：
- 每集固定 15 秒
- 误差范围：±0.5 秒

**镜头结构**：
- 6-8 个分镜头
- 每个镜头 1.5-3 秒
- 首帧 + 尾帧必须明确标注

**角色卡格式**：
- 纯文本输出
- 禁止使用任何 markdown 符号（#、**、*、- 等）
- 避免 AI 理解错误

**输出兼容性**：
- 直接可用于 Seedance 2.0 视频生成
- 无额外格式转换
- 时间戳精确到 0.1 秒

**自动工作流**：
1. 查找章节.md 文件并读取内容
2. 分析文件中的剧情内容，识别已有的分镜脚本部分
3. 从剩余未转换的原始剧情文本中选取前 15 秒容量的剧情片段（约 1-2 个关键事件）
4. 将选定的剧情转换为符合 Seedance 2.0 标准的分镜 prompt 格式
5. 将生成的分镜脚本写入章节.md 文件中，替换掉对应的原始剧情文本部分
6. 确保已转换的部分不会在下次运行时被重复转换
7. 自动生成角色卡插图prompt（写实风格，用于图片姬技能）

## 快速开始

### 第一步：剧情容量分析（新增核心功能）

当用户提供长章节内容时，必须先进行剧情分析：

分析流程：
1. 通读全文，识别核心剧情节点（开场/发展/高潮/结尾）
2. 评估 15 秒容量（仅能承载 1-2 个关键事件）
3. 选择最佳切片位置（优先选择：冲突爆发点/情感转折点/悬念揭示点）
4. 主动告知用户截取范围

输出格式：
```
【剧情分析与截取建议】

您提供的章节共 XXXX 字，包含以下核心剧情节点：
1. 开场：XXX（约 XXX 字）
2. 发展：XXX（约 XXX 字）
3. 高潮：XXX（约 XXX 字）
4. 结尾：XXX（约 XXX 字）

15 秒动态漫画剧情容量评估：
- 可容纳：1-2 个关键事件 + 1 次情绪转折
- 建议聚焦：第 X 节点的 [具体情节段落]
- 涵盖内容：[具体事件/对话/转折]
- 时间跨度：从"XXX"（原文引用）到"XXX"（原文引用）
- 衔接提示：本集结束后，下一集可从 [后续情节] 继续

是否按此方案生成分镜？如有其他偏好请告知。
```

示例：

**自动化处理规则**：

- 保持剧情连贯性，严格按照原文顺序推进

## 旁白与台词严格区分（动态漫画核心规范）

⚠️ 重要提示：动态漫画与普通视频生成的关键区别在于——更依赖旁白推动叙事！

### 标注格式规范

【旁白】标签使用场景：
1. 场景描述：交代时间、地点、环境变化
   示例：【旁白】："黄昏时分，天台上风很大。" #2.0 秒

2. 心理活动：揭示角色内心想法
   示例：【旁白】："她心中一紧，仿佛预感到了什么。" #1.5 秒

3. 剧情推进：在对话间隙补充信息
   示例：【旁白】："这一别，竟是永别。" #1.0 秒

4. 情感渲染：增强情绪感染力
   示例：【旁白】："眼泪滑过的痕迹，成了今夜最深的印记。" #2.0 秒

5. 过渡衔接：连接不同镜头或场景
   示例：【旁白】："与此同时，城市的另一端……" #1.5 秒

【配音（角色名）】标签使用场景：
1. 角色对话：人物之间的交流
   示例：【配音（张三）】："等我回来。" #1.0 秒（青年音，低沉温柔）

2. 独白/自言自语：角色说出声的想法
   示例：【配音（李四）】："为什么……为什么会这样……" #2.0 秒（颤抖，哽咽）

3. 呼喊/惊呼：情绪爆发时的声音
   示例：【配音（王五）】："小心——！" #0.8 秒（惊呼，带着恐惧）

### 音色描述规范

旁白音色要求：
- 必须标注语气特征（低沉/轻柔/哀伤/无奈等）
- 必须标注特殊效果（混响/渐弱/叹息感等）
- 避免过于具体的年龄限制，保持灵活性
- 示例：深沉男声，带混响，渐弱 / 语气轻柔，带叹息感

角色配音音色要求：
- 必须标注声线类型（青年音/少女音/大叔音等）
- 必须标注情绪状态（颤抖/哽咽/怒吼/轻蔑等）
- 可根据角色性格添加特征描述（磁性/沙哑/清甜等）
- 示例：青年音，低沉温柔 / 少女音，清甜中带哽咽

### 比例分配原则

15 秒时长中的建议分配：

方案 A（旁白主导型，适合剧情推进）：
- 旁白：6-8 秒（占比 40-53%）
- 角色台词：4-6 秒（占比 27-40%）
- 无台词纯画面：3-5 秒（占比 20-33%）

方案 B（对话驱动型，适合情感对峙）：
- 旁白：3-5 秒（占比 20-33%）
- 角色台词：7-9 秒（占比 47-60%）
- 无台词纯画面：2-4 秒（占比 13-27%）

方案 C（平衡型，最常用）：
- 旁白：5-7 秒（占比 33-47%）
- 角色台词：5-7 秒（占比 33-47%）
- 无台词纯画面：2-4 秒（占比 13-27%）

⚠️ 严禁情况：
- 全程无旁白（导致剧情理解困难）
- 旁白过少（<2 秒，无法完整叙述）
- 旁白过多（>10 秒，挤压角色表现空间）
- 依赖角色对话传达全部信息（导致节奏拖沓）

### 实战示例对比

❌ 错误示范（依赖对话，缺少旁白）：
```
[镜头 1]
配音（张三）："我走了。"
配音（李四）："不走了行不行？"
配音（张三）："不行。"
配音（李四）："那我跟你一起走。"
配音（张三）："你不能走。"
```
问题：完全靠对话堆砌，缺乏情感渲染和剧情补充，观众难以理解背景。

✅ 正确示范（旁白 + 对话平衡）：
```
[镜头 1]
【旁白】："那是他们最后一次见面，谁也没有想到，竟成永别。" #2.5 秒（低沉，带混响）
【配音（张三）】："我走了。" #1.0 秒（平静，但眼神复杂）
【旁白】："他的声音很轻，却像一把刀，割断了所有挽留的可能。" #2.0 秒（语气无奈）

[镜头 2]
【配音（李四）】："不走了行不行？" #1.5 秒（颤抖，带着哭腔）
【旁白】："她的声音里，藏着最后的乞求。" #1.5 秒（轻声叹息）
```
优势：旁白补充了背景信息和情感层次，对话更加精炼有力。

## 微动作库（Seedance 2.0 兼容）

嘴部动作（配合台词）：

**拉远（Pull Out）**：
```
用途：孤独感、结束感、揭示环境

**横移（Truck / Horizontal Move）**：
```
用途：跟随角色移动、平行叙事、展示环境全貌

   示例：从 A 角色表情平移到 B 角色手中的武器
   
3. 环境平移：缓慢扫过场景，展示环境细节
   示例：从教室前排平移到后排，展示每个学生反应

## 工作流程（15 秒适配）

完整流程：

1. 小说阅读 → 提取 15 秒切片
   选择最适合的高潮/转折/悬念片段

2. 角色分析 → 创建纯文本设定卡
   仅保留核心特征（外貌/性格/锚点）

3. 场景拆分 → 确定 6-8 个镜头
   按情绪曲线分配时长（1.5-3 秒/镜）

4. 微动作设计 → 匹配台词与情绪
   头发/衣服飘动强度，口型同步

5. 镜头规划 → 选择景别与运动
   首帧钩子 + 尾帧悬念，中间推进

6. 配音标注 → 语气/口型/时长
   精确到 0.1 秒，预留气口

7. 音效配乐 → 层次搭配
   环境/动作/特效/情绪四层叠加

8. 输出分镜 → 纯文本格式
   无 markdown 符号，Seedance 2.0 直出

时间分配（总计约 1 小时/集）：
文本分析：15 分钟
分镜撰写：30 分钟（核心）
配音标注：10 分钟

审核修改：15 分钟

## 常见问题 FAQ

### Q1：如何平衡动作量与成本？

**答案**：遵循二八原则：
- 80% 的镜头用静态 + 微动作（头发/衣服飘动）
- 20% 的关键镜头用稍复杂动作（转身/抬手）
- 极个别高潮镜头用 full motion（战斗/变身）

**成本控制公式**：
```
单集成本 = (静态镜头×100 元) + (微动镜头×300 元) + (全动镜头×1000 元)
```

### Q2：如何让静态画面不枯燥？

**答案**：使用以下技巧：
1. **镜头运动**：推拉摇移代替画面本身运动
2. **光影变化**：明暗交替制造动态感
3. **粒子特效**：花瓣、雨滴、雪花、尘埃
4. **景深切换**：焦点转移引导注意力
5. **多重分层**：前景/中景/背景独立移动（Parallax）

### Q3：配音如何与口型精准同步？

**答案**：三步法：
1. **预录制配音**：先录好音频，导入剪辑软件
2. **波形分析**：根据声波峰谷确定开口/闭口时刻
3. **关键帧标注**：在波形图上标记口型变化点

**工具推荐**：
- Adobe Character Animator（自动口型同步）
- Live2D Cubism（手动精细调整）
- Spine（2D 骨骼绑定）

### Q4：长篇小说如何分集？

**答案**：按剧情节奏分集：
- **起承转合**：每集包含一个小完整的叙事弧
- **悬念结尾**：每集结尾留钩子，吸引观众追更
- **时长控制**：每集 3-5 分钟，对应小说 3000-5000 字

**分集公式**：
```
总集数 = 小说总字数 ÷ 单集字数（向上取整）
示例：10 万字小说 ÷ 4000 字/集 = 25 集
```

## 进阶技巧

### Parallax 多层摄影

**原理**：将画面分为前景/中景/背景三层，各自独立移动

```
前景层：移动最快（速度 100%）
中景层：移动中等（速度 50%）
背景层：移动最慢（速度 20%）

效果：产生 pseudo-3D 空间感
```

### 光影动态变化

**技巧**：
- **光源移动**：烛光摇曳、车灯扫过
- **阴影变化**：云层遮挡阳光、树影婆娑
- **反射效果**：水面波光粼粼、金属反光

**成本**：仅需调整光影贴图，无需重绘角色

### 粒子系统应用

**常用粒子**：
- 花瓣飘落（浪漫场景）
- 雨滴/雪花（情绪渲染）
- 尘埃浮动（真实感）
- 能量光点（奇幻/战斗）

**工具**：Unity Particle System、After Effects Particular

## 总结

动态漫画的核心不是"动得多"，而是"动得巧"。

三大原则：
1. 微动作传情：头发飘动、眼神变化胜过千言万语
2. 镜头代劳：推拉摇移创造动态错觉
3. 声音加分：配音 + 音效 + 配乐=沉浸式体验

15 秒黄金公式：
- 开头 1.5 秒：钩子抓眼球
- 发展 7.5 秒：2-3 镜推进
- 高潮 4.5 秒：1-2 镜爆发
- 结尾 1.5 秒：悬念留白

开始创作吧，让你的小说"活"过来！
