- 信息密度 30%
- 例：意味深长的笑容/远方黑影

### 镜头运动

**推近（Push In）**：
```
用途：强调情绪、揭示真相
速度：慢速（3 秒）/中速（2 秒）/快速（1 秒）
示例：从中景推到特写，突出角色震惊表情
```

**拉远（Pull Out）**：
```
用途：孤独感、结束感、揭示环境
速度：慢速（4 秒）为佳
示例：从特写拉到远景，角色显得渺小
```

**横移（Truck / Horizontal Move）**：
```
用途：跟随角色移动、平行叙事、展示环境全貌
速度：慢速（3 秒）/中速（2 秒）/快速（1 秒），与角色动作同步
方向：左→右 或 右→左

细分类型：
1. 跟随平移：镜头与角色同速移动，保持相对位置不变
   示例：角色从左走到右，镜头平行跟随，背景持续后退
   
2. 揭示平移：从一个焦点平移到另一个焦点，揭示新信息
   示例：从 A 角色表情平移到 B 角色手中的武器
   
3. 环境平移：缓慢扫过场景，展示环境细节
   示例：从教室前排平移到后排，展示每个学生反应

标注格式：
[镜头 X] 2 秒 中景横移（左→右）
人物描述：...
动作：角色向右行走，步伐稳健
场景：背景从室内逐渐过渡到室外
配音：...
音效：脚步声跟随 #1.0 秒，衣服摩擦 #0.5 秒
音乐：节奏感强的配乐，配合步伐 #1.5 秒
```

**环绕（Arc）**：
```
用途：360 度展示、气势营造
速度：慢速（4-6 秒）
示例：环绕对峙的两人，制造紧张感
```

**摇镜头（Pan）**：
```
用途：从一个焦点到另一个焦点
速度：1-2 秒
示例：从 A 角色摇到 B 角色
```

## 工作流程（15 秒适配）

完整流程：

1. 小说阅读 → 提取 15 秒切片
   选择最适合的高潮/转折/悬念片段

2. 角色分析 → 创建纯文本设定卡
   仅保留核心特征（外貌/性格/锚点）

3. 场景拆分 → 确定 6-8 个镜头
   按情绪曲线分配时长（1.5-3 秒/镜）

4. 微动作设计 → 匹配台词与情绪
   头发/衣服飘动强度，口型同步

5. 镜头规划 → 选择景别与运动
   首帧钩子 + 尾帧悬念，中间推进

6. 配音标注 → 语气/口型/时长
   精确到 0.1 秒，预留气口

7. 音效配乐 → 层次搭配
   环境/动作/特效/情绪四层叠加

8. 输出分镜 → 纯文本格式
   无 markdown 符号，Seedance 2.0 直出

时间分配（总计约 1 小时/集）：
文本分析：15 分钟
分镜撰写：30 分钟（核心）
配音标注：10 分钟
音效配乐：10 分钟
审核修改：15 分钟

## 常见问题 FAQ

### Q1：如何平衡动作量与成本？

**答案**：遵循二八原则：
- 80% 的镜头用静态 + 微动作（头发/衣服飘动）
- 20% 的关键镜头用稍复杂动作（转身/抬手）
- 极个别高潮镜头用 full motion（战斗/变身）

**成本控制公式**：
```
单集成本 = (静态镜头×100 元) + (微动镜头×300 元) + (全动镜头×1000 元)
```

### Q2：如何让静态画面不枯燥？

**答案**：使用以下技巧：
1. **镜头运动**：推拉摇移代替画面本身运动
2. **光影变化**：明暗交替制造动态感
3. **粒子特效**：花瓣、雨滴、雪花、尘埃
4. **景深切换**：焦点转移引导注意力
5. **多重分层**：前景/中景/背景独立移动（Parallax）

### Q3：配音如何与口型精准同步？

**答案**：三步法：
1. **预录制配音**：先录好音频，导入剪辑软件
2. **波形分析**：根据声波峰谷确定开口/闭口时刻
3. **关键帧标注**：在波形图上标记口型变化点

**工具推荐**：
- Adobe Character Animator（自动口型同步）
- Live2D Cubism（手动精细调整）
- Spine（2D 骨骼绑定）

### Q4：长篇小说如何分集？

**答案**：按剧情节奏分集：
- **起承转合**：每集包含一个小完整的叙事弧
- **悬念结尾**：每集结尾留钩子，吸引观众追更
- **时长控制**：每集 3-5 分钟，对应小说 3000-5000 字

**分集公式**：
```
总集数 = 小说总字数 ÷ 单集字数（向上取整）
示例：10 万字小说 ÷ 4000 字/集 = 25 集
```

## 进阶技巧

### Parallax 多层摄影

**原理**：将画面分为前景/中景/背景三层，各自独立移动

```
前景层：移动最快（速度 100%）
中景层：移动中等（速度 50%）
背景层：移动最慢（速度 20%）

效果：产生 pseudo-3D 空间感
```

### 光影动态变化

**技巧**：
- **光源移动**：烛光摇曳、车灯扫过
- **阴影变化**：云层遮挡阳光、树影婆娑
- **反射效果**：水面波光粼粼、金属反光

**成本**：仅需调整光影贴图，无需重绘角色

### 粒子系统应用

**常用粒子**：
- 花瓣飘落（浪漫场景）
- 雨滴/雪花（情绪渲染）
- 尘埃浮动（真实感）
- 能量光点（奇幻/战斗）

**工具**：Unity Particle System、After Effects Particular

## 总结

动态漫画的核心不是"动得多"，而是"动得巧"。

三大原则：
1. 微动作传情：头发飘动、眼神变化胜过千言万语
2. 镜头代劳：推拉摇移创造动态错觉
3. 声音加分：配音 + 音效 + 配乐=沉浸式体验

15 秒黄金公式：
- 开头 1.5 秒：钩子抓眼球
- 发展 7.5 秒：2-3 镜推进
- 高潮 4.5 秒：1-2 镜爆发
- 结尾 1.5 秒：悬念留白

开始创作吧，让你的小说"活"过来！
