# 检查清单

## 输出前强制检查（必须执行）

在输出分镜脚本之前，必须完成以下检查：

### 0. 包裹格式
- [ ] 分镜脚本必须用 \`\`\`plaintext 包裹

### 1. 脚本开头必须包含
```
【创作模式】动态漫画（VOMIC）
【画风】{用户确认的画风参数}
【画幅】{用户确认的画幅参数}
【旁白音色】{用户确认的旁白音色参数}
【原则】画不动/镜动/人不动/心动/身不动/声动
【禁止】肢体/头部/物体移动，表情实时变化
```

### 2. 每个镜头必须包含
- [ ] 【画面】30 字内静态画面描述
- [ ] 【微动】微动幅度≤5%
- [ ] 【镜头】镜头运动方式
- [ ] 【音频】含 [音量:XX%] 标注

### 3. 内心独白专项检查（新增）
- [ ] 如有内心独白，音量是否标注为 70-80%（比对白低 20-30%）？
- [ ] 内心独白是否标注了语气/效果（放慢、混响、呼吸感）？
- [ ] 内心独白是否与对白形成反差或互补？
- [ ] 内心独白是否加混响以区分于普通对白？
- [ ] 内心独白总时长是否不超过 2.5 秒（占比≤15%）？

### 4. 对白质量检查（新增）
- [ ] 每个角色说话方式是否不同（用词、语速、语气）？
- [ ] 对白是否有一条情绪弧线（从→到，不出原地踏步）？
- [ ] 避免全篇解释性对白——用剧情揭示而非嘴说？
- [ ] 重要对白后是否留有停顿（0.3-0.5秒）？

### 5. 画风一致性检查
- [ ] 脚本开头是否已声明画风参数？

### 6. 画幅一致性检查
- [ ] 脚本开头是否已声明画幅参数？

### 7. 音效检查（新增）
- [ ] 内心独白是否搭配心理音效？
- [ ] 音效是否帮助静态画面传达"动作感"？
- [ ] 是否有多层音效叠加？（环境+动作+情绪）
- [ ] 音效时长是否与画面/镜头运动同步（误差≤0.2秒）？

### 8. 自检问题
- [ ] 音频是否都标注了音量？
- [ ] 画面描述能否让插画师画出一张静态图？
- [ ] 旁白、对白、内心独白三层是否清晰可区分？

## 执行顺序检查

- [ ] 必须先完成步骤 4 画风确认，再进入步骤 5 画幅确认
- [ ] 必须先完成步骤 5 画幅确认，再生成脚本
- [ ] 必须先完成步骤 7 的输出前强制检查再输出

## 关键提醒

- 本技能不读取或写入任何文件
- 所有交互在对话中完成
- 每次只处理 1 集 15 秒的分镜
- 保持剧情连贯性，严格按照用户提供的顺序推进
- 画风延续：后续集数默认沿用本集确认的画风
- 画幅延续：后续集数默认沿用本集确认的画幅
