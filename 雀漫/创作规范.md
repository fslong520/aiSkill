# 创作规范

## 一、静态画面规范

### 允许的微动作（仅以下四项）

| 微动作 | 幅度 | 说明 |
|--------|------|------|
| 发丝轻微飘动 | 3-5% | 如微风轻拂，持续整个镜头 |
| 衣角微颤 | 2-3% | 如微风轻拂 |
| 眼皮缓慢眨动 | 每 2-3 秒一次 | 极缓慢 |
| 嘴唇配合台词轻微张合 | 仅在台词时 | 配合音频 |

### 绝对禁止的动作

**出现任何一个都说明你在做视频而不是动态漫画**：

- **肢体移动**：走动、转身、抬手、握拳、迈步、伸手、触碰
- **头部移动**：点头、摇头、转头、低头、抬头
- **身体移动**：跳跃、下落、旋转、弯腰、站起
- **物体移动**：物品飞出、物体掉落、泪珠滚动、书页翻动
- **大幅度表情变化**：从微笑变成大哭、从平静变成愤怒

### 禁止词汇表

**绝对禁止出现在画面描述中**：

- 英文：walking, running, turning, jumping, falling, reaching, grabbing, throwing, punching, kicking, nodding, shaking head, waving
- 中文：走动、跑动、转身、跳跃、下落、伸手、抓取、投掷、挥拳、踢腿、点头、摇头、挥手
- 任何描述"动作过程"的语句，如"他慢慢抬起手""她转过身来""泪珠滑落"

### 如何表现"动作"

| 方法 | 说明 | 示例 |
|------|------|------|
| 镜头切换 | 不同镜头展示动作的"前"和"后"两个静态瞬间 | 镜头 1 画背影，镜头 2 画正面 |
| 旁白叙述 | 用旁白描述动作过程 | "他猛地站起身来"（画面展示站起后的姿态） |
| 音效暗示 | 用声音暗示动作 | 椅子倒地声（画面展示椅子已倒地） |
| 镜头运动 | 用镜头制造动感 | 快速推近制造冲击感，缓慢拉远制造孤独感 |

## 二、音频设计规范

### 音频三要素（更新版）

| 类型 | 时长占比 | 使用场景 | 语速 | 示例 |
|------|---------|---------|------|------|
| 旁白 | 40-60% | 场景描述、剧情推进、情感渲染、动作叙述 | 3-4字/秒 | 旁白："黄昏时分，天台上风很大。" #2.0 秒 [音量:100%] |
| 配音（对白） | 20-33% | 角色对话、冲突交锋、情绪爆发 | 3-5字/秒 | 配音（张三）："你再说一遍。" #1.0 秒 [音量:100%]（青年音+压抑怒火）|
| 内心独白 | 5-15% | 角色内心真实想法、情感反差、真相揭露 | 2-3字/秒 | 内心独白："原来他一直都知道……" #1.5 秒 [音量:80%]（放慢+混响）|
| 纯音效/留白 | 3-7% | 情绪消化、气氛营造、转场过渡 | — | （音效）风声渐起，钟声回荡 #1.0 秒 |

### 内心独白设计（新增）

#### 内心独白的本质

内心独白 = 观众唯一能听见的角色"真实声音"。

动态漫画中画面静止，内心独白是揭示角色真实情感的**主要手段**。

#### 内心独白 vs 旁白

| | 内心独白 | 旁白 |
|------|---------|------|
| 视角 | 第一人称（我） | 第三人称（上帝视角） |
| 内容 | 真实想法、隐藏情绪 | 背景信息、剧情补全 |
| 语气 | 私密、真实、脆弱 | 客观、平稳 |
| 混响 | 轻微混响（2-5%） | 无混响/旁白腔 |
| 音量 | 70-80% | 100% |

#### 内心独白使用三大时机

1. **对白反差**：嘴上说A，心里想B
   - 对白："没事。" + 内心独白："有事，天大的事。"
2. **真相揭示**：角色发现真相，但表面不动声色
   - 画面：角色面无表情看着对方
   - 内心独白："原来是你……我等了十年。"
3. **情绪临界**：表面平静，内心已波涛汹涌
   - 画面：角色坐在椅子上，神态平静
   - 内心独白："别慌别慌，他看不出来的……"

#### 内心独白标注格式

```
内心独白："内容" #时长 [音量:70-80%]（语气+效果）
```

示例：
```
内心独白："呵……真是可笑。" #1.2 秒 [音量:75%]（放慢，带嘲讽混响）
```

#### 内心独白音量规则

- 音量比对白低 20-30%（建议 70-80%）
- 加轻微混响（2-5%）以区分"心声"和"人声"
- 保留呼吸声、换气声增强真实感

---

### 对白冲突设计（新增）

#### 三种对白类型

| 类型 | 字句 | 节奏 | 效果 |
|------|------|------|------|
| **冲突式** | 短句2-6字，句句逼人 | 快，每秒5-7字 | 紧张、压迫感 |
| **递进式** | 从长到短，情绪逐句升温 | 先慢后快 | 从平静到爆发 |
| **潜台词式** | 话只说三分，留七分给观众 | 慢，多停顿 | 暧昧、张力 |

#### 冲突式对白示例

```
配音（A）："你走。" #0.5 秒 [音量:90%]（冷）
配音（B）："凭什么。" #0.5 秒 [音量:95%]（硬）
配音（A）："别让我再说一遍。" #0.8 秒 [音量:100%]（压抑怒火）
```

#### 递进式对白示例

```
配音（A）："你确定要去？" #1.0 秒 [音量:80%]（平静）
配音（A）："那里很危险。" #0.8 秒 [音量:90%]（开始担心）
配音（A）："你会死的！！" #0.6 秒 [音量:100%]（爆发）
```

#### 潜台词式对白 + 内心独白搭配

```
配音（A）："这茶不错。" #1.0 秒 [音量:100%]（微笑）
内心独白（A）："别走……再坐一会儿。" #1.2 秒 [音量:75%]（放慢+混响）
配音（B）："天快黑了。" #0.8 秒 [音量:90%]（低头）
内心独白（B）："我得走了，再待下去我怕我舍不得。" #1.5 秒 [音量:75%]（放慢+混响）
```

### 旁白使用场景详解

1. **场景描述**：交代时间、地点、环境
2. **剧情推进**：补充画面无法传达的信息
3. **情感渲染**：增强情绪感染力
4. **动作叙述**：用旁白描述画面中无法展现的动作过程
5. ~~心理活动~~ → 改用**内心独白**

> 注意：角色心理活动应由内心独白承担，不再由旁白包办。旁白更多负责客观叙事，内心独白负责主观情感。

---

### 旁白语速标准（强制执行）

**核心公式：旁白时长 = 字数 ÷ 3（正常语速）**

| 语速类型 | 每秒字数 | 适用场景 |
|---------|---------|---------|
| 慢速深情 | 2-3 字 | 情感渲染、悲伤回忆、悬念营造 |
| 正常叙述 | 3-4 字 | 场景描述、剧情推进 |
| 快速紧张 | 5-6 字 | 危机时刻、高潮爆发 |

**强制规则**：
1. 默认使用"正常叙述"语速（每秒 3-4 字）
2. 如需慢速或快速，必须在语气标注中说明
3. 单句旁白最长不超过 25 字

---

### 旁白音色控制规范

**旁白必须标注音色**，格式：`（音色 + 语气）`

**音色维度**：

| 维度 | 选项 |
|------|------|
| 性别 | 男声/女声 |
| 年龄 | 青年/中年/老年 |
| 特点 | 低沉/清亮/沙哑/温柔/磁性/冷淡 |
| 效果 | 混响/旁白腔/说书人 |

**常用旁白音色组合**：

| 场景类型 | 推荐音色 |
|---------|---------|
| 悬疑穿越 | 男声中年低沉混响 |
| 青春校园 | 女声青年清亮 |
| 古风武侠 | 男声中年说书人 |
| 都市情感 | 女声青年温柔 |
| 恐怖惊悚 | 男声低沉冷淡 |
| 历史诗剧 | 男声老年混响 |

### 音效设计规范（大幅扩展）

#### 音效六层体系

| 层 | 名称 | 作用 | 音量比例 |
|----|------|------|---------|
| 1 | 环境氛围音 | 建立时空基础 | 30-40% |
| 2 | 情绪音效 | 放大心理感受 | 40-60% |
| 3 | 心理音效 | 内心独白专属背景 | 30-50% |
| 4 | 动作音效 | 静态画面中暗示动作 | 50-70% |
| 5 | 转场音效 | 镜头过渡标记 | 60-80% |
| 6 | 特效音 | 视觉增强 | 70-90% |

#### 心理音效（新增）

内心独白时搭配专属氛围音：

| 内心独白情绪 | 心理音效 |
|-------------|---------|
| 失落/悲伤 | 混响+低频嗡鸣，如深水回音 |
| 紧张/恐惧 | 急促心跳声，逐渐加快 |
| 回忆/怀念 | 短暂回声+老旧唱片质感 |
| 震惊/顿悟 | "嗡"一声长鸣，短暂空白 |
| 决断/觉醒 | 鼓点+逐渐明亮的单音 |

#### 动态漫画中的音效使用策略

动态漫画画面不动，音效须承担原本由"动作"承担的叙事功能：

| 画面状态 | 音效策略 |
|---------|---------|
| 画面是一人站立的静态图 | 加风声+远处鸟鸣→"他在等"
| 画面是两人对视的静态图 | 加心跳声+微妙呼吸→"紧张对峙"
| 画面是空房间的静态图 | 加时钟滴答+地板咯吱→"诡异" |
| 画面是角色表情特写 | 加内心独白+心理音效→"内心戏" |

---

### 标注格式总表

```
旁白："内容" #时长 [音量:100%]（音色 + 语气）
配音（角色名）："台词" #时长 [音量:100%]（声线 + 情绪）
内心独白："内容" #时长 [音量:70-80%]（放慢+混响）
（音效）描述 #时长
（心理音效）描述 #时长
```
