# 示例：今日 ABC457 题解生成全流程

## 输入

```
用户：使用urlgo技能配合搬题姬技能搬运https://atcoder.jp/contests/abc457/tasks
      → 生成题面汇总 + 完整题包（A~E）

用户：生成abcde的题解，要包含题面，而且是启发式的题解
      → 使用析题技能生成启发式题解
```

## 产出

1. `abc457.md` —— 7 题题面汇总（搬题姬产出）
2. `abc457a_数组.zip` ~ `abc457e_交叉桌布.zip` —— 5 份完整题包（搬题姬产出）
3. `abc457_题解.md` —— 539 行启发式题解（析题产出）

## 题解特点

- 每道题从「读完题脑中一片空白」开始推演
- 展示暴力 → 撞墙 → 转向 → 优化的完整思维路径
- 代码逐行精讲（结合用户指定的标程文件）
- 每道题附常见错误分析表
- 末尾五题串讲总结

## 协作模式

```
搬题姬（获取/翻译题面）→ 析题（启发式题解撰写）
     ↓                          ↓
  URL → 题面                题面 → 详解
     题面汇总               + 标程精讲
     + 完整题包              + 错误分析
                             + 串讲总结
```
