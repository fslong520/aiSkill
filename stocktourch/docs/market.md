# 大盘分析说明

实时分析 A 股大盘走势，生成市场分析报告。

---

## 触发场景

用户询问大盘情况：
- "大盘怎么样"
- "今天大盘走势"
- "市场行情分析"

---

## 分析方式

**AI 直接使用内置搜索工具获取实时市场数据**，不依赖脚本数据源。

---

## 搜索内容

### 1. 先获取当前日期
```python
from datetime import datetime
today = datetime.now().strftime("%Y年%m月%d日")
```

### 2. 搜索查询示例（假设当前日期 2026-03-30）

| 内容 | 搜索查询 |
|------|----------|
| 指数收盘 | "2026年3月30日 A股收盘 上证指数 深证成指 创业板指" |
| 大盘分析 | "2026年3月30日 A股大盘走势分析" |
| 热点板块 | "2026年3月 A股热点板块 概念炒作" |
| 市场策略 | "2026年3月 A股投资策略 建议" |
| 外围市场 | "2026年3月 美股走势 全球市场" |

### 3. 数据源

优先搜索：
- 新浪财经
- 东方财富
- 同花顺
- 雪球
- 华尔街见闻
- 上海证券报
- 中国证券报

---

## 时效性要求

1. **必须先获取当前日期**
2. **如非交易日**：说明"截至最近交易日 ([日期]) 数据"
3. **报告标注具体日期**：每条数据注明来源和时间
4. **禁止模糊时间词**：不用"最近"、"近期"等
