# 综合分析方法论

将技术面、基本面、财务面、资金流、新闻面数据进行融合分析，生成完整投资报告。

---

## 分析流程

```
数据获取 → 数据整合 → 多维度评分 → 风险评估 → 操作建议 → 生成报告
```

---

## 第一步：数据整合

将各数据源模块获取的数据进行交叉验证：

| 数据源 | 验证点 |
|--------|--------|
| 技术 + 资金流 | 资金流向与技术指标是否背离 |
| 财务 + 新闻 | 财报数据与公告信息是否一致 |
| 同业 + 估值 | 行业估值与个股估值是否合理 |

---

## 第二步：多维度评分

基于五个维度进行量化评分（每项 0-10 分）：

### 1. 基本面评分
- PE/PB 相对行业估值水平
- ROE、毛利率、净利率水平
- 成长性（营收/净利润增长率）

### 2. 技术面评分
- 均线系统排列（多头/空头）
- RSI 超买超卖区间
- MACD 金叉死叉信号
- 多周期共振情况

### 3. 财务面评分
- 审计意见（标准无保留 = 10分）
- 现金流肖像（+-- 最佳）
- 五大黄金标准达标数
- 造假风险红旗数量

### 4. 资金流评分
- 主力资金净流入方向
- 北向资金持股变化趋势
- 资金流向持续性

### 5. 情绪面评分
- 市场关注度（机构调研频率）
- 媒体报道倾向（正面/负面）
- 行业政策影响

---

## 第三步：风险评估

### 风险等级划分

| 综合得分 | 风险等级 | 建议 |
|----------|----------|------|
| 40+ | 低风险 | 可考虑建仓 |
| 30-40 | 中低风险 | 谨慎建仓 |
| 20-30 | 中风险 | 观望为主 |
| 10-20 | 高风险 | 不建议买入 |
| <10 | 极高风险 | 建议回避 |

### 风险因素清单

1. **财务风险**：现金流异常、高负债、商誉占比高
2. **技术风险**：均线空头排列、跌破关键支撑
3. **资金风险**：主力持续流出、北向减持
4. **消息风险**：负面公告、行业政策利空
5. **估值风险**：PE 远高于行业均值

---

## 第四步：操作建议

### 买入条件
- 综合得分 ≥ 35
- 技术面出现买入信号（MACD金叉/RSI超卖反弹）
- 资金流主力净流入
- 无重大风险因素

### 持有条件
- 综合得分 25-35
- 技术面未出现卖出信号
- 基本面无恶化迹象

### 卖出条件
- 综合得分 < 20
- 技术面破位（跌破关键支撑）
- 资金流持续流出
- 出现重大利空消息

### 止损止盈建议
- 止损位：关键支撑位下方 3-5%
- 止盈位：关键压力位附近或盈利 20-30%

---

## 第五步：生成报告

### 报告结构

```markdown
# [股票名称] 投资分析报告

**报告日期**：YYYY-MM-DD HH:MM:SS
**股票代码**：XXXXXX
**数据来源**：技术数据(baostock)、财务数据(AKShare)、网络信息(search_web)

---

## 一、基本信息
- 股票名称、代码、当前价格、涨跌幅

## 二、技术面分析
- 均线系统分析
- 技术指标解读（RSI/MACD/KDJ）
- 多周期综合判断

## 三、财务面分析
- 资产负债表分析
- 利润表分析
- 现金流量表分析
- 杜邦分析结果

## 四、资金流分析
- 主力资金流向
- 北向资金变化
- 资金与技术面背离分析

## 五、同业对比
- 行业估值对比
- 关键指标排名

## 六、新闻面分析
- 公司公告摘要
- 行业动态
- 分析师观点

## 七、综合评分
| 维度 | 得分 | 说明 |
|------|------|------|
| 基本面 | X/10 | ... |
| 技术面 | X/10 | ... |
| 财务面 | X/10 | ... |
| 资金流 | X/10 | ... |
| 情绪面 | X/10 | ... |
| **综合** | **XX/50** | ... |

## 八、操作建议
- [买入/持有/卖出/观望]
- 操作理由
- 止损止盈位

## 九、风险提示
- 主要风险因素
- 需关注的信号

## 十、数据时效性说明
- 技术数据：截至 [日期] 收盘
- 财务数据：基于 [年份]Q[季度] 报告
- 网络信息：[时间范围] 搜索结果
```

### 报告保存
- 文件名：`{股票名称}_分析报告_{YYYYMMDD_HHMMSS}.md`
- 保存目录：`reports/`
