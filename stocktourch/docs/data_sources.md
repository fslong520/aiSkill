# 数据源模块说明

每个模块既是综合分析的数据源，也可独立输出给用户。

---

## 模块总览

| 模块 | 命令参数 | 独立输出 | 综合分析用途 |
|------|----------|----------|--------------|
| 技术分析 | `technical` | 技术指标解读 | 技术面评分数据 |
| 财务分析 | `finance` | 财报分析报告 | 财务面评分数据 |
| 资金流分析 | `capital_flow` | 资金流向解读 | 资金流评分数据 |
| 同业对比 | `compare_peers` | 估值对比表格 | 行业定位数据 |

---

## 1. 技术分析模块

### 获取方式
```bash
python3 run_skill.py <股票代码> technical
```

### 独立输出内容（用户只想看技术指标）
```markdown
# [股票名称] 技术分析

**日期**：截至 [日期] 收盘

## 均线系统
- MA5/10/20/60 数值及排列状态
- 均线支撑/压力位判断

## 技术指标
- **RSI**：数值 + 超买超卖判断
- **MACD**：DIF/DEA/柱状体 + 金叉死叉信号
- **KDJ**：K/D/J 数值 + 信号判断

## 综合判断
- 短期趋势：[上涨/下跌/震荡]
- 建议关注信号：[具体信号]
```

### 多周期分析
```bash
python3 run_skill.py <股票代码> technical --period daily_weekly_monthly
```

输出：
- **日线**：短期趋势（1-4周）、MA5/10/20、短期支撑压力
- **周线**：中期趋势（1-3月）、周线技术指标
- **月线**：长期趋势（3月以上）、历史位置判断
- **多周期共振**：三个级别信号一致性分析

## 2. 财务分析模块

### 获取方式

```bash
python3 run_skill.py <股票代码> finance
```

**数据来源**：akshare 的 `stock_financial_abstract` 接口

**特点**：
- ⚡ 速度快：0.5-1 秒获取数据
- 📊 数据全：80+ 指标，73+ 报告期（近 18 年）
- ✅ 准确可靠：官方数据源

### 输出格式

```
================================================================================
📋 财务核心指标
================================================================================

指标                   2025Q3      2025Q2      2024Q4      2024Q3      2023Q4
--------------------------------------------------------------------------------
营收(亿)              15582.20    11083.07    21871.48    16265.41    22655.29
归母净利润(亿)             381.82      304.04      461.87      397.02      542.64
扣非净利润(亿)             352.01      274.98      415.81      370.21      485.39
ROE(%)                   8.03        6.44       10.37        9.04        9.17
净利率(%)                 3.17        3.63        2.87        3.21        3.53
毛利率(%)                 8.72        9.43        9.86        8.83        9.12
资产负债率(%)              76.07       76.27       75.81       76.16       76.41
```

### 分析要点

获取财务数据后，**必须按《读财报》方法论深度分析**：

#### 一、审计意见检查（一票否决）
- 标准无保留意见 → 继续分析
- 保留意见/无法表示意见/否定意见 → **直接排除**

#### 二、资产负债表速读四步法
1. **看资产总貌**：资产负债率 < 70% 为安全
2. **看生产资产占比**：> 50% 为重资产公司
3. **看应收账款占比**：< 10% 优秀，> 30% 质量差
4. **货币资金 vs 有息负债**：货币资金 > 有息负债为不差钱

#### 三、现金流量表分析
- **现金流肖像**：+--（奶牛型）最优
- **五大黄金标准**：
  1. 经营现金流 > 净利润
  2. 销售收到现金 ≥ 营业收入
  3. 经营现金流连续为正
  4. 净利润含金量 > 100%
  5. 现金流肖像为奶牛型

#### 四、造假风险红旗排查
- 商誉/净资产 > 30%：高危
- 存贷双高：假钱风险
- 应收增速 > 营收增速×2：压货虚增风险
- 其他应收款金额大：关联方占款风险

#### 五、杜邦分析（ROE拆解）
```
ROE = 净利率 × 周转率 × 杠杆
```
- 高净利率型：品牌溢价
- 高周转型：效率驱动
- 高杠杆型：资金利差

#### 六、财务健康度评分（90分制）

| 检查项 | 满分 | 标准 |
|--------|------|------|
| 审计意见 | 10 | 标准无保留 |
| 资产负债率 | 10 | < 70% |
| 应收占比 | 10 | < 10% |
| 货币资金充足 | 10 | > 有息负债 |
| 经营现金流 | 10 | 连续为正 |
| 现金流肖像 | 10 | +--奶牛型 |
| 净利润含金量 | 10 | > 100% |
| 商誉风险 | 10 | < 10% |
| 无造假红旗 | 10 | 无红旗信号 |

**评分标准**：≥80优秀，≥60良好，≥40一般，<40较差

- **营收增长**：近 3 年 CAGR = X%

获取 3 年数据后，重点分析：

1. **盈利能力趋势**：ROE 是否稳定在 15% 以上？毛利率是否稳定或改善？
2. **成长性**：营收和利润是否持续增长？增长是加速还是放缓？
3. **财务风险**：资产负债率是否超过 70%？经营现金流是否持续为正？
4. **行业对比**：ROE、净利率在行业中排名如何？

## 3. 资金流分析模块

### 获取方式
```bash
python3 run_skill.py <股票代码> capital_flow
```

### 独立输出内容（用户只想看资金流向）
```markdown
# [股票名称] 资金流分析

**统计周期**：近 [N] 个交易日 ([日期范围])

## 主力资金流向
| 类型 | 净流入 | 占比 |
|------|--------|------|
| 超大单 | XXX万 | X% |
| 大单 | XXX万 | X% |
| 中单 | XXX万 | X% |
| 小单 | XXX万 | X% |

## 北向资金
- 陆股通持股比例：X%
- 近5日变化：[增持/减持] X万股
- 近20日趋势：[持续流入/持续流出/震荡]

## 资金与股价背离分析
- [资金流入但股价下跌 → 主力吸筹？]
- [资金流出但股价上涨 → 散户推动？]

## 资金流结论
- 主力意图判断：[建仓/出货/观望]
- 资金面信号：[积极/中性/消极]
```

### 数据内容

**主力资金流向**：
- 超大单净流入（机构资金）
- 大单净流入（大户资金）
- 中单净流入（中户资金）
- 小单净流入（散户资金）
- 主力净流入占比成交额

**北向资金**：
- 陆股通持股比例
- 近5日/10日/30日持股变化
- 北向资金买卖金额

**南向资金**（港股通标的）：
- 港股通持股比例变化

**背离分析**：
- 资金流入但股价下跌
- 资金流出但股价上涨
- 主力意图解读

## 5. 新闻/舆情数据

### 获取方式
AI 使用内置 `search_web` 工具搜索，**注意时效性**。

### 搜索要求
1. **先获取当前日期**
2. **禁止模糊时间词**（如"最近"、"近期")
3. **必须用具体日期范围**

### 搜索内容示例
（假设当前日期 2026-03-30）

| 内容类型 | 时间范围 | 搜索示例 |
|----------|----------|----------|
| 公司公告/重大事件 | 当前月份 | "豪威集团 603501 2026年3月 最新公告" |
| 行业动态/政策 | 近3个月 | "半导体 CIS 行业 2026年发展趋势" |
| 分析师观点 | 近3个月 | "韦尔股份 2026年 机构评级 目标价" |
| 机构调研 | 近1-3个月 | "豪威集团 2026年 机构调研 持仓变化" |
| 资金流向 | 近5/10/20日 | "豪威集团 近5日 主力资金流向" |

### 报告标注要求
- 每条信息注明来源和具体日期
- 传闻标注"据媒体统计"或"市场传闻"
- 历史数据说明时间范围

---

## 数据时效性说明

| 数据类型 | 时效性 | 报告标注 |
|----------|--------|----------|
| 技术数据 | 最新交易日 | "截至 [日期] 收盘" |
| 财务数据 | 最新财报期 | "基于 [年份]Q[季度] 报告" |
| 资金流 | 近5/10/20日 | "近 [N] 个交易日 ([日期范围])" |
| 新闻 | 当前月份 | "[日期] [来源] 报道..." |
