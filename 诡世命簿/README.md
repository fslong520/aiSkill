# 诡世命簿

回合制异世界人生模拟游戏。随机生成世界观、出身，每回合做出选择，体验完整人生。

## 快速开始

### 开始新游戏
说：**"开始新游戏"** 或 **"翻开命簿"**

系统会：
1. 随机生成异世界世界观（可指定偏好）
2. 随机生成你的出身
3. 进入人生第一回合

### 继续游戏
说：**"继续"** 或 **"下一回合"**

### 查看状态
说：**"状态"** 或 **"属性"**

### 生成图片
说：**"生成角色卡"** 或 **"生成场景图"**

## 命令列表

| 命令 | 说明 |
|------|------|
| 新游戏 / 开始 | 开始新的人生 |
| 继续 / 下一回合 | 推进时间 |
| 跳过N回合 | 快进N个回合 |
| 状态 / 属性 | 显示当前状态 |
| 故事 | 显示故事记录 |
| 世界 / 世界观 | 显示世界信息 |
| 人物 [名字] | 显示角色卡 |
| 关系 | 显示人际关系 |
| 存档 | 保存游戏 |
| 读档 [名字] | 加载存档 |
| 生成角色卡 | 生成角色卡图片 |
| 生成场景 | 生成场景图片 |

```

│   ├── templates/
```

## 游戏特色

- 🌍 **随机世界观** — 每周目都是全新世界
- 👤 **随机出身** — 命运从出生就已注定
- ⏳ **回合制** — 每回合一年，推进人生
- 🎲 **事件选择** — 你的选择塑造命运
- 📖 **故事生成** — 自动记录完整人生
- 🖼️ **图片生成** — 关键场景和角色卡插图

## 世界类型

- 奇幻中世纪 — 剑与魔法
- 东方玄幻 — 修仙问道
- 蒸汽朋克 — 机械与齿轮
- 黑暗奇幻 — 克苏鲁风格
- 异能都市 — 现代超能力
- 原始部落 — 远古文明
- 废土末世 — 文明废墟

## 开发说明

本游戏使用 Skill 系统驱动，核心文件为 `SKILL.md`。

扩展游戏内容：
1. 添加事件类型 → 编辑 `references/event_system.md`
2. 添加世界观元素 → 编辑 `references/world_gen.md`
3. 添加事件模板 → 编辑 `assets/templates/event_template.md`
