## 勇者启程

小明是个普通的中学生，某天他在图书馆发现了一本神秘的书，书中写道：

> *"当你翻开这一页，冒险已经开启。在这个数字世界里，你就是勇者。用代码作为你的武器，去解开一个个谜题吧！"*

小明深吸一口气，开始了他奇妙的编程冒险……

## 三、编程题（每题 100 分，共 600 分）

### 1、勇者集结(100分)

**题目描述**
两个勇者小明和小红决定组队冒险。系统需要统计他们的总金币数量。
输入小明和小红的金币数，输出总金币数。

**输入格式**
一行两个整数 a 和 b，分别表示小明和小红的金币数。

**输出格式**
一个整数，表示两人金币总数。

**样例输入**
```
35 48
```

**样例输出**
```
83
```

---

### 2、魔法圆阵(100分)

**题目描述**
小明学会了一个魔法，可以在地上画出一个魔法圆阵。已知圆的半径，求圆阵的面积。
公式：面积 = π × 半径²，π 取 3.14159

**输入格式**
一个实数 r，表示圆的半径。

**输出格式**
圆阵的面积，保留 2 位小数。

**样例输入**
```
3
```

**样例输出**
```
28.27
```

---

### 3、药剂炼制(100分)

**题目描述**
小明在炼制一种特殊的药剂，需要精确控制温度。炼丹炉显示的是华氏温度，但配方上写的是摄氏温度。
请帮他转换温度。公式：摄氏 = (华氏 - 32) × 5 ÷ 9

**输入格式**
一个实数 F，表示华氏温度。

**输出格式**
转换后的摄氏温度，保留 2 位小数。

**样例输入**
```
212
```

**样例输出**
```
100.00
```

---

### 4、怪物图鉴(100分)

**题目描述**
小明的图鉴上记录着怪物编号。编号为奇数的是攻击型怪物，编号为偶数的是防御型怪物。
输入怪物编号，判断它的类型。

**输入格式**
一个整数 n，表示怪物编号。

**输出格式**
如果是奇数输出 "attack"，如果是偶数输出 "defense"。

**样例输入**
```
23
```

**样例输出**
```
attack
```

---

### 5、经验积累(100分)

**题目描述**
小明打败了一群史莱姆，史莱姆的数量为 n 只。每只史莱姆的编号从 1 到 n，打败编号为 i 的史莱姆可获得 i 点经验。
求小明总共获得多少经验？
提示：1+2+3+...+n = n×(n+1)÷2

**输入格式**
一个正整数 n，表示史莱姆数量。

**输出格式**
总经验值。

**样例输入**
```
10
```

**样例输出**
```
55
```

---

### 6、装备交换(100分)

**题目描述**
小明和小红想要交换装备。小明有武器编号 a，小红有武器编号 b。
编写程序帮他们交换装备编号。

**输入格式**
一行两个整数 a 和 b，用空格分隔。

**输出格式**
交换后的两个整数，用空格分隔。

**样例输入**
```
剑盾 斧头
3 7
```

**样例输出**
```
7 3
```
