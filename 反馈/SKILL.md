# C++ 课后反馈生成器

## 技能描述
此技能用于生成符合信息学奥赛培训规范的C++课后反馈文档。生成的文档将遵循我们约定的Markdown书写规范，包括变量命名、列表格式等要求。

## 功能特点
- 生成标准格式的课后反馈文档
- 符合Markdown书写规范（列表前空行等）
- 遵循变量命名长度限制（不超过5个字母）
- 包含教学目的、重点知识、上课情况和课后作业等标准部分

## 使用场景
- 为C++编程课程生成课后反馈
- 为算法课程生成课后反馈
- 为数论、数据结构等专题生成课后反馈

## 使用方法
调用此技能时，提供以下信息：
1. 课程主题和知识点
2. 学生姓名和表现情况
3. 课堂评价和作业要求

生成的反馈将包含：
- 教学目的（5个要点）
- 重点知识（含概念回顾、核心思路和关键代码）
- 上课情况（表现情况和课堂评价）
- 课后作业

## 规范要求
- 所有有序/无序列表前必须空一行
- 变量命名长度不超过5个字母
- 代码格式清晰，注释充分
- 保持教学语言的专业性和亲和力
