## 一、重点知识

### 1. 绝对素数（反转后仍是素数）

**概念回顾**：绝对素数是指一个素数，将其各位数字反转后仍然是素数。例如：13是素数，31也是素数，则13和31都是绝对素数。

**核心思路**：

- 首先判断原数是否为素数
- 将原数各位数字反转
- 判断反转后的数是否也为素数
- 若两者均为素数，则为绝对素数

**关键代码**：
```cpp
// 反转数字函数
int revN(int n)
{
    int res = 0;
    for(int temp = n; temp > 0; temp /= 10)
    {
        int digit = temp % 10;  // 新建变量存储当前位
        res = res * 10 + digit;
    }
    return res;
}

// 判断绝对素数
bool isAP(int n)
{
    if(!isPrime(n)) return false;  // 原数不是素数
    int rev = revN(n);             // 反转数字
    return isPrime(rev);           // 判断反转后是否为素数
}
```

### 2. 素数判断

**概念回顾**：素数是只能被1和自身整除的大于1的自然数。判断一个数是否为素数是数论中最基础的问题之一。

**核心思路**：

- 对于数字n，只需检查2到√n之间的所有数
- 若存在能整除n的数，则n不是素数
- 否则n为素数
- 特殊情况：1不是素数，2是最小的素数

**关键代码**：
```cpp
// 基础素数判断
bool isPrime(int n)
{
    if(n <= 1) return false;  // 小于等于1的数不是素数
    if(n == 2) return true;   // 2是素数
    if(n % 2 == 0) return false;  // 偶数不是素数（除了2）
    
    for(int i = 3; i * i <= n; i += 2)  // 只需检查到sqrt(n)
    {
        if(n % i == 0) return false;  // 找到因子，不是素数
    }
    return true;  // 是素数
}
```

### 3. 整数的向上取整算法

**概念回顾**：向上取整是指不小于给定数字的最小整数。在数学中记作⌈x⌉，即返回大于或等于x的最小整数。

**核心思路**：

- 对于正整数除法的向上取整，可以用(a+b-1)/b来计算
- 只考虑正整数情况

**关键代码**：
```cpp
// 对于正整数除法的向上取整
int cdiv(int a, int b)  // 计算⌈a/b⌉
{
    return (a + b - 1) / b;  // 正整数向上取整的巧妙算法
}
```

**实际应用**：

- 分配资源时的最少单位计算
- 数组分块处理的块数确定
- 页码计算等实际场景
