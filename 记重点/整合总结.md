## 🔧 技术实现

### 触发检测

在记重点的触发条件中新增：
```python
# 技能学习触发
if ai_completed_first_operation:
    if generated_complete_steps:
        trigger_skill_learning_record()
```

### 信息提取

从对话中提取：
- 技能名称（从用户问题和 AI 回答中提取）
- 操作步骤（从 AI 生成的指南中提取）
- 学习来源（从 browser_use 访问的 URL 中提取）
- 注意事项（从 AI 回答的警告/提示中提取）

### 写入记忆

使用 `edit_file` 工具：
```python
edit_file(
    file_path="MEMORY.md",
    old_text="## 🧠 技能库\n\n*(当 AI 首次完成某个操作流程后，自动记录到这里)*",
    new_text="## 🧠 技能库\n\n### [技能名称]\n- 学习时间：...\n..."
)
```
