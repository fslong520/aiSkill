## 🔄 先查后执行

### 运行方式

```
识别到任务/工作类重点
    ↓
📖 先翻 MEMORY.md 搜索相关记录
    ↓
┌─────────────────────────────────────┐
│ 找到相关记录？                        │
│                                     │
│ ✅ 有 → 参考已有记录 + 按新要求调整   │
│ ❌ 没有 → 边做边记，完成后写入        │
└─────────────────────────────────────┘
```

### 比喻理解

**员工记本本**：
- 老板安排工作 → 员工记到本本
- 下次同样工作 → 先翻本本查记录
- 有记录直接用 → 不从头开始
- 没记录边做边记 → 完成后写入

**AI 记重点**：
- 识别重点 → 写入 MEMORY.md
- 遇到类似任务 → 先查 MEMORY.md
- 找到相关记录 → 参考执行
- 没找到 → 边做边记
