Domain keywords: 记重点, 情绪识别, 自动记录, 记忆系统, 关键字

Summary: 自动识别对话重点（情绪+内容），立即写入记忆文件。

Strategy:
1. 实时监听对话 → 情绪分析（🔴高/🟠中高/🟡中/🟢低）
2. 识别重点类型（情绪/决策/任务/时间/偏好/上下文）
3. 提取关键字（3-5个核心词）
4. 立即写入记忆文件（`edit_file` 或 `write_file`）
5. 确认告知用户

AVOID:
- AVOID 只输出"已记录"但不调用工具写入
- AVOID 忽略高强度情绪信号（感叹号、重复强调）
- AVOID 不知道性别时随意用"他/她"
- AVOID 记录密码、身份证号等敏感信息

## 情绪强度等级

| 等级 | 信号 | 优先级 |
|------|------|--------|
| 🔴 高 | 感叹号连用、重复强调、语气词 | 立即记录+高亮 |
| 🟠 中高 | 强调词、反问句、命令式 | 记录+标注 |
| 🟡 中 | 陈述句、一般描述 | 正常识别 |
| 🟢 低 | 闲聊、寒暄 | 忽略或简记 |

## 重点类型与存储

| 类型 | 存储位置 |
|------|----------|
| 🔴 情绪类重点 | MEMORY.md |
| 🔴 时间敏感任务 | memory/YYYY-MM-DD.md |
| 🟠 决策类 | MEMORY.md |
| 🟡 任务类 | memory/YYYY-MM-DD.md |
| 🟢 偏好类 | PROFILE.md |

## 写入执行

识别到重点后**立即调用工具**：
1. `read_file` 读取目标文件
2. `edit_file` 或 `write_file` 写入
3. 确认写入成功
4. 告知用户

**绝不假装记录！**
