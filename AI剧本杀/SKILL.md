# AI剧本杀

回合制异世界人生模拟游戏。玩家穿越到随机生成的异世界，从出生开始，每回合做出选择，经历一生。

## 核心理念

1. **游戏为主体**：强交互性，每回合玩家做出选择
2. **后台静默成书**：小说在后台自动生成，不干扰游戏
3. **剧本命名**：世界观生成后自动命名，增强代入感

---

## 游戏流程

```
新游戏 → 世界观生成 → 剧本命名 → 展示剧本名+世界观 → 选择角色 → 第一回合 → ...
```

---

## 核心命令

| 命令 | 说明 |
|------|------|
| `新游戏` / `开始` | 开始新的人生 |
| `继续` | 推进到下一回合 |
| `状态` | 显示当前状态 |
| `小说` | 查看后台生成的小说 |
| `存档` / `读档` | 存取游戏 |

---

## 执行流程（重要！）

**开始新游戏时，按顺序读取并执行：**

1. **生成世界观** → 读取 `references/world_gen.md`，生成世界
2. **命名剧本** → 根据世界观生成有诗意的剧本名
3. **展示世界** → 输出剧本名和世界观摘要
4. **生成角色** → 读取 `references/character.md`，生成3个角色选项
5. **开始游戏** → 玩家选择角色后，进入第一回合

**每回合执行：**

1. **生成事件** → 读取 `references/event_system.md`，根据阶段生成事件
2. **输出回合** → 读取 `references/output_format.md`，按格式输出
3. **后台成书** → 读取 `references/novel_gen.md`，静默写入小说段落

## 人生阶段

| 阶段 | 年龄 | 特点 |
|------|------|------|

| 中年期 | 36-55岁 | 家庭、事业 |
