# NeuralShelf - 神经网络文件整理系统

## 核心改进（v2.0）

| 改进点 | 说明 |
|--------|------|
| **内容导向整理** | 按内容相关性分组，替代类型分类 |
| **智能去重** | MD5哈希检测，识别完全相同的文件 |
| **项目结构保护** | 检测项目标记，整体保留不打散 |
| **一键撤销** | 备份快照+操作日志，可恢复 |

---

## 四阶段整理流程

详见 `docs/整理方法论.md`

```
阶段一：神经感知分析 → 分析文件内容、类型、关联
阶段二：神经决策分类 → 智能决定整理方案
阶段三：神经执行整理 → 安全执行整理操作
阶段四：神经恢复保障 → 提供一键撤销功能
```

## 快速开始

### 命令行使用

```bash
# 分析指定目录
neuralshelf analyze /path/to/directory

# 执行整理
neuralshelf organize /path/to/directory

# 查看整理方案（不执行）
neuralshelf preview /path/to/directory

# 一键撤销
neuralshelf undo
```

### 自然语言使用

```
用户："帮我整理一下Downloads文件夹"
AI：
1. 分析Downloads目录内容
2. 输出整理方案
3. 等待用户确认
4. 执行整理并生成报告
```

## 安全保护机制

详见 `docs/安全机制.md`

| 机制 | 说明 |
|------|------|
| **不删除原则** | 只移动，不删除任何文件 |
| **先备份后操作** | 操作前完整备份 |
| **完整日志** | 每一步都有日志可追溯 |
| **一键撤销** | 支持恢复到任意时间点 |

---

## 项目结构保护

自动识别项目标记：

| 标记 | 说明 |
|------|------|
| `.git/` | Git仓库 |
| `package.json` | Node.js项目 |
| `.venv/` | Python虚拟环境 |

项目文件整体保留，不打散结构。

## 智能去重

| 方法 | 说明 |
|------|------|
| **MD5哈希** | 快速识别完全相同的文件 |
| **相似度检测** | 图片内容相似度分析 |
| **去重报告** | 统计节省空间 |
