# 配置选项

NeuralShelf 的配置参数说明。

---

## 基础配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `backup_dir` | 备份目录路径 | `.neuralshelf_backup` |
| `log_level` | 日志详细程度 | `INFO` |
| `max_backup_size` | 最大备份占用空间 | `5GB` |
| `auto_cleanup` | 自动清理旧备份 | `true` |
| `cleanup_days` | 保留备份天数 | `30` |

---

## 文件类型映射

### 默认类型映射

| 类型 | 扩展名 | 目标文件夹 |
|------|--------|-----------|
| 文档 | .pdf, .doc, .docx, .txt | Documents/ |
| 图片 | .jpg, .png, .gif, .bmp | Pictures/ |
| 视频 | .mp4, .mov, .avi | Videos/ |
| 音频 | .mp3, .wav, .flac | Music/ |
| 代码 | .py, .js, .cpp, .java | Code/ |
| 压缩 | .zip, .rar, .7z | Archives/ |

### 自定义类型映射

```yaml
# config.yaml
type_mapping:
  custom_type:
    extensions: [.custom1, .custom2]
    target: CustomFolder/
```

## 项目识别配置

### 默认项目标记

```yaml
project_markers:
  - .git
  - .idea
  - package.json
  - pom.xml
  - .venv
  - CMakeLists.txt
  - requirements.txt
```

### 自定义项目标记

```yaml
# config.yaml
project_markers:
  - .myproject
  - myproject.config
```

---

## 去重配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `enable_dedup` | 是否启用去重检测 | `true` |
| `hash_method` | 奇哈希算法 | `md5` |
| `similarity_threshold` | 相似度阈值（图片） | `0.9` |

---

## 报告配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `report_format` | 报告格式 | `markdown` |
| `report_dir` | 报告输出目录 | `reports/` |
| `include_stats` | 包含统计数据 | `true` |

---

## 配置文件位置

```
~/.neuralshelf/config.yaml    # 全局配置
./.neuralshelf/config.yaml    # 项目配置（优先级更高）
```

---

## 配置优先级

```
项目配置 > 全局配置 > 默认值
```
