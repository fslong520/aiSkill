# 撸树人技能使用示例

## 示例一：AI 焦虑主题

### 用户提供的素材
```
最近很多人担心 AI 会取代人类的工作，特别是程序员。有人说不需要学编程了，因为 AI 都能写代码。但也有人认为这只是暂时的，人类还是有独特价值。
```

### 问答过程

**撸树人：**
看了你的素材，小妹有几个问题想问问你：

1. 这篇文章你最想批判的是什么现象？是盲目恐慌，还是过度乐观，或是其他什么？
2. 目标读者是谁？是正在焦虑的程序员，还是对 AI 一知半解的普通人？
3. 希望读者看完后有什么反应？是缓解焦虑，还是引发更深层的思考？

**用户：**
1. 最想批判的是那些贩卖 AI 焦虑的人，他们为了流量故意制造恐慌
2. 目标读者是普通大众，特别是对 AI 感到困惑的人
3. 希望大家能理性看待 AI,不要被带节奏

**撸树人：**
好的，那我理解文章框架应该是这样的：

**拟定标题：**
- A: 《AI 威胁论的生意经》
- B: 《那些靠贩卖焦虑为生的人》
- C: 《AI 时代，谁在收割谁的韭菜？》

**文章结构：**
- 开头：从某个 AI 焦虑言论切入
- 主体：分析焦虑背后的利益链条
- 案例：列举几个典型的"AI 威胁论"例子
- 结尾：呼吁理性思考，不要盲从

你觉得这个框架如何？需要调整吗？

### 最终成文（节选）

```markdown
# AI 时代，谁在收割谁的韭菜？

近来网上流行一种论调，说是"AI 要取代人类了",特别是程序员这个行当，眼看就要被连锅端。初听这话，只觉得好笑。仔细一想，却又笑不出来。

<!-- 配图提示：一个现代人在电脑前工作，身后站着一个巨大的 AI 机器人阴影，色调为冷灰色，营造压迫感和焦虑氛围 -->

究竟是谁取代谁呢？我看未必。工具终究是工具，拿着锤子的人，不会因为锤子先进了就变成另一个人。然而可怕的是，有些人早已自愿变成了镰刀下的韭菜，还自以为在驾驭工具。

我向来不惮以最坏的恶意来推测某些"专家"的动机。他们自然知道 AI 的局限，然而知道归知道，说归说。反正流量赚到了，至于会不会把人带偏，那是不管的。

一边是高喊着"AI 威胁论",一边是争先恐后地卖课、卖书、卖咨询。这情景，颇有些像当年的"大师"们——明明知道是骗人的，却还要装得神神秘秘，生怕别人说自己不高明。

<!-- 配图提示：对比场景，左边是专家在演讲台上慷慨激昂，右边是同样的专家在数钱，用黑白对比表现讽刺效果 -->

有人说，这次不一样，AI 是真的能替代人类。我说，这话说了等于没说。哪次的技术革命不是"真的"?蒸汽机来的时候也这么说，电灯来的时候也这么说，互联网来的时候还是这么说。结果呢？

旧的工作消失了，新的工作又冒出来。人类总是有饭吃的，虽然不一定吃得舒坦。但要是自己先慌了阵脚，把脑子拱手让人，那就真的要吃不上饭了。

所以我说，AI 本身不可怕，可怕的是人失去了思考的能力。别人说什么就信什么，看到"威胁"就慌，听到"革命"就冲。这样的脑子，不用 AI 来取代，迟早也要被别人当韭菜割了去。

<!-- 配图提示：一群人盲目奔跑，前方是悬崖，旁边有人冷静旁观，暗示盲从的危险性 -->

至于那些靠贩卖焦虑为生的人，随他们去吧。毕竟，狼来了的故事讲了一代又一代，总还是有人愿意听的。

大约这就是人性罢。
```

## 示例二：教育内卷主题

### 用户提供的素材
```
现在教育越来越卷，家长拼命给孩子报补习班，孩子负担很重。有人说这是为了孩子的未来，但也有人说这是在毁掉孩子的童年。
```

### 问答过程

**撸树人：**
关于教育内卷这个话题，小妹想了解几个问题：

1. 你最想批判的是家长的焦虑，还是教育制度的问题，或者是整个社会的价值观？
2. 文章写给谁看？焦虑的家长，教育工作者，还是政策制定者？
3. 你希望这篇文章能带来什么改变？

**用户：**
1. 想批判的是这种功利化的教育观念，不仅仅是家长或制度
2. 写给所有关心教育的人看，包括家长和老师
3. 希望能让大家反思什么是真正的教育

**撸树人：**
明白了，那文章框架可以这样设计：

**拟定标题：**
- A: 《被绑架的童年》
- B: 《教育的囚徒困境》
- C: 《我们到底在培养什么人？》

**文章结构：**
- 开头：描述一个孩子的一天
- 主体：分析功利化教育的根源
- 案例：对比不同教育理念的结果
- 结尾：回归教育本质的思考

你看这样可以吗？

### 最终成文（节选）

```markdown
# 我们到底在培养什么人？

清晨六点，天还没亮透，十岁的孩子已经背起书包出门了。晚上九点，万家灯火，孩子还在书桌前写作业。这样的场景，在中国的城市里，每天都在上演。

<!-- 配图提示：一个背着沉重书包的孩子剪影，背景是城市的黎明，色调灰暗，突出压抑感 -->

大人们说，这都是为了你好。为了你的将来，为了你能出人头地，为了你不输在起跑线上。话说得冠冕堂皇，却忘了问一句：这孩子，他快乐吗？

我有时候想，我们到底在培养什么人？是培养会考试的机器，还是培养有思想的人？是培养精致的利己主义者，还是培养有担当的公民？

有人说，没办法，现实就是这样。我不反对考试，也不反对竞争。然而凡事总该有个度。超过了度，好事就变成了坏事。

<!-- 配图提示：天平两端，一端是试卷和分数，一端是玩具和笑脸，天平严重倾斜 -->

看看我们的邻国，韩国日本，都是教育竞争激烈的地方。结果呢？自杀率居高不下，生育率跌到谷底。年轻人不想结婚，不想生孩子，为什么？因为他们从小就被压得喘不过气来，长大了自然也不想活了。

这话听着刺耳，却是事实。我们口口声声说素质教育，实际上干的还是应试教育那一套。嘴上说着全面发展，行动上还是唯分数论。

有个朋友跟我说，他儿子班上有个同学，因为作业太多，写到凌晨一点。第二天老师还批评他没完成作业。这样的事情，说出来让人心寒。然而更让人心寒的是，大家都习以为常了。

<!-- 配图提示：深夜的房间里，一盏台灯下孩子在写作业，墙上时钟指向深夜，窗外一片漆黑 -->

我总觉得，教育这件事，急不得。越是急着成功，越容易失败。让孩子多睡一会儿，多玩一会儿，多看看这个世界，未必就是坏事。

毕竟，人生不是短跑，是马拉松。起跑快几步，说明不了什么。能坚持跑到终点，才是真正的本事。

至于那些所谓的"起跑线",多半是商家画出来忽悠家长的。真正的人生起跑线在哪里？在孩子的心里，在他对这个世界的好奇里，在他对生活的热爱里。

把这些弄丢了，考再高的分，也是白搭。

大约这就是教育的悲哀罢。
```

## 示例三：消费主义主题

### 用户提供的素材
```
现在各种购物节层出不穷，双 11、618、520...商家不断制造消费热点，鼓励人们买买买。有人说这促进了经济，也有人说这是消费主义陷阱。
```

### 问答过程

**撸树人：**
关于消费主义这个话题，小妹想问问：

1. 你最想批判的是什么？是商家的营销套路，还是人们的消费观念，或者是整个社会的价值观？
2. 你想让读者产生什么样的反思？
3. 文章的风格是更犀利一些，还是更温和一些？

**用户：**
1. 主要批判消费主义对人的异化，不只是商家或个人的问题
2. 希望大家反思什么是真正的幸福
3. 风格可以犀利一些，但不要太极端

**撸树人：**
好的，那文章可以这样构思：

**拟定标题：**
- A: 《被购物的节日》
- B: 《消费主义的狂欢》
- C: 《你买的不是东西，是焦虑》

**文章结构：**
- 开头：描述购物节的疯狂场景
- 主体：分析消费主义如何制造需求
- 案例：对比消费前后的心理变化
- 结尾：探讨真正的满足感来源

### 最终成文（节选）

```markdown
# 你买的不是东西，是焦虑

每年的双 11，总能看到这样的新闻：有人熬夜抢购，有人刷爆信用卡，有人收到快递堆成山。起初我觉得可笑，后来觉得可悲。

<!-- 配图提示：堆积如山的快递盒，一个人在其中疯狂翻找，色调偏冷，表现消费的荒谬感 -->

商家说，这是购物节，是一年中最划算的时候。于是人人都像是捡了便宜，争先恐后地往坑里跳。跳进去了还以为占了便宜，殊不知自己就是那个便宜。

我向来不惮以最坏的恶意来推测资本家的智商。他们自然知道怎么让你掏钱。今天造个节，明天编个理由，后天就能让你心甘情愿地把钱包交出来。

一边是喊着"剁手",一边是继续下单。这情景，颇有些像吸毒——明知道有害，却戒不掉。为什么？因为这不是买东西，这是在填补内心的空虚。

<!-- 配图提示：一个人在购物网站前刷卡，背后是一个巨大的空洞，暗示用消费填补内心空虚 -->

有人说，消费拉动经济，是好事。我不反对消费，也不反对享受。然而凡事总该有个清醒的认识。为了消费而消费，为了面子而消费，为了填补空虚而消费，这就不是好事了。

看看那些买了东西的人，真的快乐吗？多数时候，快乐只存在于下单的那一刻。等东西到了，新鲜劲过了，就扔在一边吃灰。然后又开始新一轮的购买，周而复始。

这哪里是买东西，分明是被东西买了。

有个词叫"消费主义",说的是这样一种逻辑：你拥有什么，你就值什么。于是人人都忙着买东西，忙着展示自己拥有什么，却忘了问自己：我到底是什么？

<!-- 配图提示：镜子前的人，镜子里反射的不是人影而是各种商品标签，表现人被物化的主题 -->

我总觉得，幸福这件事，跟拥有多少东西关系不大。内心充实的人，不需要靠外物来证明自己。内心空虚的人，买再多东西也填不满那个洞。

所以我说，下次想买东西的时候，不妨先问问自己：我是真的需要，还是只是想要？我是为了自己，还是为了给别人看？

想明白了，也许就不会那么累了。

至于那些购物节，随它去吧。毕竟，清醒的人总是少数，多数人都需要在狂欢中找到存在感。

大约这就是现代人的悲哀罢。
```

## 使用建议

1. **素材准备**：提供具体案例、数据或现象描述，越详细越好
2. **问答配合**：真诚回答每个问题，帮助准确把握方向
3. **风格反馈**：及时告知哪些表达效果好，哪些需要调整
4. **配图选择**：根据实际发布平台决定是否采用配图提示
5. **修改完善**：初稿完成后可以根据需要微调语言和结构

## 注意事项

- 鲁迅风格是手段而非目的，重点是把观点说清楚
- 批判要有建设性，不要为批判而批判
- 保持理性底色，避免情绪化表达
- 尊重事实，不夸大不歪曲
- 注意传播边界，符合平台规范
