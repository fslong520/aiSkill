# 视觉风格指南

## 色调

| 类型 | 示例 |
|------|------|
| 仙侠 | 青绿冷色 + 银白剑光 |
| 治愈 | 暖黄柔光 |
| 暗黑史诗 | 墨黑 + 朱红 + 金色 |

---

## 光影

| 类型 | 特征 |
|------|------|
| 自然光 | 日光、月光、烛光 |
| 人工光 | 霓虹、聚光灯 |
| 方向 | 侧光、逆光、顶光 |
| 阴影 | 柔和、硬朗、剪影 |

---

## 画面质感

| 类型 | 特征 |
|------|------|
| 电影感 | 宽银幕、胶片颗粒、专业调色 |
| CG/动画 | 3D渲染、2D手绘 |
| 写实 | 真实光影、纪录片风格 |
| 水墨 | 中国传统水墨画效果 |

---

## 色彩心理

| 色彩 | 情感 | 适用 |
|------|------|------|
| 红 | 热情、危险、力量 | 战斗、爱情 |
| 蓝 | 冷静、科技、忧郁 | 科幻、悬疑 |
| 绿 | 自然、生机、神秘 | 仙侠、奇幻 |
| 金 | 神圣、尊贵 | 史诗、神话 |
| 紫 | 神秘、魔幻 | 魔法、暗黑 |
| 黑 | 神秘、恐惧、权威 | 悬疑、暗黑 |
| 白 | 纯洁、神圣 | 仙侠、治愈 |

---

## 光影氛围

| 氛围 | 光影特征 | 适用 |
|------|----------|------|
| 明亮欢快 | 暖光、高调 | 喜剧、治愈 |
| 紧张悬疑 | 暗调、侧光、强对比 | 悬疑、恐怖 |
| 浪漫唯美 | 逆光、柔焦 | 爱情、唯美 |
| 史诗壮观 | 广角、黄金时段、逆光 | 史诗、战争 |
| 暗黑压迫 | 低调、底光、剪影 | 暗黑、恐怖 |
| 赛博朋克 | 霓虹、反射、湿润地面 | 科幻、未来 |

---

## 避免元素

- 平台限制：如不支持写实真人脸部
- 版权限制：避免特定人物/角色
- 风格冲突：与主题不符的元素
