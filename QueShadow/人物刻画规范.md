# 人物刻画规范

## 三视图

角色一致性的核心：正面 + 侧面 + 背面。

| 视角 | 要点 |
|------|------|
| 正面 | 五官清晰、表情自然 |
| 侧面 | 轮廓清晰、发型侧面 |
| 背面 | 发型后部、服装背部 |

### 生成模板

```
【正面】[角色名]-front, [面部], [发型], [服装], front view
【侧面】[角色名]-side, [轮廓], [发型侧面], side view
【背面】[角色名]-back, [发型后部], [服装背部], back view
```

### 一致性检查

发型、服装、配饰、肤色、光照，三视图要统一。

---

## 五维刻画

| 维度 | 要素 |
|------|------|
| 外貌 | 身材、发型、面部、服装、配饰 |
| 性格 | 动作体现、台词风格 |
| 动作 | 姿态、移动、战斗风格 |
| 神态 | 眼神、微表情、气场 |
| 气质 | 整体印象 |

---

## 角色卡模板

```
【角色卡】
姓名：[名称]
身份：[定位]
外貌：身材 + 发型 + 面部 + 服装 + 配饰
性格：核心性格 + 行为习惯
动作：站姿 + 移动 + 战斗风格
神态：眼神 + 表情 + 气场
气质：整体印象
```

---

## 一致性锚点

| 类型 | 示例 |
|------|------|
| 服装锚点 | 红色围巾、金色手镯 |
| 发型锚点 | 披肩长发、马尾边发卡 |
| 动作锚点 | 撩头发、托腮 |
| 表情锚点 | 微微皱眉、嘴角上扬 |

---

## 多角色场景

- 主角占画面 60% 以上
- 不同角色用不同视角
- 群像用全景+中景
- 正邪对立、性格反差
