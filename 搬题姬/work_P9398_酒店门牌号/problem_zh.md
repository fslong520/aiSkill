# 酒店门牌号

#### 题目描述
一家酒店有 100 到 999 号共 900 个房间。酒店老板认为，门牌号的各位数字之和恰好等于某个数字 k 的房间是"风水房"。现在他想知道，对于一个给定的 k，酒店里一共有多少个"风水房"？

#### 输入格式
一行，一个正整数 k（1 ≤ k ≤ 27）。

#### 输出格式
一个整数，表示"风水房"的数量。

#### 样例输入 #1
```
6
```

#### 样例输出 #1
```
21
```

#### 数据范围
1 ≤ k ≤ 27

#### 知识点与难度
本题涉及的知识点从属于 **GESP三级**，难度等级：**⭐⭐**。
