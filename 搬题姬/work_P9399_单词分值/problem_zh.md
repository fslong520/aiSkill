# 单词分值

#### 题目描述
英语课上，老师发明了一种给单词打分的方法：
- 小写字母按字母表位置计正分：a=1、b=2、c=3 …… z=26
- 大写字母按 ASCII 码值计负分：A=-65、B=-66、C=-67 ……

一个单词的分值 = 每个字母的得分之和。例如 `Hello`：
- H = -72，e = 5，l = 12，l = 12，o = 15
- 总分 = -72 + 5 + 12 + 12 + 15 = -28

给定一个包含大小写字母的字符串，请计算它的分值。

#### 输入格式
一行，一个只包含大小写英文字母的字符串 s（长度 ≤ 1000）。

#### 输出格式
一个整数，表示该单词的分值。

#### 样例输入 #1
```
Hello
```

#### 样例输出 #1
```
-28
```

#### 数据范围
字符串长度 ≤ 1000
