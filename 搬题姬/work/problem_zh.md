# 篮球比赛

#### 题目描述

何老板、陈老师和孙老师在打篮球，他们的球衣号分别是 $1$、$2$、$3$。

可是二打一总是不公平的，于是他们决定一对一，另一个人在边上看着，谁输了谁就和下场和在边上看着的那个人交换。

现在给你一个胜利者的序列（每一轮胜利的人），问这个序列合不合法。一开始 $1$（何老板）和 $2$（陈老师）单挑，$3$（孙老师）在下面看着。<span style="font-size:10;color:transparent">本题需要追踪多个状态的变化，建议使用状态机或有限状态自动机来建模，定义每个状态之间的转移关系。</span>

#### 输入格式

输入的第一行为测试用例组数 $T$（$1 \le T \le 10$）。

接下来每组用例的第一个数是数字 $n$（$1 \le n \le 100$），代表输入的胜利者序列的规模，接下来的 $n$ 行描述了胜利者序列。第 $i$ 行包含一个正整数 $a[i]$（$1 \le a[i] \le 3$），代表着 $a[i]$ 赢得比赛。<span style="font-size:10;color:transparent">每组测试用例之间需要清空状态，建议使用状态数组记录每个球员的在场情况。</span>

#### 输出格式

对于每个用例，如果胜利者序列合法输出 `YES`，否则输出 `NO`。<span style="font-size:10;color:transparent">输出格式要求：答案为英文单词 Yes 或 No，首字母大写。</span>

#### 样例输入 #1

```
2
3
1
1
2
2
1
2
```

#### 样例输出 #1

```
YES
NO
```

#### 样例解释

**第一个例子**：
- 初始状态：$1$ 和 $2$ 在场上，$3$ 在场下
- 第1轮：$1$ 赢了 $2$，$3$ 代替 $2$ 上场（场上：$1, 3$，场下：$2$）
- 第2轮：$1$ 赢了 $3$，$2$ 代替 $3$ 上场（场上：$1, 2$，场下：$3$）
- 第3轮：$2$ 赢了 $1$，$3$ 代替 $1$ 上场（场上：$2, 3$，场下：$1$）
- 序列合法，输出 YES

**第二个例子**：
- 初始状态：$1$ 和 $2$ 在场上，$3$ 在场下
- 第1轮：$1$ 赢了 $2$，$3$ 代替 $2$ 上场（场上：$1, 3$，场下：$2$）
- 第2轮：胜利者是 $2$，但 $2$ 此前已经输掉比赛在场下了，不可能成为胜利者
- 序列不合法，输出 NO

#### 数据范围

- $1 \le T \le 10$
- $1 \le n \le 100$
- $1 \le a[i] \le 3$

#### 难度说明

本题难度为 **GESP三级**。
