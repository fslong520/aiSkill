# 对称树判断

#### 题目描述
给一棵二叉树的前序遍历和中序遍历，判断它是否是**对称树**。

对称树是指：以根节点为中心，左子树和右子树互为镜像。例如：
```
对称树：          非对称树：
    A                 A
   / \               / \
  B   B             B   C
 / \ / \           / \   \
D  E E  D         D   E   F
```

#### 输入格式
共两行，均为大写字母组成的字符串。
- 第一行：前序遍历
- 第二行：中序遍历

约定树结点用大写字母表示，节点个数 $n \le 8$。

#### 输出格式
共一行，如果是对称树输出 `Yes`，否则输出 `No`。

#### 样例输入 #1
```
ABDEBED
DBEAEBD
```

#### 样例输出 #1
```
Yes
```

#### 样例解释
对应的树结构：
```
        A
       / \
      B   B
     / \ / \
    D  E E  D
```
左子树和右子树互为镜像，所以是对称树。

#### 数据范围
- $n \le 8$，节点用大写字母表示

本题涉及的知识点从属于 **GESP六级**（二叉树、树的遍历、递归），难度等级：**提高/省选-**。
