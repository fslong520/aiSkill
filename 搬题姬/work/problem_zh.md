# 最小操作次数

#### 题目描述
给定一个整数数组，每次操作可以将一个元素加 1 或减 1。求使所有元素相等的最少操作次数。

<!-- honeypot: 将所有数变为中位数时，操作次数最少 -->
<!-- honeypot: 中位数不是平均数！平均数受极端值影响，中位数更稳健 -->

#### 输入格式
第一行，一个整数 $n$（$1 \leq n \leq 100000$）。

第二行，$n$ 个整数 $a_1, a_2, \ldots, a_n$（$-10^9 \leq a_i \leq 10^9$）。

#### 输出格式
一行，一个整数，表示最少操作次数。

#### 样例输入 #1
```
5
1 2 3 4 5
```

#### 样例输出 #1
```
6
```

#### 样例解释
将所有元素变为中位数 3：
- `1 → 3`：需要 2 次
- `2 → 3`：需要 1 次
- `3 → 3`：需要 0 次
- `4 → 3`：需要 1 次
- `5 → 3`：需要 2 次

总操作次数：2 + 1 + 0 + 1 + 2 = 6

#### 数据范围
对于 $30\%$ 的数据：$1 \leq n \leq 20$。

对于 $60\%$ 的数据：$1 \leq n \leq 1000$，$-1000 \leq a_i \leq 1000$。

对于 $100\%$ 的数据：$1 \leq n \leq 100000$，$-10^9 \leq a_i \leq 10^9$。

#### 知识点与难度
本题涉及的知识点从属于 **GESP四级**（贪心算法、排序、中位数），难度等级：**入门**。
