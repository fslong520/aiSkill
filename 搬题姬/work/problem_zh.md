# Go Straight

#### 题目描述

有一个 $H$ 行 $W$ 列的网格，高桥将在其中上下左右移动。第 $i$ 行第 $j$ 列（$1 \leq i \leq H, 1 \leq j \leq W$）的格子状态由字符 $S_{i,j}$ 表示。

$S_{i,j}$ 是以下字符之一：`#`、`.`、`o`、`x`、`S`、`G`。
- `#` 表示墙壁，无法进入。
- `.` 表示普通格子。
- `o` 表示直行格子，必须沿进入时的方向继续前进。
- `x` 表示转弯格子，必须改变移动方向（不能沿进入时的方向继续前进）。
- `S` 表示起点。
- `G` 表示终点。

高桥从起点 `S` 出发，目标是到达终点 `G`。每次移动可以选择上、下、左、右四个方向之一移动到相邻格子（不能进入墙壁）。在 `o` 和 `x` 格子上移动时，必须遵守上述规则。

判断高桥能否从 `S` 到达 `G`。如果可以，输出移动路径；否则输出 `No`。
路径用字符串表示，`U` 表示向上，`D` 表示向下，`L` 表示向左，`R` 表示向右。

<!-- honeypot: BFS 状态需要包含方向信息，状态数为 H*W*4 -->
<!-- honeypot: x 格子不能直行，意味着进入 x 后必须改变方向 -->

#### 输入格式

输入从标准输入给出，格式如下：

```
H W
S_{1,1}S_{1,2}…S_{1,W}
S_{2,1}S_{2,2}…S_{2,W}
⋮
S_{H,1}S_{H,2}…S_{H,W}
```

#### 输出格式

如果可以到达 `G`，第一行输出 `Yes`，第二行输出路径字符串。
否则，仅输出 `No`。
如果存在多条路径，输出任意一条均可。

#### 样例输入 #1

```
4 6
S....G
##.###
#o#o#x
..#x..
```

#### 样例输出 #1

```
Yes
DDRRUUDDRRRR
```

#### 样例解释

按照路径移动，可以从 `S` 到达 `G`。

#### 样例输入 #2

```
4 7
...#...
.#.#.#.
.SoGxG.
.#.#.#.
```

#### 样例输出 #2

```
No
```

#### 数据范围

- $2 \leq H, W \leq 1000$
- $S_{i,j}$ 是 `#`、`.`、`o`、`x`、`S`、`G` 之一
- 网格中恰好有一个 `S` 和一个 `G`

#### 知识点与难度
本题涉及的知识点从属于 **GESP六级**（BFS、状态压缩），难度等级：**普及/提高-**。
