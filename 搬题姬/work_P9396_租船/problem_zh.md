# 租船

#### 题目描述
春游去划船，全班 n 个同学，每条船最多坐 k 个人。问至少需要租几条船？

#### 输入格式
一行，两个正整数 n 和 k（1 ≤ k ≤ n ≤ 10000），分别表示总人数和每条船最多能坐的人数。

#### 输出格式
一个整数，表示至少需要租的船数。

#### 样例输入 #1
```
23 5
```

#### 样例输出 #1
```
5
```

#### 数据范围
1 ≤ k ≤ n ≤ 10000

#### 知识点与难度
本题涉及的知识点从属于 **GESP一级**，难度等级：**⭐**。
