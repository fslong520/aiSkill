# Concat Power of 2

<div class="water">

我们称满足以下条件的正整数为**好整数**：

条件：它可以通过选择一个或多个2的幂次（$1, 2, 4, 8, 16, \ldots$）（允许重复和重新排序），将它们作为字符串拼接起来，然后将结果解释为一个整数而得到。

求第 $N$ 小的好整数。保证第 $N$ 小的好整数不超过 $10^9$。

## 输入格式

一个正整数 $N$。

## 输出格式

输出答案。

## 样例

### 样例输入 1
```
10
```

### 样例输出 1
```
21
```

### 样例解释 1
从小到大列出好整数：$1, 2, 4, 8, 11, 12, 14, 16, 18, 21, \ldots$

### 样例输入 2
```
69
```

### 样例输出 2
```
328
```

### 样例输入 3
```
1099898
```

### 样例输出 3
```
819264512
```

## 数据范围

- $N$ 是正整数
- 第 $N$ 小的好整数不超过 $10^9$
