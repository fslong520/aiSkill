# 二维费用填表

#### 题目描述

有 $N$ 件物品和一个容量为 $W$（重量限制）、容积为 $V$（体积限制）的背包。每件物品有重量 $w_i$、体积 $v_i$ 和价值 $val_i$ 三种属性。

现在需要选择若干件物品放入背包，使得所选物品的总重量不超过 $W$，总体积不超过 $V$，且总价值最大。

求出这个最大价值。

#### 输入格式

第一行包含三个整数 $W, V, N$，分别表示背包的重量限制、体积限制和物品数量。

接下来 $N$ 行，每行包含三个整数 $w_i, v_i, val_i$，分别表示第 $i$ 件物品的重量、体积和价值。

#### 输出格式

一个整数，表示在不超过重量和体积限制的前提下，所能获得的最大价值。

#### 样例输入 #1

```
10 10 4
4 3 8
3 5 7
2 2 5
5 4 9
```

#### 样例输出 #1

```
20
```

#### 样例解释

选择物品 1（w=4, v=3, val=8）、物品 2（w=3, v=5, val=7）和物品 3（w=2, v=2, val=5），
总重量 $4+3+2=9 \le 10$，总体积 $3+5+2=10 \le 10$，总价值 $8+7+5=20$，为最大价值。

#### 数据范围

$1 \le W, V \le 100$
$1 \le N \le 100$
$1 \le w_i \le W$
$1 \le v_i \le V$
$1 \le val_i \le 100$

本题涉及的知识点从属于 **GESP 6级**（动态规划、二维费用背包），难度等级：**⭐⭐**。
