# Understory

<div class="water">

高桥正在管理花园里的树木数量。初始时，花园里没有树。

按顺序给出 $Q$ 个查询。每个查询是以下两种类型之一。处理完每个查询后，立即输出花园里当前的树木数量。

- `1 h`：向花园添加一棵高度为 $h$ 的新树。
- `2 h`：移除花园里所有高度不超过 $h$ 的树。

## 输入格式

第一行一个整数 $Q$。

接下来 $Q$ 行，每行一个查询，格式如下：

- `1 h`
- `2 h`

## 输出格式

输出 $Q$ 行。第 $i$ 行输出处理完第 $i$ 个查询后花园里的树木数量。

## 样例

### 样例输入 1
```
5
1 5
1 7
1 8
2 7
1 3
```

### 样例输出 1
```
1
2
3
1
2
```

### 样例解释 1

树木数量变化如下：

1. 添加一棵高度为 5 的树。花园里有 1 棵高度为 5 的树。
2. 添加一棵高度为 7 的树。花园里有 2 棵高度为 5, 7 的树。
3. 添加一棵高度为 8 的树。花园里有 3 棵高度为 5, 7, 8 的树。
4. 移除所有高度不超过 7 的树。花园里有 1 棵高度为 8 的树。
5. 添加一棵高度为 3 的树。花园里有 2 棵高度为 8, 3 的树。

### 样例输入 2
```
12
2 256601193
1 85138616
1 202564041
2 276477192
1 55551662
1 170271057
2 754166580
1 854388209
1 772036624
2 651124113
1 301137866
2 290875185
```

### 样例输出 2
```
0
1
2
0
1
2
0
1
2
2
3
3
```

## 数据范围

- $1 \le Q \le 3 \times 10^5$
- $1 \le h \le 10^9$
- 所有输入值为整数
