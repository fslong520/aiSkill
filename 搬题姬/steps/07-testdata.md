## 测试数据要求

生成的 25 组测试数据必须覆盖以下类型：

```cpp
void test(int case_num, ofstream& fout) {
    if (case_num <= 2) {
        // 第1-2组: 样例数据（直接复制题目样例）
    }
    else if (case_num <= 5) {
        // 第3-5组: 小规模随机数据（最小 N，验证基本功能）
    }
    else if (case_num <= 8) {
        // 第6-8组: 特殊性质数据
        // - 单调性（递增/递减）
        // - 所有值相同
        // - 特殊数据结构（如全0、全1等）
    }
    else if (case_num <= 11) {
        // 第9-11组: Hack 数据（针对常见错误写法）
        // - 溢出测试（int 溢出 → 需用 long long）
        // - 边界条件（N=1, N=max）
        // - 特殊输入使错误算法失败
    }
    else if (case_num <= 15) {
        // 第12-15组: 中等规模数据（验证效率）
    }
    else if (case_num <= 20) {
        // 第16-20组: 大规模数据（压力测试）
    }
    else {
        // 第21-25组: 随机回归测试
    }
}
```

## 特殊性质数据示例

根据题目的不同，特殊性质可能包括：

| 性质 | 说明 | 针对的错误 |
|------|------|-----------|
| 单调性 | 输入有序（递增/递减） | 错误排序算法 |
| 所有值相同 | 数组所有元素相等 | 未处理重复值 |
| 二分图性质 | 特定图结构 | 错误图算法 |
| 素数密集 | 大量素数 | 错误素数判断 |
| 极端值 | 最大/最小可能值 | 溢出错误 |

## Hack 数据设计

针对常见错误写法设计数据：

| 常见错误 | Hack 数据特征 |
|---------|--------------|
| int 溢出 | 使用接近限制的大数 |
| 边界漏判 | N=0, N=1, N=max 等 |
| 精度错误 | 需要精确计算的小数 |
| 超时错误 | 迫使 O(n²) 算法超时 |
| 错误贪心 | 使贪心策略失败的数据 |
