# Step 9: 自文件取题面

## 目标

自用户所供本地文件（如 .md、.txt、截图等）取题目信息。

## 支文件类

| 类 | 措 |
|------|---------|
| .md / .txt | 直 `read_file` 读 |
| 图片 (.png/.jpg) | 用 OCR 或视觉模型取文 |
| HTML | 析 DOM 取题面 |

## 步骤

1. 确文件路径存
2. 读文件内容
3. 取要信息：
   - 题名
   - 题述
   - 入格式
   - 出格式
   - 样例
   - 数据范围
4. 生 `{WORK_DIR}/problem_zh.md`（参 04-problem.md 格式）
