# Step 9: 从文件提取题面

## 目标

从用户提供的本地文件（如 .md、.txt、截图等）提取题目信息。

## 支持的文件类型

| 类型 | 处理方式 |
|------|---------|
| .md / .txt | 直接 `read_file` 读取 |
| 图片 (.png/.jpg) | 使用 OCR 或视觉模型提取文字 |
| HTML | 解析 DOM 提取题面 |

## 步骤

1. 确认文件路径存在
2. 读取文件内容
3. 提取关键信息：
   - 题目名称
   - 题目描述
   - 输入格式
   - 输出格式
   - 样例
   - 数据范围
4. 生成 `work/problem_zh.md`（参考 04-problem.md 格式）
