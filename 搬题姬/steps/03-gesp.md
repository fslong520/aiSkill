# Step 3: GESP 知识点等级判定

## 目标

判定题目知识点等级（GESP 1-8级）和难度等级。

## 等级表

| 级别 | 算法 | 数据结构 |
|------|------|----------|
| 一级 | 无 | 基本类型 |
| 二级 | 无 | 无新增 |
| 三级 | 枚举、模拟 | 一维数组、字符串 |
| 四级 | 递推、排序 | 二维数组、结构体 |
| 五级 | 二分、递归、贪心、分治 | 链表 |
| 六级 | DFS、BFS、简单DP | 树、栈、队列 |
| 七级 | 复杂DP、图遍历 | 图、哈希表 |
| 八级 | 最小生成树、最短路 | 无新增 |

## 判定规则

1. 取知识点最高等级
2. 算法优先，数据结构辅助
3. 知识点等级 ≠ 题目难度

## 难度等级

| 难度 | 特征 |
|------|------|
| 入门 | 直接应用知识点 |
| 普及- | 简单变形 |
| 普及/提高- | 知识点组合 |
| 提高/省选- | 复杂问题 |
| 省选/NOI | 高难度 |

## 快速判定

```
数据结构：基本→一二 | 数组→三 | 二维→四 | 链表→五 | 树→六 | 图→七
算法：无→一二 | 枚举→三 | 排序→四 | 二分贪心→五 | 搜索DP→六 | 复杂DP→七 | 最短路→八
```

## 常见题目

| 类型 | 级别 |
|------|------|
| A+B、判断奇偶 | 一级 |
| 数组统计、进制转换 | 三级 |
| 矩阵、排序 | 四级 |
| 二分、高精度 | 五级 |
| 树遍历、迷宫 | 六级 |
| LIS/LCS | 七级 |
| 最短路 | 八级 |
