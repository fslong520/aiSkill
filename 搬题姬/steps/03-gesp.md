# Step 3: GESP 知识点等级判定

## 目标

定题目知识点等级（GESP 1-8级）与难度等级（二者独立评估）。

## 原则

知识点等级 ≠ 题目难度。同知识点可出简题，亦可出难题。

**先定级，后定难**：先按所涉知识点定GESP级别，后独立评估题目难度。

---

## 第一步：定级（知识点范围）

### 等级表（2026年最新大纲）

| 级别 | 核心算法 | 数据结构 | 数学/其他 |
|------|---------|----------|----------|
| 一级 | 顺序/分支/循环 | 基本类型（int/float/char/bool） | 计算机基础、基本运算 |
| 二级 | 多层分支/循环嵌套 | 无新增 | ASCII编码、类型转换、数学函数（abs/sqrt/max/min） |
| 三级 | 枚举法、模拟法 | 一维数组（不含变长）、字符串 | 进制转换、位运算、数据编码 |
| 四级 | 递推、排序（冒泡/插入/选择） | 二维数组、结构体、指针 | 函数、文件操作、异常处理 |
| 五级 | 递归、分治（快排/归并）、贪心、二分 | 链表（单/双/循环） | 高精度（数组模拟）、数论（GCD/素数筛/唯一分解） |
| 六级 | DFS、BFS、简单DP、背包 | 树（遍历/哈夫曼/二叉排序树）、栈、队列 | 面向对象基础、哈夫曼/格雷编码 |
| 七级 | 复杂DP（二维/LIS/LCS）、图遍历 | 图（邻接矩阵/表）、哈希表 | 数学库函数（三角/对数/指数） |
| 八级 | 最小生成树、最短路、倍增 | 无新增 | 组合数学、复杂度分析、算法优化、代数与平面几何 |

### 定级规则

1. **算法优先**：若题兼涉算法与基础数据结构，**以算法为准**定级——数据结构仅算法之容器，不决等级。例如「尺取法 + 字符串」当为尺取法等级（五级），非字符串之三级。
2. **取最高**：涉多算法/知识点时，按最高级别定级。
3. **只看考纲**：不管实际难度，看涉及何考纲知识点。

### 常见误判

| 误判 | 正 |
|------|------|
| 尺取法 + 字符串 → 定三级（按字符串） | **五级**（算法优先：尺取法乃五级算法） |
| 二分 + 数组 → 定三级（按数组） | **五级**（算法优先：二分乃五级算法） |
| DFS + 二维数组 → 定四级（按二维数组） | **六级**（算法优先：DFS乃六级算法） |
| 递推 + 一维数组 → 定三级（按一维数组） | **四级**（算法优先：递推乃四级算法） |

### 快速定级

```
【数据结构径】
基本类型（int/char/bool等）→ 一级
一维数组/字符串             → 三级（注：不含vector等变长数组）
二维数组                    → 四级
结构体/指针                 → 四级
链表                        → 五级
树/图                       → 六～七级
哈希表                      → 七级

【算法径】
基本顺序/分支/循环    → 一级
多层分支/循环嵌套     → 二级
枚举/模拟             → 三级
递推/排序             → 四级
二分/贪心/分治        → 五级
搜索(DFS/BFS)         → 六级
动态规划              → 六～七级
图论算法              → 八级
```

### 常见题应

| 题目特征 | GESP级别 |
|---------|----------|
| A+B、判奇偶、计算器、公式计算 | 一级 |
| ASCII转换、数学函数应用、多层嵌套 | 二级 |
| 进制转换、位运算、数组统计、枚举模拟 | 三级 |
| 二维数组、结构体、排序、递推 | 四级 |
| 高精度、链表、二分答案、GCD | 五级 |
| 树遍历、迷宫(DFS/BFS)、背包DP | 六级 |
| LIS/LCS、图遍历、复杂DP | 七级 |
| 最短路、最小生成树、组合数学 | 八级 |

## 第二步：定难（思维复杂度）

**独立于GESP级别**：一级题可甚难（复杂嵌套），八级题可甚简（直套模板）。

### 难度等级（五星制）

| 星级 | 特征 | 举例 |
|:----:|------|------|
| ⭐ | 直套公式/模板，一步到位 | 直分支判断、数组逆序、简公式 |
| ⭐⭐ | 需简变形或组合2知识点 | 结构体排序、多条件筛选 |
| ⭐⭐⭐ | 需算法综合或思维转化 | DP态设计、图遍历变体 |
| ⭐⭐⭐⭐ | 复杂优化、数据结构结合 | 线段树优化、复杂图论 |
| ⭐⭐⭐⭐⭐ | 高难竞赛题 | 高级算法组合 |

### 定难规则

1. **看拐点**：需"想一步"否，而非"直做"？
2. **看组合**：需组多不相关知识点否？
3. **看边界**：正理边界需特殊思维否？
4. **看数据**：需算法效率上动脑否？

### 常见误判

| 误判 | 例 | 正 |
|-----|------|------|
| 用数组即三级难度 | 成绩倒序（一级知识点+一级难度） | GESP三级、难度 ⭐ |
| 用排序即四级难度 | 冒泡排序给5数排序 | GESP四级、难度 ⭐ |
| 一级知识点只能 ⭐ | 复杂嵌套条件+多种特殊情况 | GESP一级、难度 ⭐⭐ |

## 2026大纲要变

1. **明排超纲**：三级/四级数组不含变长数组（vector）；四级函数不含Lambda/运算符重载
2. **C++标准统一**：诸试以C++11为准
3. **重效**：当析时间空间复杂度
