# Step 3: GESP 知识点等级判定

## 目标

判定题目知识点等级（GESP 1-8级）和难度等级。

## 等级表（2026年最新大纲）

| 级别 | 核心算法 | 数据结构 | 数学/其他 |
|------|---------|----------|----------|
| 一级 | 无（顺序/分支/循环） | 基本类型 | 计算机基础、ASCII |
| 二级 | 枚举、模拟 | 无新增 | 数学函数、类型转换 |
| 三级 | 枚举、模拟 | 一维数组（不含变长）、字符串 | 进制转换、位运算 |
| 四级 | 递推、排序（冒泡/插入/选择） | 二维数组、结构体、指针基础 | 函数、文件操作 |
| 五级 | 递归、分治（快排/归并）、贪心、二分 | 链表（单/双/循环） | 高精度、数论（GCD/素数筛） |
| 六级 | DFS、BFS、简单DP | 树（遍历/哈夫曼）、栈、队列 | 面向对象基础 |
| 七级 | 复杂DP、图遍历 | 图（邻接矩阵/表）、哈希表 | 数学库函数 |
| 八级 | 最小生成树、最短路、倍增 | 无新增 | 组合数学、复杂度分析 |

## 判定规则

1. 取知识点最高等级（就高不就低）
2. 算法优先，数据结构辅助
3. 知识点等级 ≠ 题目难度（简单题也可能用高级语法）

## 难度等级

| 难度 | 特征 | 对应GESP级别 |
|------|------|--------------|
| 入门 | 直接应用知识点 | 一～三级 |
| 普及- | 简单变形、组合 | 四～五级 |
| 普及/提高- | 算法综合应用 | 六～七级 |
| 提高/省选- | 复杂问题与优化 | 八级 |
| 省选/NOI | 高难度竞赛题 | 超纲（NOI范围） |

## 快速判定

```
【数据结构路径】
基本类型     → 一级
数组/字符串  → 三级（注意：不含vector等变长数组）
二维数组      → 四级
结构体/指针   → 四级
链表          → 五级
树/图         → 六～七级
哈希表        → 七级

【算法路径】
顺序/分支/循环 → 一～二级
枚举/模拟      → 三级
递推/排序      → 四级
二分/贪心/分治 → 五级
搜索(DFS/BFS) → 六级
动态规划       → 六～七级
图论算法       → 八级
```

## 常见题目对应

| 题目特征 | GESP级别 |
|---------|----------|
| A+B、判断奇偶、计算器 | 一级 |
| 进制转换、位运算、数组统计 | 三级 |
| 二维数组、结构体、排序、递推 | 四级 |
| 高精度、链表、二分答案、GCD | 五级 |
| 树遍历、迷宫(DFS/BFS)、背包DP | 六级 |
| LIS/LCS、图遍历、复杂DP | 七级 |
| 最短路、最小生成树、组合数学 | 八级 |

## 2026大纲重要变更

1. **明确排除超纲内容**：三级/四级数组不含变长数组（vector）；四级函数不含Lambda/运算符重载
2. **C++标准统一**：所有考试以C++11为准
3. **强调效率意识**：需分析时间和空间复杂度
