# Step 4: 生成题面

## 目标

生成中文题面 `work/problem_zh.md`。

## 铁律：仅做翻译

1. 只做翻译，保持原有结构
2. 严禁 AI 内心戏：禁止分析、推理、计算过程
3. 样例解释：仅翻译原题解释
4. **严禁修改样例**：样例输入/输出必须原封不动复制，不得增删改任何字符
5. **保留图片链接**：题面中的图片链接（如 `![](url)` 或 `<img>` 标签）必须原样保留，不得删除或替换
6. **保留示意图**：题面原有的示意图、表格、公式等必须完整保留

## 标题格式

`中文标题(英文原标题)`

示例：
- `移除前导o(Trimo)`
- `传感器数据记录(Sensor Data Logging)`

## 题面格式

```markdown
<div class="water">

# 中文标题(英文原标题)

#### 题目描述
[中文翻译]

#### 输入格式
...

#### 输出格式
...

#### 样例输入 #1
...

#### 样例输出 #1
...

#### 数据范围
...

#### 知识点与难度
本题涉及的知识点从属于 **GESP X级**（[知识点列表]），难度等级：**[难度等级]**。

</div>
```
