# Step 4: 生题面

## 目标

生中文题面 `{WORK_DIR}/problem_zh.md`。

## 铁律：仅译不编

1. 仅译，保原构
2. 禁 AI 内心戏：禁分析、推理、计算过程
3. ❌ **禁解题思路/解法分析/题解**：题面只含描述、输入输出格式、样例、数据范围，不附任何解题指引或算法讲解
4. 样例释：仅译原题释
5. **禁改样例**：样例输入/输出必原封不动复，不增删改任一字
6. **图片链处理**：题面中图片链接区分配图来源：
   - **互联网链**（`http://` / `https://` 起）：原样保留，不删不替
   - **本地图/附件**（PDF 提取、用户本地文件等）：拷贝至 `{WORK_DIR}/additional_file/`，题面中引用改为 `![](file://文件名)`，禁用 `images/` 目录
7. **留示意**：题面原有示意图、表格、公式等必完整留

## 标题格式

`中文标题(英文原标题)`

例：
- `移除前导o(Trimo)`
- `传感器数据记录(Sensor Data Logging)`

本题涉及的知识点从属于 **GESP X级**，难度等级：**⭐（星级）**。

> ⚠️ **GESP X 级 和 星级 不得凭印象填！**
> 必先读 **`03-gesp.md`**，以第三步「定级」结果填 X，以第五步「定难」结果填星级。
> 二者均自 03-gesp.md 来，非 AI 自由发挥。

### 测试点分布

## 更名工作目录（要务！）

生题面后，**将标题加入工作目录名**，增辨识：

```bash
# 定标题简写：
# - 英文标题：取全标题，空格→下划线，转小写
# - 中文标题：取前 4~8 字
# 例：标题为"移除前导o(Trimo)" → 简写为"移除前导o_Trimo"
#      标题为"Sensor Data Logging" → 简写为"sensor_data_logging"

mv {WORK_DIR} {WORK_DIR}_{标题简写}
# {WORK_DIR} = {WORK_DIR}_{标题简写}
```

> **注**：若 PID 为 null（原创题），径用标题简写：
> ```bash
> BASE_DIR=$(detect_desktop)
> mv $BASE_DIR/work $BASE_DIR/work_{标题简写}
> # {WORK_DIR} = $BASE_DIR/work_{标题简写}
> ```
