# Step 4: 生题面

## 目标

生中文题面 `{WORK_DIR}/problem_zh.md`。

## 铁律：仅译不编

1. 仅译，保原构
2. 禁 AI 内心戏：禁分析、推理、计算过程
3. 样例释：仅译原题释
4. **禁改样例**：样例输入/输出必原封不动复，不增删改任一字
5. **留图片链**：题面中图片链接（如 `![](url)` 或 `<img>` 标签）必原样留，不删不替
6. **留示意**：题面原有示意图、表格、公式等必完整留

## 标题格式

`中文标题(英文原标题)`

例：
- `移除前导o(Trimo)`
- `传感器数据记录(Sensor Data Logging)`

## 题面格式

```markdown
<div class="water">

# 中文标题(英文原标题)

#### 题目描述
[中文翻译]

#### 输入格式
...

#### 输出格式
...

#### 样例输入 #1
...

#### 样例输出 #1
...

#### 数据范围
...

#### 知识点与难度
本题涉及的知识点从属于 **GESP X级**（[知识点列表]），难度等级：**⭐（星级）**。

</div>
```

## 更名工作目录（要务！）

生题面后，**将标题加入工作目录名**，增辨识：

```bash
# 定标题简写：
# - 英文标题：取全标题，空格→下划线，转小写
# - 中文标题：取前 4~8 字
# 例：标题为"移除前导o(Trimo)" → 简写为"移除前导o_Trimo"
#      标题为"Sensor Data Logging" → 简写为"sensor_data_logging"

mv {WORK_DIR} {WORK_DIR}_{标题简写}
# {WORK_DIR} = {WORK_DIR}_{标题简写}
```

> **注**：若 PID 为 null（原创题），径用标题简写：
> ```bash
> mv work work_{标题简写}
> # {WORK_DIR} = work_{标题简写}
> ```
