## 8.4 GESP 等级复核

按 **03-gesp.md** 第三步（定级）重验定级之逻辑：

- [ ] 算法识别完整——题中所有算法皆已识别，未遗漏
- [ ] 取最高级别——按最高级算法定级，非次级
- [ ] 数据结构不独立定级——未错以数据结构（如数组、哈希表）为定级依据
- [ ] 数学不计级——组合数学、数论等理解性知识未误入定级
- [ ] 对照常见误判速查表（03-gesp.md 4.2 节）——无同类错误

**若以上任一条不通过，则回 03-gesp.md 重走定级流程。**

## 8.5 分值复核

按 **03-gesp.md 第六步（定分）** 重验 score：

- [ ] `score` 取整至 50 倍数（如 1300、1450、1700）
- [ ] CF 已知 Rating 者直接采用（非估算）
- [ ] 非 CF 题目：基准分 = GESP等级 对应基准分（第六步 6.1 表）
- [ ] 难度星级修正已应用（第六步 6.2 表）
- [ ] 数据规模修正已考虑（第六步 6.2 表）
- [ ] 最终 score 落于合理区间（不偏离类似题平均 ±200）

**分值推算记录**：

```

```
