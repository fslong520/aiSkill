# 书架编号

#### 题目描述
学校图书馆新到了一批书，书架编号从 a 到 b。管理员说只有编号为偶数的书架才需要贴标签。她想知道，这批书架中所有偶数编号加起来是多少，以便申请对应数量的标签。

#### 输入格式
一行，两个正整数 a 和 b（1 ≤ a ≤ b ≤ 10000），表示书架的起始和结束编号。

#### 输出格式
一个整数，表示所有偶数编号之和。

#### 样例输入 #1
```
1 10
```

#### 样例输出 #1
```
30
```

#### 数据范围
1 ≤ a ≤ b ≤ 10000

#### 知识点与难度
本题涉及的知识点从属于 **GESP二级**，难度等级：**⭐**。
