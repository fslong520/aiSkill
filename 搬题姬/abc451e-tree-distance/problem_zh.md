# Tree Distance

<div class="water">

判断是否存在一棵带权无向树满足以下条件。

对于任意两个整数 $i$ 和 $j$（$1 \le i < j \le N$），顶点 $i$ 和 $j$ 之间的距离为 $A_{i,j}$。

这里，顶点 $i$ 和 $j$ 之间的距离定义为连接这两个顶点的唯一简单路径上的边权之和。

## 输入格式

第一行一个整数 $N$。

接下来 $N-1$ 行，第 $i$ 行包含 $N-i$ 个整数 $A_{i,i+1}, A_{i,i+2}, \ldots, A_{i,N}$。

## 输出格式

如果存在满足条件的树，输出 Yes；否则输出 No。

## 样例

### 样例输入 1
```
4
2 5 4
3 2
5
```

### 样例输出 1
```
Yes
```

### 样例解释 1
例如，以下边的树满足条件：
- 边 (1,2) 权重为 2
- 边 (2,3) 权重为 3
- 边 (2,4) 权重为 2

### 样例输入 2
```
4
2 5 4
3 2
6
```

### 样例输出 2
```
No
```

### 样例输入 3
```
10
1039 1802 3781 231 5828 1944 392 262 1481
763 2742 1270 4789 905 1431 1301 442
1979 2033 5552 142 2194 2064 1205
4012 7531 2121 4173 4043 3184
6059 2175 161 493 1712
5694 6220 6090 5231
2336 2206 1347
654 1873
1743
```

### 样例输出 3
```
Yes
```

## 数据范围

- $2 \le N \le 3000$
- $1 \le A_{i,j} \le 9999$
- 所有输入值为整数
