# 搬砖流水线(Brick Conveyor Line)

#### 题目描述
工地上有 n 块砖要按重量从小到大排好，用冒泡排序（加 flag 优化）。砖头基本已经有序，只有少数几块位置不对。输出实际需要跑多少趟。

#### 输入格式
第一行一个整数 n（1 ≤ n ≤ 100）。  
第二行 n 个整数，表示砖头重量。

#### 输出格式
一个整数，表示排序所需的趟数。

#### 样例输入 #1
```
8
1 2 3 5 4 6 7 8
```

#### 样例输出 #1
```
2
```
第 1 趟：5 和 4 交换 → 1 2 3 4 5 6 7 8（有交换）  
第 2 趟：无交换，flag 未变，break。共 2 趟。

#### 数据范围
1 ≤ n ≤ 100  
重量为不超过 1000 的正整数。

#### 知识点与难度
本题涉及的知识点从属于 **GESP二级**（数组、冒泡排序、flag优化），难度等级：**⭐**。
