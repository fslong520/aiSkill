# 零花钱

#### 题目描述
小明记了他 n 天的零花钱收支（每天的金额为正整数，表示当天收到的零花钱）。他想知道，有多少个"连续日期段"的零花钱总和恰好等于 k 元。

例如，5 天的零花钱分别是 1、2、3、4、5，目标金额 k = 5：
- 第 1~2 天：1+2 = 3 ✗
- 第 1~3 天：1+2+3 = 6 ✗
- 第 2~3 天：2+3 = 5 ✓
- 第 4~4 天：4 ✗
- 第 5~5 天：5 ✓

所以有 2 个连续日期段满足条件。

#### 输入格式
第一行两个正整数 n 和 k（1 ≤ n ≤ 1000，1 ≤ k ≤ 10000）。  
第二行 n 个正整数，表示每天的零花钱金额。

#### 输出格式
一个整数，表示总和恰好为 k 的连续日期段数量。

#### 样例输入 #1
```
5 5
1 2 3 2 1
```

#### 样例输出 #1
```
2
```

#### 数据范围
1 ≤ n ≤ 1000，1 ≤ k ≤ 10000，1 ≤ a_i ≤ 1000

#### 知识点与难度
本题涉及的知识点从属于 **GESP三级**，难度等级：**⭐⭐**。
