# 等式

<div class="water">

输入一个整数 $a$，求出一个最小的 $n$ 和 $k$（$n > k \ge a$），使得能在 $1$ ~ $n$ 之间能找到一个 $k$，且 $1$ ~ $k-1$ 的和等于 $k+1$ ~ $n$ 的和。

## 输入格式

一行一个正整数 $a$。

## 输出格式

一行两个数，表示最小的正整数 $k$ 和 $n$，之间用一个空格隔开。

## 样例

### 样例输入 1
```
3
```

### 样例输出 1
```
6 8
```

### 样例解释 1

- $k = 6$，$n = 8$
- $1 + 2 + 3 + 4 + 5 = 15$
- $7 + 8 = 15$
- 两边相等

## 数据范围

- $3 \le a \le 1940500$
