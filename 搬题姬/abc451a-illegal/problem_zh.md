# illegal

<div class="water">

在 Takahashi 居住的 AtCoder 国，有一条奇怪的法律："你不能写长度为 5 的倍数的字符串。"

Takahashi 写了一个字符串 $S$。判断他是否违反了这条法律。

## 输入格式

一个字符串 $S$。

## 输出格式

如果 Takahashi 违反了法律，输出 Yes；否则输出 No。

## 样例

### 样例输入 1
```
legal
```

### 样例输出 1
```
Yes
```

### 样例解释 1
`legal` 的长度为 5。因此 Takahashi 违反了法律，输出 Yes。

### 样例输入 2
```
atcoder
```

### 样例输出 2
```
No
```

### 样例解释 2
`atcoder` 的长度为 7。因此 Takahashi 没有违反法律，输出 No。

### 样例输入 3
```
illegal
```

### 样例输出 3
```
No
```

## 数据范围

- $S$ 的长度在 $1$ 到 $10$ 之间
- $S$ 由小写英文字母组成
