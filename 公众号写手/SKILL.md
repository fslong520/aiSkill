## Keywords

公众号文章, 爆款推文, 标题优化, 内容创作, AI去痕, 多平台适配, 排版优化

## Summary

创作爆款公众号推文，五步法：标题→排版→配图→封面→输出。

## Strategy
1. 创作5个爆款标题（悬念/痛点/反直觉/利益/情绪型）
2. 内容排版优化（H2/H3分层、每段≤4行、加粗金句）
3. 智能配图插入（3-5张，关键位置）
4. 封面设计方案（3个方案，21:9比例）
5. Markdown格式输出

AVOID:
- AVOID 用"我有个朋友"开头，这是AI最大破绽
- AVOID 每段结构一样，要有长短变化
- AVOID 用"说实话""所以啊"等AI常用过渡词
- AVOID 结尾用编号列表总结，简单收尾即可
- AVOID 无中生友，真人没那么多朋友

## 爆款标题五型

| 类型 | 示例 |
|------|------|
| 悬念型 | 《为什么你学了AI却用不上？真相让人意外》 |
| 痛点型 | 《AI时代必备的3个核心技能，90%的人都忽略了》 |
| 反直觉型 | 《别再追风口了，真正的高手都在做这件事》 |
| 利益相关型 | 《从零开始掌握AI的5个关键步骤》 |
| 强情绪型 | 《AI技能焦虑？这3个方法让你快速上手》 |

## 真人化写作要点

**核心原则：随意感 > 口语化**

| 避免 | 改为 |
|------|------|
| 每段结构一样 | 句子长短不一 |
| "我有个朋友..." | "前几年..."、"听说..." |
| "说实话""所以啊" | 直接说结论 |
| 结尾编号列表 | 逗号分隔，简单收尾 |
| 满篇金句 | 自然节奏，细节回扣 |

## 配图策略

- 数量：3-5张
- 位置：封面、核心步骤、总结
- 格式：`🖼️ [配图位置：描述]`
