## 核心原则

爆文非运气，乃系统工程。**五维体系**缺一不可：

```
选题（决定80%爆款概率）
  └─→ 标题（决定打开率）
       └─→ 结构（决定完读率）
            └─→ 情绪（决定转发率）
                 └─→ 传播（决定裂变规模）
```

每篇文章不必五维皆满分，但须知每一维在做什么。

## 一、2025-2026 公众号生态认知

**此乃一切方法之前提。** 若按旧订阅逻辑创作，方向全错。

### 算法变革三要点

1. **推荐为主，订阅为辅**——公众号已从"订阅优先"全面转向"算法推荐 + 社交裂变"双引擎。非粉丝流量约占阅读量80%。新号第二、三篇出10万+已有大量案例。
2. **社交互动权重大幅提升**——朋友转发、点"在看"、评论等社交行为影响推荐权重约 **35%**。优质内容推荐周期可达 **7天**。
3. **AI内容严控**——平台对简单粘贴的AI文章识别准确率极高，直接限流。AI是辅助，**必须人工深度加工**，注入真实观点与独家信息。

### 运营启示

- 质量远重于数量。日更会"冲掉"正在推荐的好文章。每周2-3篇高质量足矣。
- 深度内容逆势增长。3000+字深度文章反而更容易获推荐。
- 关注月度数据趋势，而非单篇阅读量涨跌。

## 二、选题——80%的爆款概率在此

**选对题，文章成一半。**

### 爆款选题公式

```
爆款选题 = 具体场景 + 普遍痛点 + 可行动性
```

例："深夜加班改方案，领导却说再优化一下"（具体场景）
   + "职场人都经历过的委屈"（普遍痛点）
   + "3个方法高效沟通"（可行动性）
   = 10万+潜质

### 选题五原则（优先级从高到低）

| 原则 | 本质 | 自问 |
|------|------|------|
| **社交货币** | 转发能替用户表达"我是谁" | 读者转发后会显得怎样？ |
| **情绪共振** | 踩中焦虑/渴望/愤怒/共鸣 | 目标群体此刻什么情绪？ |
| **增量价值** | 打破认知的新视角/独家方法 | 读者看完知道什么新东西？ |
| **普适或垂直** | 破圈选广，垂直扎深 | 目标人群有多大？ |
| **确定性** | 读者能一句话说清"得到了什么" | 看完有什么确定的收获？ |

### 选题四要素组合（刘润法）

爆款选题往往是以下四要素的组合：

- **热点**——方丈出事了、马斯克建党了……
- **干货**——学到了什么知识、懂得了什么道理
- **贪婪/恐惧**——帮你赚钱的10种方法、90%的人都踩过的坑
- **情绪**——成为大家的"嘴替"，替读者说出想说的话

≥2个要素叠加，爆款概率大增。

### 选题素材库建设

1. **微信搜一搜**——搜领域关键词，看爆款，微信算法会持续推荐同类内容
2. **套娃式拓展**——从一篇爆文中提取3-5个关键词，分别搜索，层层扩散
3. **评论区挖掘**——爆文评论区常有高赞观点，是选题富矿
4. **跨平台迁移**——小红书/知乎/抖音的热门话题，经改编后搬到公众号
5. **年度选题日历**——固定热点（春节/国庆/行业大会）+ 周期话题（季报/年度盘点）+ 灵活空白

### 三种选题类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **热点型** | 借势流量，快速反应 | 《刚刚，XX政策出台，对普通人影响最大的三点》 |
| **干货型** | 提供实用知识/方法 | 《记住这5条公式，普通人也能把AI用到极致》 |
| **情绪型** | 成为嘴替，引发共鸣 | 《35岁被裁员，存款不足10万，我做了什么》 |

## 三、标题——300毫秒定生死

公众号平均打开率仅1.13%。标题决定100个粉丝里那一个人是否点开。

### 五大吸睛公式

| 公式 | 结构 | 示例 |
|------|------|------|
| **数字+具体承诺** | 数字 + 可量化收益 | 《3个方法，让工作效率提升200%》 |
| **身份标签+痛点** | 特定人群 + 扎心问题 | 《95后职场新人必看！这3个坑90%的人都踩过》 |
| **冲突对比+反差** | 否定旧认知 + 引出新观点 | 《别再追风口了，真正的高手都在做这件事》 |
| **情绪词+悬念** | 情绪词 + 留白/省略 | 《婆婆竟然这样说…让100万妈妈泪崩》 |
| **强调词+损失厌恶** | 千万/一定/别 + 错过损失 | 《千万别用这3个方法减肥！越减越胖》 |

### 标题自检清单

- □ 字数15-28字（超30字手机端截断）
- □ 核心词在前15字内
- □ 包含至少一个情绪词（焦虑/震惊/遗憾/惊喜……）
- □ 目标读者一眼认出"这是写给我的"
- □ 包含搜一搜高频关键词
- □ 每篇准备 **≥5个备选标题**，选最佳

### 底层心理学

- **信息缺口**——标题制造"不完整感"，大脑自动想补全
- **损失厌恶**——"别再犯这5个错误" 比 "学会这5个技巧" 打开率高约15%
- **社交认同**——"为什么越来越多的年轻人选择这样做" 暗示社会趋势，怕落伍者必点

**关键原则：标题与内容必须匹配。** 标题党短期拉高点击率，但读者读完发现落差大就不会分享，算法将"高点击低分享"判定为低质内容并降权。

## 四、结构——让读者一口气读完

结构控制阅读节奏。**开头前150字决定读者是否划走。**

### 四种经验证爆文结构

#### 模板一：故事型（情感/个人成长）
```
悬念开头 → 事件起因 → 情节转折 → 升华结尾
```
适合个人经历分享、案例分析。用真实故事降低防御心，自然引出观点。

#### 模板二：并列型（方法论/技巧分享）
```
问题提出 → N个方法平行展开（每个独立完整）→ 总结提炼
```
最常见，写作有条理，读者看得清。注意每个案例之间要有区分度。

#### 模板三：逻辑型（知识科普/深度分析）
```
是什么 → 为什么 → 怎么做
```
SCQA模型变体：情境（Situation）→ 冲突（Complication）→ 问题（Question）→ 答案（Answer）

#### 模板四：清单型（工具推荐/资源整理）
```
全貌表格（快速扫描）→ 逐项深度展开 → 行动清单收尾
```
信息密度高，收藏率高。每个工具篇幅大致相当，不厚此薄彼。

### 三明治干货法（内容填充）

```
上层：引发共鸣的案例/故事
中层：结构化的知识体系（理论框架）
底层：数据/事实背书
```

例：《斯坦福学霸的5大深度工作法》，先讲MRI数据证明专注者大脑灰质密度高23%（底层），再讲5个方法（中层），每个方法配实战故事（上层）。

### 段落节奏控制

- **开头150字定生死**：开门见山，直接告诉读者"这篇解决你什么问题"。无寒暄无铺垫。
- **每段≤200字**（手机上约3-4行）：密密麻麻的长段在手机上像一堵墙，读者直接划走。
- **每300字设一个"钩子"**：用提问/反转/悬念拉住注意力，防止中途离开。
- **结尾给出行动号召**：不引导，用户就不会动。"若对你有用，点个赞，转给一个正迷茫的朋友。"

## 五、情绪设计与金句——转发驱动力

情绪驱动传播。没情绪的内容不会被转发。

### 三个心理触发点

#### 触发点一：共鸣式
让读者觉得 **"这说的不就是我吗"**。靠大量细节实现。

> ❌ "职场中经常遇到这种情况……"
> ✅ "周五下午5点半正准备下班，突然收到领导微信：'明天要用的PPT改好了吗？'"

细节唤起共鸣。时间、地点、对话、动作，越具体越好。

#### 触发点二：对立式
制造 **"我们 vs 他们"** 的群体认同。帮读者找到"同盟"，获得归属感。

> "只有真正带过娃的人才知道……"

慎用，不能太极端，但确实有效。

#### 触发点三：激励式
给希望和行动动力。

> "从0到10万+，只有一步之遥——掌握方法，下一篇可能就是你的。"

**约束：** 负面情绪不能太重，必须给解决方案；正面激励必须具体可行，避免空洞鸡汤。

### 情绪递进曲线

一篇爆文的情绪走向往往如此：

```
开头：戳痛点（"你有没有发现……"）
  ↓
中段：放大冲突/共鸣（"这说的不就是我吗"）
  ↓
转折：给出新认知（"原来我一直理解错了"）
  ↓
结尾：升华 + 行动（一句话点醒 + 引导转发）
```

### 金句创作方法论

金句不是满篇散落，而是关键节点的一击即中。

- **金句本质**：读者看完后最想说的一句话。是他想转发到朋友圈的那句话。
- **金句位置**：每段末尾、转折处、结尾升华处。
- **金句类型**：
  - 反常识型："你不是不努力，你只是没有路径"
  - 共鸣型："穷，不是问题，没认知才是"
  - 行动型："种一棵树最好的时间是十年前，其次是现在"

**警惕：AI生成的金句往往华丽但空洞。** 金句是进入情绪之后脱口而出的话，不是修辞堆砌。

## 六、配图与排版——让眼睛舒服

### 排版三三原则

1. **每段不超过3行**（手机端）：密密麻麻像一堵墙，读者直接划走
2. **每3段插入一个视觉元素**：图片/引用块/表格/分隔线，打破单调
3. **关键信息用3种方式强调**：加粗 + 背景色块 + （适度使用），不滥用

### 配图策略

- **数量**：3-5张
- **位置**：封面、核心步骤/观点处、总结升华处
- **格式**：`🖼️ [配图位置：描述]`
- **质量**：清晰、与内容相关、风格一致
- **AI配图注意**：GPT Image等工具可生成，但须保持篇内风格统一

## 七、传播机制——爆款是推出来的

### 两次传播模型

```
一次传播：公众号送达 → 会话打开 → 首次分享
  （由标题+封面决定，粉丝基数和打开率）
       ↓
二次传播：朋友圈扩散 + 算法推荐
  （由内容价值和转发诱因决定，决定能否破圈）
```

### 阅读量公式

```
阅读量 = (粉丝基数 × 打开率) + (转发人数 × 朋友圈人数 × 朋友圈打开率)
```

粉丝不多 → 提高转发率；转发率不高 → 优化内容的传播性。

### 社交货币四型

用户转发一篇文章，本质是表达"我是谁"。你的选题/金句必须提供以下至少一种社交货币：

| 货币类型 | 本质 | 设计方法 |
|---------|------|---------|
| **身份货币** | 转发显得我是XX样的人 | 文末附"仪式感打卡"图，用户完成可分享 |
| **谈资货币** | 有新奇观点/冷知识可聊 | 文中埋反常识数据或独家洞察 |
| **利他货币** | 对某个朋友有用，@TA | 标题用"建议所有XX人看完" |
| **情绪货币** | "这说的就是我" | 开头描述目标群体的共性状态，让读者觉得被懂得 |

### 算法互动设计

- 文章中段设置问题/投票/话题讨论
- 结尾引导"在看"和评论
- **回复每条用户留言**——互动提升社群粘性，也提升推荐权重
- 发布时间：工作日上午8-10点（通勤）或晚8-10点（睡前）为阅读高峰

## 八、热点操作框架

### 三层解读法

| 层级 | 内容 | 黄金时间 |
|------|------|---------|
| **信息层** | 还原事件本身：发生了什么、谁参与了 | 热点发生2小时内 |
| **分析层** | 专业角度解读：为什么发生、意味着什么 | 6-24小时 |
| **洞察层** | 提炼通用规律：对读者有什么启发、怎么行动 | 24-48小时 |

越晚出手，分析深度必须越高。第一波拼速度，第二波拼角度，第三波拼深度。

### 追热点决策三问

热点出现后，先快问快答三个问题：

1. 这个热点和我的账号定位相关吗？
2. 我能提供什么独特的解读视角？
3. 这个话题的争议度会带来负面风险吗？

**三个回答皆肯定 → 追。任何一个是"否" → 放弃。**

十个热点通常只追两三个。克制是追热点的核心美德。

## 九、真人化写作要点

**核心原则：随意感 > 口语化**

### 避坑清单

| 避免 | 改为 |
|------|------|
| 每段结构雷同 | 句子长短不一，段落大小穿插 |
| "我有个朋友……" | "前几年……"、"听说……"、"有个读者曾跟我说……" |
| "说实话""所以啊""不得不说" | 直接说结论，省去过渡 |
| 结尾编号列表（1/2/3） | 逗号分隔自然收尾，或一句话点题 |
| 满篇金句 | 自然行文节奏，细节回扣核心观点 |
| "无中生友" | 用真实经历或公开案例，真人没那么多朋友 |
| "随着……的发展""在……背景下" | 开头直接上场景，不铺垫 |
| 万能套话（"值得深思""引人深思"） | 给出具体判断，不用空洞总结词 |

### 检查文章有无AI味

- 读一遍：像不像一个真人坐你对面说的话？
- 去掉所有过渡词后，意思是否更清晰？
- 有没有只有你才知道的独特观点/经历/数据？

## 十、创作检查清单（发前必过）

### 选题自检
- □ 是否低门槛（不需专业知识也能懂）？
- □ 是否高共鸣（目标受众能对号入座）？
- □ 是否有情绪点（焦虑/共鸣/愤怒/希望）？
- □ 是否有增量（反常识/新视角/独家方法）？

≥3个✅，选题有戏。否则重选。

### 标题优化
- □ 字数15-28字？
- □ 核心词在前15字内？
- □ 用了吸睛公式（数字/身份标签/冲突/情绪词）？
- □ 制造了好奇心缺口？
- □ 有没有 ≥5个备选标题？

### 结构检查
- □ 开头150字内抓住读者？
- □ 使用了四种结构模板之一？
- □ 每段≤200字？
- □ 每300字设了一个"钩子"？
- □ 结尾有行动号召？

### 情绪设计
- □ 开头戳中了痛点？
- □ 中间有共鸣细节？
- □ 情绪有起伏（不是一路平铺）？
- □ 负面情绪给了解决方案？
- □ 有至少一句可转发的金句？

### 传播优化
- □ 文章提供至少一种社交货币（身份/谈资/利他/情绪）？
- □ 有金句方便摘抄转发？
- □ 观点鲜明便于表态？
- □ 能触发"必须收藏/转发"的反应？

### 发布策略
- □ 发布时间合适（早8-10点/晚8-10点）？
- □ 标题和封面在朋友圈缩略图下是否仍有效？
- □ 是否准备在社群/朋友圈做首轮推广？

- 被验证有效的标题/公式
- 阅读量/分享率/打开率

- AVOID 按旧订阅逻辑创作，必须理解算法推荐生态
