# 破晓 - 智能资讯早报生成器

## 🚨 核心铁律（必须遵守！）

### 铁律一：日期强制（违反此条 = 任务失败）

**所有搜索关键词必须包含今日日期！没有日期的搜索 = 无效搜索！**

```
❌ 错误：搜索 "AI 大模型 最新动态"
✅ 正确：搜索 "AI 大模型 最新动态 2026 年 3 月 13 日"

❌ 错误：搜索 "半导体 股票"
✅ 正确：搜索 "半导体 龙头股 3 月 13 日"

❌ 错误：搜索 "entertainment news"
✅ 正确：搜索 "entertainment news March 13 2026"
```

**为什么？**
- 不带日期的搜索会返回几天甚至几周前的旧新闻
- 早报的价值在于"今早"，不是"上周"
- 投资决策需要最新信息，过期新闻可能导致错误判断
- **新闻必须在 24 小时内**，超过 24 小时 = 废纸

### 铁律二：流式处理（边搜边做，不等不拖）

**不要等所有搜索完成再分析！边搜索、边验证、边整合！**

```
传统模式（慢且容易出错）：
搜索所有领域 → 等待 → 分析 → 发现旧闻 → 重新搜索 → ...

流式模式（快且可靠）：
搜索领域 A → 立即验证时效性 → 通过则整合，不通过则重搜
     ↓（同时）
搜索领域 B → 立即验证时效性 → 通过则整合，不通过则重搜
     ↓（同时）
搜索领域 C → 立即验证时效性 → 通过则整合，不通过则重搜
     ↓
全部完成 → 立即生成 HTML
```

#### 时效性验证标准（24 小时铁律）

| 新闻时间 | 处理方式 |
|---------|---------|
| "X 小时前"（1-24 小时） | ✅ 通过 |
| "今日" / "今天" | ✅ 通过 |
| "昨天" / "昨日" | ⚠️ 仅限昨夜 20:00 后 |
| "1 天前" 超过 24 小时 | ❌ 立即重搜 |
| "2 天前" 及更早 | ❌ 立即重搜 |

**24 小时计算示例**：
- 当前时间：3 月 13 日 09:00
- ✅ 有效：3 月 12 日 09:00 之后的新闻
- ❌ 无效：3 月 12 日 09:00 之前的新闻

**重搜时必须调整关键词**：
- 添加更具体的日期格式
- 缩小范围（如 "今日发布"、"刚刚"）
- 尝试英文搜索

## 常见问题与解决方案

### Q: 搜索结果都是旧新闻怎么办？

**A: 检查关键词是否包含日期！**

```
问题：搜索 "AI 大模型" 返回 3 天前的新闻
解决：搜索 "AI 大模型 3 月 13 日" 或 "AI 大模型 今日"
```

### Q: 某个领域搜索不到新闻怎么办？

**A: 调整关键词重试，最多 3 次**

1. 第一次：`"领域关键词 {日期}"`
2. 第二次：`"领域关键词 最新 今日"`
3. 第三次：`"领域关键词 英文日期"`
4. 仍无结果：标注"该领域今日暂无重要动态"

### Q: 链接是媒体主页怎么办？

**A: 这不是有效新闻链接，重新搜索！**

```
❌ 无效：https://36kr.com/newsflashes
✅ 有效：https://36kr.com/p/1234567890
```
