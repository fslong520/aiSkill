# 破晓 - AI 资讯日报生成器

## 核心理念

**需求倒逼学习**：用真实痛点驱动 AI 能力提升，而非为学工具而学工具。

**信息价值公式**：
```
信息价值 = 真实性 × 新鲜度 × 稀缺性
```

- **真实性**：越接近原始信源越高（官方>媒体>社交）
- **新鲜度**：24 小时内发生的事件
- **稀缺性**：独家报道 vs 同质化搬运

---

## 快速开始

### 第一步：明确你的信息痛点

问自己三个问题：
1. 我每天需要花多少时间刷资讯？效率如何？
2. 我关注的信息类型有哪些？（技术/投资/产品/政策）
3. 我的核心诉求是什么？（学习/投资/创业/就业）

### 第二步：配置你的信源池

按质量分梯队管理：

**第一梯队（官方信源）**
- OpenAI Blog、Anthropic、Google AI Blog
- arXiv 论文、GitHub Trending
- 各公司官方发布会、技术报告

**第二梯队（专业媒体）**
- The Verge、TechCrunch、Wired
- 机器之心、量子位、新智元
- MIT Technology Review、IEEE Spectrum

**第三梯队（社交媒体）**
- Twitter/X 上的 AI 研究者
- Reddit r/MachineLearning
- 国内公众号、知乎专栏

**⚠️ 避坑指南**：
- ❌ 初期不要超过 50 个信源（信息过载）
- ✅ 优先关注离原始事件最近的信源
- ✅ 定期清理低质量重复信源

### 第三步：定义筛选规则

使用信息价值评分系统：

```python
# 评分示例（伪代码）
def calculate_score(news):
    authenticity = get_authenticity_level(source)  # 1-5 分
    freshness = hours_since_event / 24  # 24 小时内=1，每过 1 天衰减 0.5
    scarcity = count_similar_reports  # 独家=5，全网都有=1
    
    return authenticity * freshness * scarcity
```

**筛选阈值**：
- ≥15 分：必选（高质量独家）
- 10-14 分：可选（有价值但非独家）
- <10 分：舍弃（噪音信息）

### 第四步：设计输出结构

推荐 6 大板块结构：

```markdown
# 【AI 日报】YYYY-MM-DD

## 🔍 元洞察
今日最值得关注的 1-3 条消息，帮你抓住全局重点

## 📈 行业动态
大厂动态、政策变化、行业趋势

## 💰 投融资
钱流向哪里，未来就在哪里
- 投资事件、并购、IPO

## 🧠 技术突破
真正的技术进步，不是 PPT
- 论文解读、开源项目、benchmark 刷新

## 🚀 产品上线
跟普通人直接相关的产品变化
- 新功能、新应用、价格调整

## 📚 教育资讯
小白友好型内容
- 教程、课程、学习资源

## 📊 A 股联动（可选）
AI 相关的美股/A 股动向
- 概念股涨跌、产业链变化
```

## 工作流设计

### 自动化工作流

```
1. 定时抓取（每天 6:00 AM）
   ↓
2. 去重清洗（移除重复、低质内容）
   ↓
3. 智能评分（真实性×新鲜度×稀缺性）
   ↓
4. 分类整理（归入 6 大板块）
   ↓
5. 生成 HTML（响应式设计、可点击超链接）
   ↓
6. 推送发布（邮件/微信/GitHub）
```

### 手动优化环节

**必须人工介入的部分**：
- 元洞察的选择（需要判断真正的重要性）
- A 股联动分析（需要财经知识）
- 创业机会拆解（需要商业洞察）

**可以自动化的部分**：
- 信息抓取与去重
- 基础评分与排序
- HTML 生成与排版

### 内容质量标准

每条新闻必须包含：
- ✅ 标题（准确反映内容，不标题党）
- ✅ 原文链接（可追溯至原始信源）
- ✅ 发布时间（精确到小时）
- ✅ 信息来源（标注信源梯队）
- ✅ 一句话摘要（50 字内说清核心价值）
- ✅ 相关性标签（#技术 #投资 #产品 #政策）

## 进阶功能

### A 股联动分析

**数据维度**：
- 美股 AI 概念股涨跌（NVDA、MSFT、GOOGL 等）
- A 股 AI 产业链（算力、算法、应用）
- 港股科技公司（腾讯、阿里、百度、字节）

**分析方法**：
```
海外技术突破 → 利好 A 股供应链
国内政策变化 → 影响相关概念股
大厂投资动向 → 预示行业风向
```

### 创业机会拆解

**分析框架**：
1. **技术成熟度**：这项技术是否到了商业化临界点？
2. **市场需求**：解决了什么真实痛点？
3. **竞争格局**：是红海还是蓝海？
4. **变现路径**：ToB/ToC？订阅/广告/增值服务？

**案例模板**：
```markdown
## 💡 创业机会：AI视频生成工具

**技术背景**：Sora、Runway 等模型成熟
**市场需求**：短视频创作者需要低成本高质量内容
**竞争格局**：头部集中，垂直领域有机会
**变现路径**：SaaS 订阅 + 定制服务
**建议切入点**：特定行业模板（教育/电商/本地生活）
```

## 常见问题 FAQ

### Q1：完全没有编程基础怎么办？

**答案**：可以用无代码工具组合实现：
- 抓取：Zapier / IFTTT
- 处理：Notion AI / Mem
- 发布：GitHub Desktop / 微信公众号后台

但建议学习基础 Python，能大幅提升灵活性。

### Q2：如何平衡全面性和效率？

**答案**：遵循二八原则：
- 80% 的内容来自 20% 的高质量信源
- 先保证核心信源质量，再逐步扩展
- 定期复盘，砍掉低价值信源

### Q3：怎么判断信息的真实性？

**答案**：交叉验证法：
1. 查看原始出处（论文/官方公告/发布会）
2. 对比多个权威媒体的报道
3. 警惕情绪化表达和夸张标题
4. 查证数据来源是否可靠

### Q4：如何盈利？

**答案**：常见模式：
- **订阅制**：9.9 元/月或 99 元/年
- **企业版**：定制化报告（999 元+/月）
- **广告合作**：相关产品推广
- **咨询服务**：一对一行业分析

**关键**：先提供价值，再考虑变现。质量够高，用户会主动付费。
