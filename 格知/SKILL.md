# 格知 — 一格一知识

## 技能说明

一键生成单张宫格排版的科普长图，AI根据科普主题**自主判断**最佳视觉风格、宫格数量和内容结构，将所有知识点浓缩在一张宫格图中输出。支持可选IP形象约束，适合小红书、公众号等平台直接发布。

---

## 执行流程

### Step 1：确认科普主题

询问用户："你想科普什么主题？"
- 记录用户的科普主题
- 如果用户在触发技能时已经提供了主题，直接记录，不必重复询问

---

### Step 2：AI自主分析并推荐风格

#### 2.1 AI全维度内容分析

AI自主分析科普主题的所有相关维度，包括但不限于：
- 主题所属领域和属性
- 目标受众画像
- 内容调性和氛围
- 知识深度和密度
- 传播场景适配
- 视觉呈现最佳方式

**AI自行决定分析哪些维度，无需预设框架。**

#### 2.2 AI自主推荐风格

**根据分析结果，AI自主生成风格推荐方案。**

**推荐框架（AI自行填充所有内容）：**
```
【风格智能推荐】

基于「{主题}」的分析：

📊 内容画像：{AI自主生成的内容分析结论}

🎨 推荐风格：

方案A（首选）：{AI自主定义的风格名称}
   推荐理由：{AI结合分析说明为什么适合}
   视觉特征：{AI自主定义的配色、线条、氛围等}

方案B（备选）：{AI自主定义的风格名称}
   推荐理由：{AI说明适用情况}
   视觉特征：{AI自主定义}

💡 提示：你也可以告诉我想要的感觉，如"要可爱一点"、"要正式一点"
```

**AI可自由定义风格名称和特征，不受任何预设限制。**

#### 2.3 用户确认与调整

- 用户选择方案 → 记录风格名称及完整描述，进入Step 3
- 用户提出调整意见 → AI根据反馈调整后确认
- 用户自定义风格 → 记录用户的风格描述，进入Step 3

---

### Step 3：AI判断IP形象需求

#### 3.1 AI自主分析与建议

AI根据主题特性自主判断是否需要IP形象，并给出建议：

```
【IP形象建议】

基于「{主题}」的特性分析：

{AI自主判断是否需要IP形象，并说明理由}

如需IP，建议方向：{AI根据主题推荐的形象类型}
如不需要，说明：{AI解释为什么纯图形表达更合适}

💡 你也可以直接告诉我你的偏好
```

#### 3.2 用户选择处理

- **不需要IP** → 记录无IP形象，进入Step 4
- **使用已有IP** → 用户提供图片URL，记录作为约束
- **生成新IP** → AI询问形象特征后生成并记录
- **接受AI建议** → 按AI建议执行

**重要：确认IP选择后，才能进入下一步！**

---

### Step 4：AI自主判断宫格数量

#### 4.1 AI自主分析内容复杂度

AI根据科普主题的内容复杂度、知识点数量、受众认知门槛等因素，自主判断最合适的宫格数量。

**AI自行决定分析维度，无需预设框架。**

#### 4.2 AI输出推荐方案

```
【宫格方案推荐】

根据「{主题}」的内容分析：

📐 推荐方案：{N}宫格
   推荐理由：{AI自主分析的结论}
   内容框架：{AI自主规划的内容结构}

❓ 请确认，或告诉我你的偏好
```

#### 4.3 用户确认

- 用户确认方案 → 记录宫格数量，进入Step 5
- 用户提出偏好 → AI根据偏好调整后确认
- 用户指定宫格数 → 直接记录，进入Step 5

---

### Step 5：AI自主规划内容结构

AI根据主题、宫格数量、目标受众等因素，自主规划每个格子的内容结构。

**输出格式：**
```
【内容结构规划】

{N}宫格内容安排：

第1格：{主题} — {核心表达}
第2格：{主题} — {核心表达}
...（依次列出所有格子）

❓ 请确认内容结构，或告诉我需要调整的地方
```

**AI自行决定每格的内容主题和表达方式，无需预设框架。**

用户确认后进入生成环节。

---

### Step 6：AI自主决定文字密度

AI根据内容特性和传播场景，自主决定每格的文字密度。

**AI在生成时自动调整：**
- 概念简单、视觉优先 → 文字精简
- 知识密度高、需要理解 → 文字适当增加
- 根据风格特点灵活适配

无需用户单独确认，AI在生成时自动优化。

---

### Step 7：生成单张宫格科普图

使用 `imageGenerate` 工具生成宫格图。

**生成参数配置：**

| 参数 | 值 |
|------|---|
| model | `gemini-3.1-flash-image-preview` |
| source_image_urls | 如果有IP形象，传入Step 3获得的图片URL；否则不使用此参数 |
| aspect_ratio | AI根据宫格数量自行决定最佳比例 |
| quality | `medium` |

**Prompt 构建模板：**

```
生成一张{N}宫格科普知识长图，主题是"{用户主题}"。

【视觉风格】
{AI在Step 2中自主定义的完整风格描述}

【整体布局】
- {AI决定的行列排列}
- 每个格子之间有清晰的分隔线
- 整体是一张完整的图片

【各格内容】
第1格：{该格标题} — {该格核心文案}
...（依次列出所有格子）

【文字要求】
- 所有文字必须是中文
- 每格标题加粗醒目
- {AI根据内容特性自主决定的文字密度}
```

**IP形象Prompt补充（如果有IP）：**
```
【角色形象要求 - 必须完全一致】
{根据参考图描述形象特征}
角色必须出现在多个格子中，保持形象完全一致，不能变形！
每格配有角色的不同动作/表情
```

---

### Step 8：整理输出文档

使用 `create_file` 工具创建输出文档，包含：

1. **生成的宫格科普图**（嵌入图片）
2. **创作信息**
   - 主题：{主题}
   - 风格：{AI定义的风格}
   - 宫格：{N}宫格
   - IP形象：{有/无，如有则嵌入参考图}
3. **各格文案整理**
4. **发布建议**：推荐平台、标题、标签、正文模板

文档标题格式：`「{主题}」{N}宫格科普图`

---

## 快速模式（可选）

如果用户希望跳过详细配置，可直接使用快速模式：

**触发方式：** 用户说"快速生成"或"用默认设置"

**快速模式：**
- AI自动完成所有分析和决策
- 确认主题后直接生成，无需逐步确认
- 风格、宫格数、内容结构、文字密度全部由AI自主判断
