# 笔记手账 - 参考文档

## 风格定义

### 视觉DNA

**核心美学**：模拟真实笔记本/手账的质感，营造"学习笔记"的亲切感。

| 元素 | 特征 | 技术关键词 |
|------|------|-----------|
| **基底** | 米色/奶油色纸张，撕纸边缘，活页孔 | cream_paper, torn_edge, hole_punches |
| **装订** | 螺旋装订（底部或左侧） | spiral_binding |
| **配色** | 深蓝 + 橙色 + 绿色 + 黄色 | navy, orange, green, yellow |
| **装饰** | 便利贴、纸夹、和纸胶带、星星、手绘箭头 | sticky_notes, paper_clips, washi_tape, stars, arrows |
| **字体** | 标题粗圆体，正文无衬线，手写体装饰 | bold_rounded, sans-serif, handwritten |
| **卡片** | 白色圆角卡片，浅灰阴影 | white_card, rounded_corners, shadow |

### 配色系统

| 颜色 | 色值 | RGB | 用途 |
|------|------|-----|------|
| 深蓝 | #1A3A8A | 26,58,138 | 标题、标签、图标主色 |
| 橙色 | #F97316 | 249,115,22 | 编号、重点、装饰 |
| 绿色 | #2D8C3C | 45,140,60 | 副标题、箭头、记忆框 |
| 黄色 | #FBBF24 | 251,191,36 | 下划线、便利贴、星星 |
| 米色 | #F5F0E8 | 245,240,232 | 背景纸张 |
| 白色 | #FFFFFF | 255,255,255 | 内容卡片 |
| 深灰 | #374151 | 55,65,81 | 正文文字 |

### 装饰元素库

#### 固定装饰
- **星星☆**：橙色和绿色，散落在角落
- **纸夹**：绿色或蓝色，固定在左上角
- **便利贴**：黄色，右上角，手写标语
- **和纸胶带**：蓝色或橙色，装饰性粘贴
- **手绘箭头**：绿色，指向关键元素

#### 教育元素
- **螺旋笔记本**：左下角，学习计划区
- **打开的书本**：底部中央，象征学习
- **铅笔/钢笔**：右下角，书写元素
- **灯泡**：高频易错点图标
- **靶心**：一句话记忆图标
- **叶子**：自然装饰，增添生机

## 内容模块详解

### 1. 封面页

**用途**：课程/打卡系列的封面

**必备元素**：
- 主标题（深蓝粗体 + 黄色下划线）
- 副标题（绿色）
- 蓝色标签（左上角）
- 黄色便利贴（右上角）
- 装饰元素（星星、纸夹、箭头）

**可选元素**：
- 学习计划（螺旋笔记本）
- 知识图谱（简图）
- 课程概览（小卡片）

### 2. 知识页

**用途**：知识点讲解

**必备元素**：
- 页眉（蓝色标签 + 便利贴页码）
- 标题（绿色）
- 分区卡片（橙色编号 + 白色内容）
- 高频易错点（灯泡图标）
- 一句话记忆（靶心图标）

**分区数量**：2-5个，根据内容量调整

### 3. 练习页

**用途**：配套练习题

**必备元素**：
- 题目列表（编号）
- 答案区（可选）
- 提示区（可选）

## Prompt生成规则

### 维度填充顺序

1. **style**：确定整体风格（notebook_scrapbook）
2. **background**：纸张质感（cream_paper, torn_edge, hole_punches）
3. **layout**：页面布局（title_centered, sections_grid）
4. **title**：主标题（深蓝粗体 + 黄色下划线）
5. **subtitle**：副标题（绿色）
6. **decorations**：装饰元素（sticky_notes, paper_clips, stars）
7. **sections**：内容分区（橙色编号 + 白色卡片）
8. **special_sections**：特殊模块（高频易错点、一句话记忆）
9. **icons**：图标（简约线条，深蓝/绿色/橙色）
10. **aspect_ratio**：比例（3:4竖版或4:3横版）

### 文字处理

| 情况 | 处理 |
|------|------|
| 用户提供文字内容 | 直接使用 |
| 用户未提供 | 生成占位符，建议填写 |
| 语言 | 默认中文 |

### 比例选择

| 比例 | 用途 |
|------|------|
| 3:4 | 竖版知识卡片、打卡海报（默认） |
| 4:3 | 横版知识卡片 |
| 16:9 | 宽幅信息图 |
| 1:1 | 社交媒体分享 |

## 质量检查表

### 风格一致性
- [ ] 使用正确的配色系统（深蓝、橙、绿、黄）
- [ ] 包含撕纸边缘和活页孔
- [ ] 使用白色圆角卡片
- [ ] 包含装饰元素（星星、纸夹、便利贴）

### 内容完整性
- [ ] 标题清晰醒目
- [ ] 分区编号正确（一、二、三...）
- [ ] 高频易错点有灯泡图标
- [ ] 一句话记忆有靶心图标

### 视觉层次
- [ ] 标题 > 副标题 > 正文 > 重点
- [ ] 颜色使用符合规范
- [ ] 装饰不遮挡内容
- [ ] 留白适当，不拥挤
