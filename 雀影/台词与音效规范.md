# 台词与音效规范

## 一、对白设计

### 三种对白类型

#### 1. 冲突式对白
两人立场对立，话里有话，句句带刺。

```
例：
A："你走。"
B："凭什么。"
A："我不欠你。"
B："你欠我一命。"
```

节奏：短句（2-6字），语速快，句句逼人。
情绪：对抗、紧张。

#### 2. 递进式对白
从平静到爆发，情绪层层推进。

```
例：
A："你确定？"（平静）
A："你真确定？"（怀疑）
A："你会后悔的。"（威胁）
```

节奏：从长到短（12字→6字→4字），情绪逐句升温。
情绪：压抑→临界→爆发。

#### 3. 潜台词对白
嘴上说A，心里想B。话只说三分，留七分给观众猜。

```
例：
A："这茶不错。"
——潜台词：我想多待一会儿。
B："天快黑了。"
——潜台词：你得走了。
```

节奏：语速慢，停顿多，每句后有留白（0.3-0.5秒）。
技巧：对白和内心独白搭配使用，效果最佳。

### 对白设计原则

1. **长短结合**：不要全部是短句，会显得刻意；不要全部是长句，会拖节奏
2. **人各有声**：每个角色说话方式不同——用词、语速、语气皆有差异
3. **留白即金**：对话之间的沉默，比对话本身更有力量
4. **避免解释性对白**："你知道吗，我们是兄妹"→用剧情揭示而非嘴说
5. **对白冲突三原则**：①立场不同 ②信息不对等 ③情绪不对齐

### 对白节奏控制

| 节奏类型 | 每句字数 | 语速 | 停顿间隔 | 效果 |
|---------|---------|------|---------|------|
| 平叙 | 8-12字 | 每秒3-4字 | 0.2-0.3秒 | 自然流畅 |
| 冲突 | 2-6字 | 每秒5-7字 | 0.1秒 | 紧张压迫 |
| 暧昧 | 6-10字 | 每秒2-3字 | 0.5-1秒 | 张力十足 |
| 爆发 | 2-4字 | 每秒6-8字 | 几乎无停顿 | 情绪炸裂 |

## 二、内心独白

### 内心独白 vs 旁白

| | 内心独白 | 旁白 |
|------|---------|------|
| 视角 | 第一人称（我） | 第三人称（上帝视角） |
| 内容 | 真实想法、隐藏情绪、未说出口的话 | 背景信息、时间地点、剧情补全 |
| 作用 | 建立观众与角色之情感连接 | 辅助叙事，推动剧情 |
| 语气 | 私密、真实、脆弱、有情绪 | 客观、平稳、信息型 |

### 内心独白设计原则

1. **必须揭露不可见的情绪**：表情冷酷，内心崩溃——这才是内心独白
2. **必须与对白反差或互补**：嘴上说"没事"，心里说"我快撑不住了"
3. **不宜过长**：每句6-15字，太长老生厌
4. **语速放慢**：每秒2-3字，有呼吸感，像真的在心里说话
5. **用语气词增强真实感**："呵……""原来如此。""罢了。"

### 内心独白格式

```
【内心独白】"呵……原来你也会怕。" #1.5秒 [音量:80%]（放慢，带嘲讽）
```

### 三种内心独白场景

| 场景 | 说明 | 示例 |
|------|------|------|
| 真相揭露 | 角色发现真相，内心震惊但表面平静 | 内心："原来是他……" 对白："没事。" |
| 情感压抑 | 想哭不能哭，想说不能说 | 内心："别走……求你了。" 对白："你走吧。" |
| 计划盘算 | 表面应付，内心算计 | 内心："再等等，就快到手了。" 对白："听你的。" |

## 三、音效六层体系

### 层次结构

| 层 | 名称 | 作用 | 音量比例 | 示例 |
|----|------|------|---------|------|
| 1 | 环境氛围音 | 建立时空基础 | 40% | 雨声、风声、蝉鸣、机械嗡鸣 |
| 2 | 动作音效 | 强化视觉动作 | 60% | 脚步声、关门声、倒酒声 |
| 3 | 情绪音效 | 放大心理感受 | 50-70% | 心跳声、叹息、深呼吸 |
| 4 | 转场音效 | 镜头过渡标记 | 70% | "咻"、低轰鸣、钟表滴答 |
| 5 | 特效音 | 视觉增强 | 80% | 剑鸣、爆炸、能量升腾 |
| 6 | 心理音效 | 内心独白专属背景 | 30-50% | 空旷回音、水下感、低频嗡鸣 |

### 心理音效（新增重点）

内心独白时配专属氛围音，让观众"听进心里"。

| 内心独白情绪 | 心理音效 |
|-------------|---------|
| 失落/悲伤 | 混响+低频嗡鸣，如深水回音 |
| 紧张/恐惧 | 急促心跳声，逐渐加快 |
| 回忆/怀念 | 短暂回声+老旧唱片质感 |
| 震惊/顿悟 | "嗡"一声长鸣，短暂空白 |
| 决断/觉醒 | 鼓点+逐渐明亮的单音 |

### 音效时间标注

```
（音效）沉重脚步声，木质地板咯吱作响 #1.2秒
（心理音效）低频嗡鸣，如深海回响 #2.0秒
```

- `#X.X秒` 表示音效时长
- 音效与动作同步，误差不超过0.2秒

## 四、综合示例

一个完整的镜头音频层设计：

```
【角色﹣林夜】:"你看到了？" #0.8秒 [音量:100%]（男声低沉，压抑）
【内心独白】"不，他不能知道……" #1.2秒 [音量:80%]（语速慢，带慌乱）
（心理音效）低沉嗡鸣，心跳声隐约 #2.0秒
（环境音）夜风穿堂，远处犬吠 #持续
```

### 音频层次在观众耳中的效果

```
观众听到 →
  │ 内心独白（最贴耳，私密感）
  │ 对白（稍远，角色间距离）
  │ 环境音（背景，空间感）
  └ 心理音效（底色，情绪渲染）
```
