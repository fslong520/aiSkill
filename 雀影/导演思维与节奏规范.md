## 景别速查表

### 六大景别定义

| 景别 | 画面范围 | 叙事功能 | 情感强度 |
|------|----------|----------|----------|
| **远景** | 全景 + 环境 | 交代场景、气势 | ★☆☆☆☆ |
| **全景** | 全身 + 周围 | 展示姿态、关系 | ★★☆☆☆ |
| **中景** | 膝盖以上 | 叙事核心、自然对话 | ★★★☆☆ |
| **近景** | 胸口以上 | 强调表情、情感 | ★★★★☆ |
| **特写** | 肩部以上 | 聚焦细节、情绪爆发 | ★★★★★ |
| **大特写** | 局部（眼/手/物） | 强调关键、悬念 | ★★★★★★ |

### 景别选择指南

| 场景类型 | 常规景别 | 情绪高潮景别 | 收尾景别 |
|----------|----------|--------------|----------|
| 对话 | 中景 | 近景 | 特写 |
| 战斗 | 中景→全景 | 近景→特写 | 全景 |
| 独白 | 近景 | 特写 | 大特写 |
| 追逐 | 全景 | 中景 | 全景 |
| 展示 | 全景 | 中景→近景 | 特写 |

### 景别切换原则
- **渐进式**：远景→全景→中景→近景→特写（叙事推进）
- **跳切式**：特写→全景→特写（情绪冲击）
- **禁忌**：相同景别连续超过 3 个镜头

## 运镜控制词

### 基础运镜

| 运镜类型 | 控制词 | 效果 |
|----------|--------|------|
| **推** | slow push-in / push forward | 聚焦、强调 |
| **拉** | slow pull-out / pull back | 交代、释放 |
| **摇** | gentle pan / tilt up-down | 扫描、过渡 |
| **移** | smooth tracking / dolly | 跟随、观察 |
| **跟** | following shot | 强调、临场感 |
| **环绕** | orbit / circle around | 展示、压迫 |

### 进阶运镜

| 运镜类型 | 控制词 | 适用场景 |
|----------|--------|----------|
| **俯冲** | swooping down | 战斗、登场 |
| **拉升** | rising shot | 转场、收尾 |
| **旋转** | spinning / rotating | 眩晕、情绪爆发 |
| **变速** | slow-mo / speed ramp | 强调关键动作 |
| **一镜到底** | continuous shot | 流畅、沉浸 |

### 运镜与情绪匹配

| 情绪状态 | 推荐运镜 | 控制词示例 |
|----------|----------|------------|
| 压抑/沉思 | 缓慢推近 | slow push-in, contemplative |
| 紧张/危机 | 快速切换 | quick cut, handheld shake |
| 愤怒/爆发 | 急剧推进 | rapid push-in, camera shake |
| 悲伤/孤独 | 缓慢拉远 | slow pull-out, isolated |
| 力量/威严 | 环绕+拉升 | orbit around, majestic pull |

### 运镜速度参考

| 速度类型 | 描述 | 适用节奏 |
|----------|------|----------|
| 极慢 | 5-10 秒完成 | 诗意、意境 |
| 慢速 | 3-5 秒完成 | 叙事、情感 |
| 正常 | 1-3 秒完成 | 自然、流畅 |
| 快速 | 0.5-1 秒完成 | 紧张、战斗 |
| 极快 | 0.2-0.5 秒完成 | 爆发、切换 |

## 视觉引导原则

### 原则一：景别递进，按需运动
**核心**：每个镜头景别服务于叙事目的。

### 原则二：情绪外化——镜头替角色说话
**核心**：运镜方式与角色情绪匹配。

## 镜头数量配置

- 5 秒：3-4 镜头，每 1-1.5 秒
- 10 秒：5-7 镜头，每 1-2.5 秒
- 15 秒：7-10 镜头，每 1-2.5 秒
- 30 秒：12-18 镜头，每 1.5-3 秒

## 爽感节奏黄金比例

```
开头钩子：10% 时长，50% 信息密度 → 0.5 秒抓眼球
发展推进：50% 时长，70% 信息密度 → 稳步推进
高潮爆发：30% 时长，100% 信息密度 → 视觉冲击
结尾余韵：10% 时长，30% 信息密度 → 留白收束 + 快切黑屏
```

## 视频衔接规范

**尾帧转场设计**：
- 最后一个镜头必须包含**快切黑屏**效果
- 黑屏时机：最后 0.3-0.5 秒
- 黑屏描述："画面快速切黑"、"瞬间黑屏"、"快速淡出至全黑"

**音乐完整性要求**：
- 配乐必须**完整结束**，不得中途切断
- 结尾音乐描述："音乐渐弱至无声"、"音符自然消散"、"尾音完整收束"

**示例尾帧格式**：
```
[镜头 N-尾帧] 2 秒 大远景
人物描述：...
动作：...
场景：...
【角色】:"..." #1.5 秒
（音效）... #1.0 秒
（音乐）...渐弱至无声 #2.0 秒
**转场处理**：最后 0.3 秒快速切黑，音乐完整收尾
```

## 节奏设计

| 节奏类型 | 适用场景 | 镜头时长 | 特点 |
|---------|---------|---------|------|
| 快节奏 | 战斗/追逐 | 1-1.5 秒/镜头 | 快速切换，跳景别 |
| 中节奏 | 对峙/情感 | 2-3 秒/镜头 | 渐进景别，细节丰富 |
| 慢节奏 | 意境/诗意 | 3-4 秒/镜头 | 远景→近景，慢动作 |

## 爽感禁忌

- ❌ 超过 3 秒静态镜头
- ❌ 超过 5 秒无动作变化
- ❌ 连续相同景别超过 3 个
- ❌ 无目的空镜头

## 镜头组合

**三段式（5-10 秒）**：开场钩子→剧情推进→高潮定格

**四段式（15 秒）**：开场→人物展示→高潮→收束

**六段式（30 秒）**：开场→铺垫×2→高潮×2→收束

## 时间戳分镜（13-15 秒）

```
0-3 秒：[画面 + 镜头]
4-8 秒：[画面 + 镜头]
9-12 秒：[画面 + 镜头]
13-15 秒：[画面 + 镜头]
```

**参数指定**：`[画幅比]2.35:1 + [帧率]24fps + [时长]Xs + [色调/风格]`

**禁止项**：`禁止：文字、字幕、LOGO、水印`

## AI 漫剧行业分析与变现路径

> 🎯 **适用场景**：项目可行性评估、客户报价、工具选择

### 一、行业现状

| 维度 | 说明 |
|------|------|
| **市场热度** | AI漫剧在抖音等平台流量表现突出，直播数据优于传统内容 |
| **成本结构** | 前期投入较大（AI生图/视频工具日消耗可能超过课程学费） |
| **质量壁垒** | 自动化/免费方案质量不可靠，高质量内容需要精细化制作 |
| **收益参考** | 商单：高质量剧约 1000 元/分钟；副业分集更新可日入过千 |

### 二、入局门槛

**硬件要求**：
- 稳定网络连接
- 高性能显卡（推荐 RTX 3060 及以上）
- 充足存储空间（素材管理）

**软件工具链**：
| 环节 | 推荐工具 | 替代方案 |
|------|---------|----------|
| 角色生成 | Midjourney / 即梦 | Stable Diffusion |
| 场景生成 | Midjourney / DALL-E | Stable Diffusion |
| 视频生成 | Seedance 2.0 / Veo 3.1 | 可灵 / 海螺 |
| 配音制作 | 剪映 / 飞书妙记 | TTS 工具 |
| 后期剪辑 | Adobe Premiere / DaVinci | CapCut |

### 三、变现路径

**1. 商单变现**（适合团队）
- 高质量漫剧：1000 元/分钟
- 批量交付：按集数计费
- 甲方需求：品牌定制、IP 改编、教育内容

**2. 平台分成**（适合个人）
- 抖音/快手：流量分成 + 带货
- B站：创作激励 + 充电
- YouTube：广告分成

**3. 知识付费**（适合教练）
- 教程售卖：录播课 + 答疑
- 培训带徒：一对一指导
- 社群运营：付费圈子

**4. IP 衍生**（适合长期）
- IP 授权：周边产品
- 版权交易：影视改编
- 品牌联名：跨界合作

### 四、成本控制建议

1. **工具选择**：根据预算选择，避免过度订阅
2. **素材复用**：建立角色库，一次生成多次使用
3. **流程优化**：标准化 SOP，减少重复劳动
4. **批量生产**：集中生图、集中配音，提高效率
