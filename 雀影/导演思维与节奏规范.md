# 导演思维与节奏规范

## 景别

| 景别 | 范围 | 功能 | 情感 |
|------|------|------|------|
| 远景 | 全景+环境 | 交代场景 | ★☆☆☆☆ |
| 全景 | 全身+周围 | 展示姿态 | ★★☆☆☆ |
| 中景 | 膝盖以上 | 叙事核心 | ★★★☆☆ |
| 近景 | 胸口以上 | 强调表情 | ★★★★☆ |
| 特写 | 肩部以上 | 聚焦细节 | ★★★★★ |
| 大特写 | 局部 | 强调关键 | ★★★★★★ |

### 切换原则

- 渐进式：远景→全景→中景→近景→特写
- 跳切式：特写→全景→特写
- 禁忌：相同景别连续超过 3 个镜头

---

## 运镜

| 类型 | 控制词 | 效果 |
|------|--------|------|
| 推 | push-in | 聚焦 |
| 拉 | pull-out | 释放 |
| 摇 | pan / tilt | 扫描 |
| 移 | tracking | 跟随 |
| 环绕 | orbit | 展示 |
| 俯冲 | swooping down | 登场 |
| 旋转 | spinning | 情绪爆发 |

### 情绪匹配

| 情绪 | 运镜 |
|------|------|
| 压抑 | slow push-in |
| 紧张 | quick cut, handheld shake |
| 愤怒 | rapid push-in |
| 悲伤 | slow pull-out |
| 威严 | orbit + pull |

---

## 镜头数量

| 时长 | 镜头数 |
|------|--------|
| 5s | 3-4 |
| 10s | 5-7 |
| 15s | 7-10 |
| 30s | 12-18 |

---

## 爽感节奏

```
开头钩子：10% 时长，50% 信息密度
发展推进：50% 时长，70% 信息密度
高潮爆发：30% 时长，100% 信息密度
结尾余韵：10% 时长，30% 信息密度 + 快切黑屏
```

---

## 爽感禁忌

- 超过 3 秒静态镜头
- 超过 5 秒无动作变化
- 连续相同景别超过 3 个
- 无目的空镜头

---

## 尾帧处理

- 最后 0.3 秒快速切黑
- 音乐完整收尾
