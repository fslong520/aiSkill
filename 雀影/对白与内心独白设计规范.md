# 对白与内心独白设计规范

## 一、核心理念

**短剧推进剧情之三大音频引擎：对白、内心独白、旁白。**

三者关系：

| 引擎 | 推进什么 | 速度 | 示例 |
|------|---------|------|------|
| 对白 | 角色间矛盾 | 快 | "你骗我。" |
| 内心独白 | 角色内心变化 | 慢 | "原来他从一开始就在骗我。" |
| 旁白 | 时空/背景 | 中 | "三年前的那个雨夜。" |

**最精彩的短剧段落，往往是三层同时推进：**
- 对白在说A
- 内心独白在想B
- 旁白在交代C

三层交织，信息密度极高，张力拉满。

---

## 二、对白设计进阶

### 2.1 信息不对等对白

A知道B不知道 → 张力产生。

```
A："你知道他为什么来吗？"
B："出差吧。"
A："他辞职了。"
（B震惊，沉默）
```

设计思路：让观众知道A知道真相，B不知道。观众站在"全知"视角看两人的对话，比剧中人更紧张。

### 2.2 立场反转对白

对话中途，角色立场翻转，观众的认知也被颠覆。

```
A："你不该来找我。"
B："我必须来。"
A："你知道我是谁吗？"
B："我知道。你是我哥。"
（A沉默——他才是那个不知道的人）
```

### 2.3 沉默对白

不用台词，用反应和停顿传递信息。

```
A盯着B看了五秒。
B移开目光。
A笑了——那笑容比哭还难看。
【内心独白】"果然是你。"
```

## 三、内心独白设计进阶

### 3.1 三层真实原则

角色在短剧中有三层"真实"：

| 层 | 名称 | 谁听见 | 特征 |
|----|------|--------|------|
| 表层 | 对白 | 剧中人 | 社交面具，角色想让人听见的 |
| 中层 | 动作/表情 | 观众看见 | 身体出卖的真实 |
| 深层 | 内心独白 | 只有观众 | 赤裸裸的灵魂 |

好戏的标准：三层不一致。

```
对白："我不在乎。"
动作：手指在发抖
内心独白："我快撑不住了。"
```

### 3.2 内心独白的三个时机

| 时机 | 效果 |
|------|------|
| 对白之前 | 揭示动机，"他要说一个谎……" |
| 对白之间 | 揭示反差，"嘴上说没事，心里在哭" |
| 对白之后 | 揭示真相，"那句话是假的" |

### 3.3 内心独白的节奏

内心独白本身也应有节奏变化，不可全程平铺。

```
慢→快→慢
平静→波动→爆发→空
长句→短句→断句
```

示例：

```
【内心独白】"他来了。" #1.0秒（慢，低语）
【内心独白】"他真的来了……" #1.2秒（微颤）
【内心独白】"十年了，我以为这辈子——" #1.5秒（快，情绪上涌）
【内心独白】"算了。" #0.8秒（突然收住，气声）
```

## 四、对白+内心独白组合模板

### 模板1：真相压制

```
对白："你听我解释。"
内心独白："别说了，越描越黑。"
对白："真的是误会！"
内心独白："他什么都不知道……这样也好。"
（音量逐渐降低，内心独白吞没对白）
```

### 模板2：双重博弈

```
对白（A）："你爱她吗？"
内心独白（A）："求你摇头。"
对白（B）："爱。"
内心独白（A）："好。"
```
——内心独白只有一个字，比任何长篇大论都有力。

### 模板3：角色反转

```
对白（反派）："我才是为你好。"
内心独白（反派）："蠢货，真信了。"

对白（主角）："谢谢大哥。"
内心独白（主角）："等我找到证据……"
```
——正反双方各怀鬼胎，观众同时听到两边的内心，信息量爆炸。

## 五、检查清单

- [ ] 每场对白有一条情绪弧线（从→到，不原地踏步）
- [ ] 所有角色说话方式不同（用词、语速、口癖）
- [ ] 内心独白与对白形成反差或互补
- [ ] 内心独白有专属混响/音量处理
- [ ] 重要对白后留有停顿（0.5-1秒）
- [ ] 避免对白全篇都在"解释剧情"
- [ ] 内心独白不超过对白时长之50%（不可反客为主）
- [ ] 三层（对白+内心独白+音效）同时用时，信息不矛盾
