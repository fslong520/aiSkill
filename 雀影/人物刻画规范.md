## 三视图生成方案

### 为什么要三视图？

三视图（正面 + 侧面 + 背面）是 AI 漫剧角色一致性的核心保障。

**核心原则**：
1. **正面**：五官清晰、表情自然，是角色识别的基础
2. **侧面**：轮廓清晰、发型侧面、后背设计可见
3. **背面**：发型后部、服装背部、配饰完整

### 三视图生成模板

```
【正面视图】
[角色名]-front-view, [面部特征], [发型发色], [服装正面],
[配饰], [站姿], [表情], [光照], [背景]
关键词：front view, full face visible, facing camera

【侧面视图】
[角色名]-side-view, [侧面轮廓], [发型侧面], [服装侧边],
[配饰], [侧身姿态], [光照], [背景]
关键词：side view, profile visible, 90-degree angle

【背面视图】
[角色名]-back-view, [发型后部], [服装背部], [配饰],
[背影姿态], [光照], [背景]
关键词：back view, rear visible, from behind
```

### 三视图一致性要求

| 检查项 | 正面 | 侧面 | 背面 |
|--------|------|------|------|
| 发型 | ✅ 完整 | ✅ 侧边轮廓 | ✅ 后部形态 |
| 服装 | ✅ 前襟设计 | ✅ 侧边线条 | ✅ 背部图案 |
| 配饰 | ✅ 正面可见 | ✅ 侧面可见 | ✅ 背部可见 |
| 肤色 | ✅ 统一 | ✅ 统一 | ✅ 统一 |
| 光照 | ✅ 统一方向 | ✅ 统一方向 | ✅ 统一方向 |

### 多风格一致性方案

**方案一：服装锚点固定**
```
核心锚点：
- 标志性服装（如：红色披风 + 金色纹路）
- 固定发型（如：高马尾 + 左侧银发夹）
- 特征配饰（如：左手银色手镯 + 腰间玉佩）

使用场景：同角色多镜头、多集连续使用
```

**方案二：视角约束**
```
固定视角范围：
- 主视角：正面 45° - 正面 - 侧面 45°
- 避免背面，除非特殊动作需要
- 避免侧脸角度超过 90°

使用场景：追求角色稳定性的连续镜头
```

**方案三：参考图复用**
```
1. 生成一张高质量角色图作为基准
2. 后续所有镜头引用该图：
   - Seedance: @图片1 + [新场景描述]
   - 即梦: 参考图 + [场景调整]
3. 每次引用后补充一致性约束

使用场景：高要求的角色特写镜头
```

### 角色标签系统

**标签格式**：使用一致的标签便于后续引用
```
角色名-视角-表情-服装-场景
```

**示例**：
```
xiaofeng-front-smile-white dress-garden
xiaofeng-side-neutral-formal robe-study
xiaofeng-back-determined-battle armor-field
```

### 多角色场景生成要点

1. **主次分明**：主角占画面 60% 以上
2. **视角差异**：不同角色使用不同视角（正面/侧面/背影）
3. **景别控制**：群像用全景+中景组合
4. **冲突设计**：正邪对立、性格反差

### 角色一致性控制

**角色卡规范**（参考小云雀优秀提示词）：
```
[角色名]: 性别 + 年龄 + 面部特征 + 发型发色 + 服装 + 主要配饰
```

**一致性约束**（用于每次生成）：
- 固定特征：服装、发型、标志性配饰
- 表情范围：可变化表情，但需在角色设定内
- 视角限制：避免生成角色背面（除非特殊需要）

## 五维人物刻画法

每个角色从五个维度立体刻画：

### 1. 外貌维度

**必填要素**：身材体型、发色发型、面部特征、服装款式、配饰特征

**描写原则**：
- 每个特征有目的（体现性格/身份/背景）
- 避免模糊词（好看、美丽），用具体细节
- 服装与身份匹配（战士→铠甲、刺客→紧身衣）
- 质感：皮肤纹理、材质光泽、光影折射

### 2. 性格维度

通过动作/台词/表情展现性格：
- 动作：急躁→动作快、冷静→动作稳
- 台词：话唠→长句、沉默者→短句

### 3. 动作维度

姿态特征、移动方式、战斗风格、习惯性动作
- 快节奏：战斗/追逐/危机
- 慢节奏：对峙/思考/情感
- 动静结合：慢动作→快动作→定格

### 4. 神态维度

眼神特点、微表情（嘴角/瞳孔）、气场、情绪变化

### 5. 气质维度

整体印象：英雄气质/反派气质/仙气飘渺/霸气侧漏/江湖气息

## 角色卡模板

```
【角色卡】
姓名：[角色名称]
身份：[身份定位]

【外貌】身材 + 发型 + 面部 + 服装 + 配饰
【性格】核心性格 + 行为习惯 + 情绪表达
【动作】站姿 + 移动 + 战斗风格 + 细节
【神态】眼神 + 表情 + 气场
【气质】整体印象 + 他人感受
```

## 一致性锚点

| 类型 | 说明 | 示例 |
|------|------|------|
| 服装锚点 | 固定服装特征 | 红色围巾、金色手镯 |
| 发型锚点 | 固定发型特征 | 披肩长发、马尾边发卡 |
| 动作锚点 | 习惯性动作 | 撩头发、托腮 |
| 表情锚点 | 标志性表情 | 微微皱眉、嘴角上扬 |
