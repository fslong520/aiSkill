## 示例1：史诗战斗型（15秒）

**用户需求**：古代战场，多位英雄准备决战

**生成结果**：
```
镜头猛烈向前推近，伴随着强烈的能量冲击波！左侧红衣女将眼神锐利如刀，猛然拔出腰间长剑，剑身爆发出炽热的火焰，她振臂高呼："犯我疆土者，虽远必诛！"。镜头围绕中间的白衣男主和绿衣女主进行快速环绕俯冲，绿衣女主手中的法杖顶端宝石光芒大盛，地面上浮现出古老的魔法阵。右侧金甲武将发出一声震天怒吼，将巨斧重重砸在地面，激起一圈金色的能量涟漪，风云为之变色。背景天空中的流光变成巨大的火流星划过，远处的城市燃起熊熊烽火，整个画面充满了山雨欲来的紧张感和毁天灭地的战斗气势。最后，镜头快速拉远，定格在五位英雄共同释放力量、准备迎接最终决战的宏大史诗场面。
```

**结构分析**：
- 镜头1：推近+左侧人物动作（3秒）
- 镜头2：环绕俯冲+中间人物（4秒）
- 镜头3：右侧人物+环境特效（4秒）
- 镜头4：拉远定格（4秒）

## 使用建议

### 如何选择模板

| 场景类型 | 推荐模板 | 特点 |
|---------|---------|------|
| 战斗/对决 | 连续叙事型 | 能量冲击强，节奏紧凑 |
| 舞蹈/表演 | 分镜脚本型 | 动作细节丰富，节奏可控 |
| 魔法/特效 | 时间轴型 | 精确控制特效节奏 |
| 剧情/反转 | 时间轴型 | 情绪曲线清晰 |
| 产品展示 | 连续叙事型 | 突出产品特性 |

### 关键成功要素

1. **镜头运动要丰富**：避免静态画面，多用推、拉、摇、移、跟
2. **时间分配要合理**：开场2-3秒，高潮4-6秒，收尾2秒
3. **视觉层次要分明**：前景、中景、背景、天空都要有内容
4. **情绪要有起伏**：压抑→紧张→爆发→释放的完整曲线
5. **音效要分层**：环境音→动作音→特效音→BGM→台词
