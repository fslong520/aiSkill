# 场景生成规范

## 三要素

| 要素 | 内容 |
|------|------|
| 环境 | 空间类型、空间大小、地面材质 |
| 氛围 | 时间、天气、情绪基调 |
| 细节 | 装饰物、光影效果、特效元素 |

---

## 溶图合成

### 核心原则

1. 光源统一：角色光照与场景光源匹配
2. 视角一致：角色视线与场景透视一致
3. 比例协调：角色与场景元素比例合理
4. 色调融合：色调统一，无明显割裂

### 质量检查

| 检查项 | 标准 |
|--------|------|
| 光源匹配 | 阴影方向一致 |
| 透视一致 | 地平线高度一致 |
| 色调统一 | 无明显色温差 |
| 边缘融合 | 无明显锯齿/白边 |
| 比例合理 | 角色与环境比例正常 |

### 溶图模板

```
[场景描述], [光照方向], [色调氛围]
+ [角色描述]
+ 融合要求: 光线一致性, 透视统一, 色调协调
```

---

## 常见问题

| 问题 | 解决 |
|------|------|
| 光源方向不一致 | 统一光照描述 |
| 角色悬浮或陷入 | 添加地面接触描述 |
| 角色过大或过小 | 调整相对尺寸 |
| 色调冲突 | 统一色彩描述 |
| 透视不匹配 | 统一视角描述 |
