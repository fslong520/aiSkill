# 场景生成规范

> 📖 **来源**：AI漫剧制作与变现全流程指南报告（2026-04-16）

## 场景构建三要素

每个场景由三个核心要素构成：

### 1. 环境要素
- **空间类型**：室内/室外、开放/封闭
- **空间大小**：宽阔广场/狭小牢房
- **地面材质**：草地/石板/水面/云端

### 2. 氛围要素
- **时间氛围**：晨曦/正午/黄昏/深夜
- **天气氛围**：晴朗/阴雨/风雪/雷暴
- **情绪基调**：压抑/明快/神秘/肃杀

### 3. 细节要素
- **装饰物**：家具/植物/器物/符文
- **光影效果**：自然光/烛光/魔法光
- **特效元素**：粒子/烟雾/能量波

## 溶图合成原则

### 角色与场景融合

**核心原则**：
1. **光源统一**：角色光照必须与场景光源匹配
2. **视角一致**：角色视线方向与场景透视一致
3. **比例协调**：角色与场景元素比例合理
4. **色调融合**：色调统一，无明显割裂感

### 溶图质量检查清单

| 检查项 | 合格标准 |
|--------|----------|
| 光源匹配 | 阴影方向一致，无穿帮 |
| 透视一致 | 地平线高度一致 |
| 色调统一 | 无明显色温差 |
| 边缘融合 | 边缘无明显锯齿/白边 |
| 比例合理 | 角色与环境物体比例正常 |

### 溶图提示词模板

```
[场景描述], [光照方向], [色调氛围]
+ [角色描述]
+ [融合要求]: 光线一致性, 透视统一, 色调协调
```

**示例**：
```
Ancient Chinese palace, soft golden sunlight from the right,
+ A young woman in white dress standing by the window,
+ unified lighting, consistent perspective, harmonious color tone
```

## 场景角色一致性检查

| 检查项 | 问题表现 | 解决方法 |
|--------|----------|----------|
| 光源方向 | 角色阴影与场景不一致 | 统一光照描述 |
| 地面高度 | 角色悬浮或陷入 | 添加地面接触描述 |
| 空间比例 | 角色过大或过小 | 调整相对尺寸 |
| 色调反差 | 角色与场景色调冲突 | 统一色彩描述 |
| 透视角度 | 角色与场景透视不匹配 | 统一视角描述 |

## 不同类型场景生成要点

### 古风场景
- 建筑形制（斗拱/飞檐/榫卯）
- 材质质感（青砖/木质/丝绸）
- 氛围细节（灯笼/香炉/屏风）

### 仙侠场景
- 元素组合（云海/仙山/瀑布）
- 特效氛围（灵气/仙雾/祥云）
- 光影效果（天光/剑光/法阵光）

### 现代场景
- 真实质感（玻璃/金属/皮革）
- 光源类型（霓虹/自然/室内）
- 细节层次（前景/中景/背景）

### 奇幻场景
- 超现实元素（浮空/扭曲/异形）
- 氛围营造（神秘/危险/神圣）
- 光影魔法（发光体/能量/传送门）
