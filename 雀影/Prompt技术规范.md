# Prompt 技术规范

## 万能结构

```
主体 + 动作 + 场景 + 光影 + 镜头/运镜 + 风格 + 画质 + 约束
```

## 可控三件套

| 要素 | 内容 |
|------|------|
| 主体 | 年龄气质、服装配饰、材质形态 |
| 动作 | 动作节奏与连续性（慢、连贯、自然） |
| 镜头 | 景别 + 运镜 + 稳定性 |

## 动作技巧

- ✅ 连续动作链：A→B→C
- ✅ 身体部位 + 小幅动作：眼神、手指、肩颈
- ✅ 节奏词：缓慢、轻柔、连贯、自然
- ❌ 避免高速/剧烈：狂奔、激烈打斗
- ❌ 避免大幅形变：身体扭曲、脸部快速变化

## 镜头规范

| 类型 | 控制词 |
|------|--------|
| 景别 | 特写、近景、中景、全景、远景 |
| 运镜 | 缓慢推近、轻微拉远、平稳平移、稳定跟拍 |
| 稳定 | 固定/云台稳定、无抖动、丝滑 |

## 角色稳定

```
角色卡："黑色齐肩发、浅色针织衫、银色耳环、五官清晰、服装发型一致"
约束：五官清晰、面部稳定不变形、人体结构正常、服装发型不变
```

## 避雷

- 避免模糊词：好看、非常、极其
- 避免自相矛盾：超高速 + 极度稳定
- 避免角色过载：多人复杂互动
- 严禁高风险内容：色情、暴力、版权人物

## 能力模式

| 模式 | 结构 |
|------|------|
| 纯文本 | 主体 + 动作 + 环境 + 镜头 + 风格 |
| 一致性 | [角色]@图片 + [动作] + [场景]@图片 |
| 运镜复刻 | 参考@视频运镜 + [主体]@图片 |
| 特效复刻 | 参考@视频特效 + [元素]替换为@图片 |
| 视频延长 | @视频延长 Xs + [新增内容] |

---

## 分镜脚本格式

```
（时长 X 秒）【标题】（风格）（色调）

【剧作要求】...
【剧情梗概】...
【人物重点】...
【场景重点】...

[镜头 1-首帧] X 秒 景别+运镜 人物 动作（场景）
【角色】:"台词" #Xs
（音效）音效 #Xs
...
[镜头 N-尾帧] X 秒 ... 最后 0.3 秒快速切黑
```

### 旁白与台词

- 旁白：描述画面、动作（括号内），不超过 15 字
- 台词：角色说的话（引号内），不超过 10 字

## Seedance 2.0 语法

| 类型 | 示例 |
|------|------|
| 固定 | static shot, centered composition |
| 推镜 | slow push-in, close-up on face |
| 拉镜 | slow pull-out, revealing environment |
| 摇镜 | gentle pan, left to right |
| 移镜 | smooth tracking shot |
| 慢动作 | slow motion, graceful movement |
| 暖色调 | warm golden lighting |
| 冷色调 | cool blue tones |
| 电影感 | cinematic lighting, volumetric light |
