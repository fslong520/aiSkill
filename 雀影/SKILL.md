## Keywords

雀影, 视频生成, AI视频, 分镜, Seedance, 即梦, prompt, 视频脚本

## Summary

交互式引导完善视频创意，生成分镜脚本和 Seedance 提示词。

## Strategy
1. 用户描述视频内容 → 确认素材、时长、类型
2. 分析故事 → 推荐视觉风格方案
3. 设计角色锚点、镜头节奏、台词音效
4. 确认画面比例 → 生成分镜脚本 → 输出 prompt

AVOID:
- AVOID 不读规范文档就生成，应该先读人物刻画规范、Prompt技术规范、台词与音效规范
- AVOID 镜头数量超出范围，5s=3-4镜，10s=5-7镜，15s=7-10镜，30s=12-18镜
- AVOID 台词全部短句无变化，长短结合、冲突递进、潜台词皆有
- AVOID 忘记内心独白——内心独白是短剧推进剧情之核心手段
- AVOID 忘记尾帧处理，最后0.3秒快速切黑

## 镜头数量

| 时长 | 镜头数 |
|------|--------|
| 5s | 3-4 |
| 10s | 5-7 |
| 15s | 7-10 |
| 30s | 12-18 |

## 台词规范

- 日常对白：每句 4-12 字，自然口语化
- 情绪爆发：每句 2-6 字，语速快（每秒 5-7 字）
- 内心独白：每句 6-15 字，语速慢（每秒 2-3 字），有呼吸感
- 潜台词：言外有意，话说一半，留白让观众品
- 节奏：对白之间留自然停顿，不可连珠炮无呼吸

## 输出格式

```
（时长 X 秒）【标题】（风格）（色调）

【剧作要求】...
【剧情梗概】...
【人物重点】...
【场景重点】...

[镜头 1-首帧] X 秒 景别+运镜 人物描述 动作（场景）
【角色】:"台词" #Xs
【内心独白】"..." #Xs
（音效）音效 #Xs
...
[镜头 N-尾帧] X 秒 ... 最后 0.3 秒快速切黑
```

### 音频三要素（按优先级排列）

| 要素 | 作用 | 字数/时长 |
|------|------|----------|
| 对白 | 角色间交锋、情绪碰撞 | 4-12 字/句 |
| 内心独白 | 角色内心真实想法，观众唯一能听到的心声 | 6-15 字/句 |
| 旁白 | 交代背景、补画面不足、推进剧情 | 8-15 字/句 |

### 音频层次（从近到远）

```
观众耳中 →
  第一层：内心独白（最贴近观众）
  第二层：对白（角色间的交流）
  第三层：旁白（全知视角）
  第四层：环境音（氛围基础）
  第五层：配乐（情绪底色）
  第六层：特效音（动作/转场增强）
```
