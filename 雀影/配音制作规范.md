| 剪映 | 免费、简单 | 入门首选 |

## 制作流程

1. 提取台词，标注角色、情感、对白类型
2. 设计情绪线索：整场对话从开始到结束之情绪变化线
3. 选择音色：年龄、性别、性格、情绪
4. 调整参数：语速、音调、停顿、重音
5. 对白层处理：主对白、内心独白、旁白三层混合
6. 后期处理：降噪、标准化、混响、空间定位

---

## 一、对白情绪线索设计（新增重点）

### 情绪弧线

每场对白须有一条情绪弧线，不可从头到尾一个调：

```
冷→淡→波动→爆发→落
平→紧→更紧→崩→松
笑→微笑→不笑→哭
```

### 标记格式

在分镜脚本中对白后加情绪标记：

```
【角色】:"我真走了。"（→ 平静，准备转身）
【角色】:"我真走了。"（→ 提高音量，试探对方反应）
【角色】:"我真走了。"（→ 爆发，摔门而去）
```

### 对白三种情绪走向

| 走向 | 适用 | 操作 |
|------|------|------|
| 升温式 | 争吵、对峙 | 语速逐渐加快，音量逐渐提高 |
| 降温式 | 和解、认输 | 语速逐渐放慢，音量逐渐降低 |
| 断崖式 | 突然反转 | 中段突然沉默，然后用完全不同的语气说话 |

## 二、内心独白配音规范（新增）

### 录音处理

| 要素 | 规范 |
|------|------|
| 空间感 | 加轻微混响（2-5%），与对白区分 |
| 音量 | 比对白低 20-30%，营造"心底的声音" |
| 语速 | 比对白慢 30%，每秒 2-3 字 |
| 气息 | 保留呼吸声、换气声，增强真实感 |
| 特效 | 情绪强烈时可加低频嗡鸣作为背景 |

### 内心独白与对白叠加处理

```
同一角色，同时有对白和内心独白时：

对白（正常音量，近距感）→ 说给别人听
内心独白（音量低20%，带混响）→ 说给自己/观众听

两层叠加，对白为主，内心独白为"画外心声"
```

## 三、音量标准

| 类型 | 音量 | 优先级 |
|------|------|--------|
| 对白（主） | -6dB 到 -3dB | ★★★★★ |
| 内心独白 | -10dB 到 -7dB | ★★★★ |
| 旁白 | -8dB 到 -5dB | ★★★★ |
| 音效（动作/特效） | -6dB 到 0dB | ★★★ |
| 心理音效 | -12dB 到 -8dB | ★★ |
| 背景音乐 | -15dB 到 -10dB | ★★ |
| 环境音 | -12dB 到 -9dB | ★★ |
| 峰值限制 | 不超过 -1dB | — |

## 四、对白叠层与衔接规范（新增）

### 对话重叠

两角色话头撞话尾，营造真实感：

```
A:"你到底想—"
B:"我想走！"
```

标注方式：
```
【A】:"你到底想—" #0.5秒 [→与B重合]
【B】:"我想走！" #0.6秒 [↗接A尾，爆发]
```

### 对话打断

```
【A】:"听我说—"
【B】:"够了。"（抢话，音量+20%）
```

### 对话后留白

重要对白后留 0.5-1 秒纯音效/环境音，让观众消化情绪。

## 五、常见问题

| 问题 | 解决 |
|------|------|
| 录音模糊 | 使用 44.1kHz 以上采样率 |
| 电流声 | 降噪处理或更换设备 |
| 音量不一 | 分段调整 |
| 口型不匹配 | 调整语速或编辑时间轴 |
| 对白平无起伏 | 标注情绪弧线，逐句调语调 |
| 内心独白听不出 | 加混响+降音量，和正常对白区分 |

## 六、多角色配音

```
主角A：年轻女性，温柔有力量 → 音色清澈，语速适中，内心独白带气声
主角B：成熟男性，沉稳冷静 → 音色低沉，语速较慢，内心独白加低频
反派C：阴险狡诈 → 音色尖锐，语速较快，有"假笑"感
配角D：憨厚可爱 → 音色圆润，带口音，内心独白几乎没有（心直口快）
```

---

## 同步原则

- 台词开始 = 口型动作开始
- 内心独白：无需口型同步，画面定格或特写
- 误差容忍：±0.2 秒
- 情感爆发点可提前（声音先行）
- 内心独白可延后 0.3 秒（先有表情再有内心活动）

---

## 完整音频层次（更新版）

```
第六层（最远）：环境音 — 氛围基础（30-40%）
第五层：背景音乐 — 情绪底色（40-50%）
第四层：心理音效 — 内心独白专属（30-50%）
第三层：动作/特效音 — 视觉增强（60-80%）
第二层：旁白 — 叙事补充（80%）
第一层（最近/主体）：对白 + 内心独白（100%）

核心原则：对白和内心独白在最近层，观众听感最清晰
```
