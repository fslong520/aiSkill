## 配音制作流程

### 步骤 1：台词整理
```
1. 提取分镜脚本中的所有台词
2. 标注每句台词的说话角色
3. 标注情感状态（平静/激动/悲伤/愤怒）
4. 统计总时长，匹配视频长度
```

### 步骤 2：音色选择
```
根据角色设定选择音色：
- 年龄：年轻/成熟/老年
- 性别：男声/女声/童声
- 性格：温柔/强势/俏皮/冷酷
- 情绪：平静/激动/低沉/高亢
```

### 步骤 3：情感参数调整
| 参数 | 说明 | 调整建议 |
|------|------|----------|
| 语速 | 字/秒 | 激动时加快，悲伤时放慢 |
| 音调 | 高低 | 男性低沉，女性高亢 |
| 停顿 | 句间/段间 | 强调前停顿，留白感 |
| 重音 | 关键词强调 | 重要信息加重音 |

### 步骤 4：后期处理
```
1. 降噪处理（去除杂音）
2. 音量标准化（-3dB 到 -6dB）
3. 添加混响（增加空间感）
4. 音效融合（脚步/呼吸等）
```

## 声音处理参数

### 音量标准化
```
- 语音音量：-6dB 到 -3dB
- 背景音乐：-12dB 到 -9dB
- 音效音量：-6dB 到 0dB
- 峰值限制：不超过 -1dB
```

### 常见问题处理

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 录音模糊 | 采样率过低 | 使用 44.1kHz 以上 |
| 电流声 | 设备接地不良 | 降噪处理或更换设备 |
| 音量不一 | 录音距离变化 | 分段调整或重新录音 |
| 口型不匹配 | 语速偏差 | 调整语速或编辑时间轴 |

## 多角色配音要点

### 音色差异化
```
主角A：年轻女性，温柔但有力量 → 音色清澈，语速适中
主角B：成熟男性，沉稳冷静 → 音色低沉，语速较慢
反派C：阴险狡诈 → 音色尖锐，语速较快，配合笑声
配角D：憨厚可爱 → 音色圆润，语速较慢，带口音
```

### 情感层次表达
```
情绪递进示例：
1. 平静："这件事...我需要想想。"
2. 疑惑："你说什么？"
3. 震惊："怎么可能！"
4. 愤怒："你们骗我！"
5. 崩溃："为什么...为什么是我..."
```

## 配音与画面同步

### 时间轴对齐原则
```
- 台词开始时间 = 角色口型动作开始时间
- 误差容忍度：±0.2 秒
- 关键动作（如张嘴、闭嘴）需精确对齐
- 情感爆发点可适当提前（声音先行）
```

### 同步检查清单
```
□ 台词时长是否匹配口型动作
□ 重音位置是否与表情动作同步
□ 停顿位置是否有画面配合
□ 情绪转折是否与运镜节奏匹配
□ 多人对话是否有自然的重叠/间隔
```

## 音效与配音配合

### 音效层次
```
第一层（背景）：环境音、氛围音乐
第二层（主体）：角色配音、对话
第三层（点缀）：动作音效、情绪音效

混合比例建议：
配音 > 背景音乐 > 环境音 > 点缀音效
  100%      60%          40%         20%
```
