# 配音制作规范

## 工具选择

| 工具 | 优点 | 适用场景 |
|------|------|----------|
| 剪映 | 免费、简单 | 入门首选 |
| Azure TTS | 音色自然 | 专业制作 |
| ElevenLabs | 克隆音色 | IP 角色声线 |

---

## 制作流程

1. 提取台词，标注角色、情感
2. 选择音色：年龄、性别、性格、情绪
3. 调整参数：语速、音调、停顿、重音
4. 后期处理：降噪、标准化、混响

---

## 音量标准

| 类型 | 音量 |
|------|------|
| 语音 | -6dB 到 -3dB |
| 背景音乐 | -12dB 到 -9dB |
| 音效 | -6dB 到 0dB |
| 峰值限制 | 不超过 -1dB |

---

## 常见问题

| 问题 | 解决 |
|------|------|
| 录音模糊 | 使用 44.1kHz 以上采样率 |
| 电流声 | 降噪处理或更换设备 |
| 音量不一 | 分段调整 |
| 口型不匹配 | 调整语速或编辑时间轴 |

---

## 多角色配音

```
主角A：年轻女性，温柔有力量 → 音色清澈，语速适中
主角B：成熟男性，沉稳冷静 → 音色低沉，语速较慢
反派C：阴险狡诈 → 音色尖锐，语速较快
配角D：憨厚可爱 → 音色圆润，带口音
```

---

## 同步原则

- 台词开始 = 口型动作开始
- 误差容忍：±0.2 秒
- 情感爆发点可提前（声音先行）

---

## 音效层次

```
第一层（背景）：环境音、氛围音乐
第二层（主体）：角色配音
第三层（点缀）：动作音效、情绪音效

比例：配音 > 背景音乐 > 环境音 > 点缀音效
       100%      60%        40%       20%
```
