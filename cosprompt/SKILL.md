# Cosprompt - 结构化 AI 绘图提示词生成器

## 这个技能是做什么的？

**一句话**：输入角色名，输出结构化的 AI 绘图提示词。

**使用场景**：
- 想用 Midjourney/SD 生成角色海报，不知道怎么写提示词
- 想生成 Cosplay 海报、电商海报、角色立绘
- 需要结构化、专业化的提示词模板

**工作流程**：
```
用户输入角色名 → AI 搜索角色信息 → 分析标志性特征 → 决定合适风格 → 输出结构化提示词
```

**输出格式**：
- JSON 格式（便于程序调用）
- Markdown 格式（便于阅读复制）
- 纯文本版（直接复制到绘图工具）

---

## 核心理念

**提供思考框架，不写死内容。**

AI 自主完成：
1. 搜索/分析角色信息
2. 决定风格方向
3. 填充具体内容
4. 输出结构化提示词

---

## 金字塔Prompt架构（总纲）

输出按宏观→微观递进，层层约束。角色prompt尤重第六层（服装）与第三层（微表情），cosplay成败在此。

```
第1层 空间与构图    → 比例、场景、世界观背景（定调）
第2层 主体与人物    → 角色名、年龄、种族概貌（锚点）
第3层 微相与情绪    → 五官、微表情、眼神量化、角色气质（传神）
第4层 姿态与动势    → 站立/动态、武器/道具交互、衣摆联动（生姿）
第5层 服装与质感    → 款式、面料、铠甲/布料/配饰、标志性细节（定性）
第6层 光影与镜头    → 光质、色温、焦段、景深、质量标签（定质）
```

每层须呼应角色世界观。不可出现"第1层仙侠古风，第6层赛博朋克"之语义冲突。

---

## 思考维度框架（17 维度）

### 一、主体层（6 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **subject** | 主体是什么？ | 角色、产品、场景 |
| **age** | 年龄设定？ | 少女、青年、成熟、御姐、萝莉 |
| **model** | 身体特征？ | 体型、肤色、比例 |
| **face** | 面部特征？见"微表情精度控制" | 五官、表情、眼神方向 |
| **pose** | 姿态动作？ | 站姿、坐姿、动态、武器/道具交互 |
| **hair** | 发型特征？ | 颜色、长度、造型、发饰 |

### 二、服装层（1 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **costume** | 服装特征？见"服装深度描述" | 款式、材质、颜色、铠甲/布料、标志性配饰、版型 |

### 三、风格层（2 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **style** | 整体风格？ | 杂志风、电影风、东方玄幻、赛博朋克、极简 |
| **mood** | 情绪氛围？ | 梦幻、冷酷、温暖、神秘、寂寥 |

### 四、环境层（3 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **environment** | 场景环境？ | 室内、室外、虚拟、抽象、根据世界观 |
| **lighting** | 光影效果？见"光质描写指南" | 自然光、舞台光、霓虹、柔光 |
| **composition** | 构图方式？ | 比例、视角、焦点、留白 |

### 五、镜头层（2 维度）← 新增

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **lens** | 镜头语言？见"镜头语言指南" | 焦段、光圈、景深、透视、电影感 |
| **quality_tags** | 质量标签？见"质量标签体系" | 渲染精度、平台特定词、风格定调词 |

### 六、文字层（1 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **typography** | 文字排版？ | 标题、副标题、字体风格、层级 |

### 七、控制层（2 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **negative** | 避免什么？ | 常见错误、不需要的元素 |
| **aspect_ratio** | 画面比例？ | **根据用户需求决定** |

## 层级化描述法则

每个维度的描述遵循三层结构：

```
维度名称
├── 基础描述（一句话定义核心）
├── 特征列表（3-5 个关键词）
└── 细节补充（可选，具体参数）
```

关键：**每层都须考量正反双向约束**——不仅要写"是什么"，还要写"不是什么"。

## 美学深度维度（可选）

对于追求更高美学品质的Prompt，可增加以下思考：

### 情感内核（悦意层）
- 这个角色/场景的情感本质是什么？
- 画面应触发观者的何种情绪？
- 关键词：孤独、温暖、力量、脆弱、神秘、希望...

### 隐喻表达（悦神层）
- 有无象征性元素可融入？
- 是否需要留白或开放性构图？
- 文化/哲学内涵如何通过视觉传达？

### 示例
**普通Prompt**：角色海报，赛博朋克风格
**深度Prompt**：角色海报，赛博朋克风格，情感内核是"高科技低生活"的孤独感，通过霓虹灯下的阴影和模糊的路人传达

## 微表情精度控制框架（face 维度强化）

角色prompt中，表情是角色灵魂的窗口。模糊的"微笑""冷酷"远不如精确的微表情描述。

### 六维微表情控制

| 控制维度 | 描述方法 | 示例 |
|---------|---------|------|
| **眼神方向** | 量化+锚点：直视度0~100% + 目光落点 | "眼睑微垂约1/5，目光落于镜头前寸许" |
| **嘴部细节** | 开合度+嘴角方向 | "唇角微扬不露齿，不刻意微笑" |
| **眉毛/眼睑** | 高度+弧度量化 | "眉微蹙，眼睑半垂" |
| **头部倾斜** | 方向+角度 | "头微左倾约10度" |
| **角色气质** | 将角色性格融入表情 | "神情冷冽中带一丝悲悯" |
| **克制美学** | 负向约束：不做什么 | "不夸张大笑，不做作表情" |

### 眼神精度分级

| 级别 | 描述 | 效果 |
|------|------|------|
| 直视镜头 | 100%正视，目光相接 | 冲击、对峙、邀请 |
| 镜头附近 | 目光落于镜头前后 | 互动感+遐想空间 |
| 微垂（1/5~1/3） | 眼睑轻度下垂 | 慵懒、温柔、内敛 |
| 半垂（1/3~2/3） | 明显下垂 | 沉思、疏离、忧郁 |
| 侧视 | 目光转向画外 | 叙事感、猜疑、向往 |
| 闭目 | 完全闭合 | 沉浸、享受、冥想 |

### 执行要求

填充face维度时，必须考量至少3个微表情维度，并将角色性格融入。
克制美学优先——能用"不做什么"指导的，比"做什么"更有力。

## 服装深度描述框架（costume 维度强化）

Cosplay和角色prompt中，服装是第一识别要素。须覆盖五层：

```
第1层 款式与轮廓    → 连衣裙/铠甲/长袍，剪裁特征（外观）
第2层 面料与质感    → 丝绸/金属/皮革/棉麻，光泽度（触觉）
第3层 颜色与花纹    → 主色、副色、纹样、渐层（视觉）
第4层 版型与结构    → 修身/宽松/多层/硬质，不强调什么（约束）
第5层 配饰与标志性  → 项圈/腰带/徽章/武器/头饰（点睛）
```

### 版型约束（关键）

cosplay服装描述中，"不"和"要"同等重要：

| 约束类型 | 示例 |
|---------|------|
| 合身约束 | "贴合身形但不紧绷" |
| 裸露约束 | "不露肤，高领长袖，符合角色设定" |
| 结构约束 | "铠甲硬质有金属光泽，内衬柔软" |
| 还原约束 | "完全还原角色原设，不改动配色" |

### 触觉词汇表

prompt中加入触觉词，AI对材质的渲染精度显著提升：

| 触觉词 | 对应材质 | 视觉暗示 |
|--------|---------|---------|
| 轻盈飘逸 | 雪纺、真丝 | 半透明、随风动 |
| 厚重垂坠 | 羊毛、呢绒 | 自然褶、有分量 |
| 坚硬冷冽 | 金属、铠甲 | 反光、高光锐利 |
| 柔滑细腻 | 缎面、丝绒 | 柔和反光、华贵 |
| 粗粝纹理 | 皮革、粗麻 | 亚光、质感强 |

## 光质描写指南（lighting 维度强化）

光影不分"亮"与"暗"，分"质"与"温"。

### 光质九宫格

```
             硬光（点光源）               柔光（大面积光源）
      ┌─────────────────────┬─────────────────────┐
  暖  │  暖硬：阳光/钨丝灯     │  暖柔：黄昏/暖白漫射     │
  色  │  戏剧感、强烈阴影      │  温馨、柔和、通透        │
      ├─────────────────────┼─────────────────────┤
  冷  │  冷硬：月光/无影灯     │  冷柔：阴天/柔光箱       │
  色  │  清冷、锐利、科技感    │  平静、均匀、纯净        │
      ├─────────────────────┼─────────────────────┤
  混  │  混硬：多色点光源      │  混柔：环境光+补光       │
  合  │  霓虹、舞台、冲突      │  电影感、层次丰富        │
      └─────────────────────┴─────────────────────┘
```

### 技术控制四要素

| 要素 | 描述 | 示例 |
|------|------|------|
| 光源色温 | 暖白/冷白/中性 | "暖白色室内漫射光" |
| 光线方向 | 顶光/侧光/背光/漫射 | "环境光均匀柔和" |
| 高光控制 | 过曝/细腻/高反差 | "高光细腻不过曝" |
| 特效 | Bloom/光晕/辉光/光束 | "轻微Bloom柔光效果" |

### 空气感

"空气感充足"意谓：空间中有介质感，色调通透不闷，主体与背景间有呼吸空间。
表达方式：直接用"空气感""通透感""呼吸感"等词写入prompt。

## 镜头语言指南（lens + composition 维度）

### 焦段选择

| 焦段 | 透视特性 | 适用场景 |
|------|---------|---------|
| 35mm | 小广角，略强调纵深 | 环境人像，展示服装全貌 |
| 50mm | 标准视角，自然真实 | 半身角色，平衡环境与人物 |
| **85mm** | **中长焦，浅景深，无畸变** | **角色人像黄金焦段** |
| 135mm | 长焦，空间压缩感强 | 面部特写，铠甲/细节质感 |
| 200mm+ | 超长焦，极浅景深 | 远摄意境，虚化极致 |

### 景深三档

| 景深 | 描述方式 | 效果 |
|------|---------|------|
| 浅 | "大光圈浅景深，背景自然虚化" | 突出角色，杂志风 |
| 中 | "中等光圈，前后景适度清晰" | 展示环境与角色关系 |
| 深 | "小光圈全景清晰" | 全方位展示服装细节 |

### 电影感三板斧

1. **色彩叙事** — 限定色调板（"暗金+墨绿"）
2. **画面呼吸** — 留有负空间（"留白充足"）
3. **纹理真实** — 强调材质（"真实皮肤纹理，铠甲金属质感"）

## 正反双向约束原则

每个prompt维度的描述须同时考虑：

### 正向指令（Positive）
描述"要什么"——具体、可感知、可渲染
> "暖白色漫射光，环境光均匀柔和"

### 负向约束（Negative-in-Place）
描述"不要什么"——嵌入主体描述中，而非只塞进negative prompt
> "不直视镜头，不刻意微笑，不强调身体曲线"

### 为什么嵌入比--no有效？

`--no`或negative prompt是宽泛过滤，而主体描述中的"不…"是精确语义控制。
AI对主描述中的否定词敏感度远高于negative prompt。

### 执行要求

**每个维度在填充时，都必须自问：**
- 这个维度的正向描述完整了吗？
- 有什么AI容易"默认生成"但我想避免的？
- 能不能加一个"不…"来控制？

**face和costume维度尤其需要负向约束。**

## 执行流程

### 步骤 1：理解需求

询问用户：
- 生成什么主题的提示词？（角色/产品/场景）
- 有什么特殊要求？（风格偏好、用途、画面比例）

### 步骤 2：收集信息

**AI 自主完成**：
- 搜索角色/主题的背景资料
- 分析标志性特征
- 确定世界观设定

### 步骤 3：决定风格

**AI 自主判断**：
- 根据主题特性选择合适风格
- 参考用户偏好调整
- 保持风格一致性

### 步骤 4：组织架构（参照金字塔架构）

按金字塔六层组织思路，从宏观到微观：
1. 定调（空间+比例）→ 2. 锚点（角色概貌）→ 3. 传神（微表情+角色气质）
4. 生姿（姿态+道具）→ 5. 定性（服装还原）→ 6. 定质（光+镜头+标签）

确保跨维度一致性——六层讲同一个角色故事。

### 步骤 5：逐个填充维度（参照正反双向约束）

按17维度框架逐一填充，每维度自问：
- 正向描述完整否？（含角色特征）
- 负向约束充足否？（需要"不……"吗？）
- 细节具体否？（能用数字/材质量化吗？）

### 步骤 6：输出提示词

提供三种格式：
1. **JSON 格式** — 结构化，便于程序调用
2. **Markdown 格式** — 便于阅读和复制
3. **纯文本版** — 直接复制到绘图工具

## 注意事项

1. **AI 自主决策**：风格、内容、细节由 AI 根据主题特性自主决定
2. **用户偏好优先**：用户有明确要求时（风格、比例、年龄等），优先遵循用户偏好
3. **灵活调整**：维度数量可根据主题需要增减
4. **版权意识**：商业用途需注意角色版权
5. **正反双向约束**：每个维度同时考虑"要什么"和"不要什么"
6. **金字塔递进**：从空间到镜头，层层约束，跨维度一致性
