---

C++在C语言的基础上增加了 **面向对象程序设计** ，这是一种更加高级的编程思想和功能，将一个复杂的程序拆分抽象成若干个对象，然后每个对象都有若干`功能`和`属性`，通过在必要时候执行必要功能来实现这个程序，在**大型软件、游戏**中尤为适合。

## 三、类的使用(30分钟)

<!-- _class: cols-2 -->


<div class="ldiv">

### (一)、开胃菜

```cpp
class 宠物
{
public:
    string 名字;
    int 年龄;    
    宠物(string 取的名字, int 多大了)
    {
        名字 = 取的名字;
        年龄 = 多大了;
    }    
    void 自我介绍()
    { 
        cout << "我是" << 名字 << "，今年" << 年龄 << "岁啦！" << endl;
    }    
    void 卖萌()
    {
        cout << 名字 << "：主人主人，快来陪我玩嘛~" << endl;
    }
};
```
</div>

<div class="rdiv">

```cpp
// 宠物类
int main()
{
    宠物 小猫咪("咪咪", 2);
    小猫咪.自我介绍();
    小猫咪.卖萌();
    return 0;
}
```

> 📌 当我们需要创建多个具有相同属性和行为的对象时，可以使用类来定义模板。
> 例如在游戏中创建多个宠物角色，每个宠物都有名字和年龄属性，以及自我介绍和卖萌等行为。这种方式比为每个宠物单独定义变量和函数更加高效和便于管理。
</div>


---

**总结**：
- 类模板和类的语法相似，在声明模板template后面加类，此类称为类模板
- 类模板使用只能用显示指定类型方式
- 类模板中的模板参数列表可以有默认参数

> 📌 使用场景说明：类模板适用于需要为多种数据类型提供相同功能的场景。例如STL中的容器类(vector、list、map等)，它们可以存储不同类型的数据(int、string、自定义类型等)但操作方式相同。通过类模板，我们可以编写一次代码就能支持多种数据类型，大大提高了代码的复用性和灵活性。在开发通用数据结构和算法库时非常有用。

### 3、课后作业

今天的作业会很有趣哦！让我们来创建一些有趣的对象类：

1. 创建一个"宠物"类，让它有名字、年龄等属性，还有吃饭、玩耍等方法。可以创建不同的宠物对象，比如小狗、小猫等。

2. 创建一个"游戏角色"类，包含角色的名字、生命值、攻击力等属性，以及攻击、防御、使用道具等方法。试着创建几个不同的游戏角色。

3. 创建一个"学生"类，包含姓名、学号、各科成绩等属性，以及计算平均分、判断是否及格等方法。

4. 挑战题：创建一个"魔法学院"类，它可以包含多个学生对象，实现招生、开除、查询学生信息等功能。

试试看你能不能用上我们今天学到的构造函数、成员函数、访问控制等知识！
