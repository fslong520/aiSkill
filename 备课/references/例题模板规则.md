# 讲义例题模板规则

所有例题必须使用以下模板格式：

```markdown
<!-- _class: cols-2 -->
#### 章节号 <a href="题目链接" target="_blank">题目标题</a>
<div class="ldiv">

**题目描述**    
- 题目描述内容
   
**输入**  
1. 输入格式1
2. 输入格式2
   
**输出**  
1. 输出格式
</div>
<div class="rdiv">

**输入样例**  
```
输入示例内容
```
**输出样例**  
```
输出示例内容
```

**算法提示**：
- 算法相关提示内容

###### <a href="题目链接" target="_blank">点我提交评测</a>
</div>
```

## 注意事项：
1. 必须使用 `<!-- _class: cols-2" -->` 开头
2. 左侧使用 `<div class="ldiv">`，右侧使用 `<div class="rdiv">`
3. 题目链接必须设置为 `target="_blank"`，在新标签页打开
5. 必须包含 "点我提交评测" 链接
