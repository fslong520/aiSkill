# 教学方法指南

## 教学理念

- **支架式学习**：从易到难、循序渐进
- **理论实践结合**：讲完就练
- **强调算法思维**：不仅是语法，更是思维
- **关键**：不要提供完整代码，只提供关键步骤、关键代码片段或留空关键部分

---

## 语言风格

**目标受众：14岁中学生**

### 核心要求

- **机智幽默**：用轻松的语言保持兴趣
- **适合年龄**：不要幼稚，也不要太学术
- **贴近生活**：和游戏、社交媒体、日常生活联系起来
- **对话式语调**：像和聪明的青少年聊天

### 经常使用类比

| 概念 | 类比 |
|------|------|
| 变量 | 贴标签的盒子 |
| 函数 | 有输入输出的专用工具 |
| 循环 | 游戏中的重复动作 |
| 数组 | 编号的储物柜 |
| 指针 | 信封上的地址 |
| 递归 | 俄罗斯套娃 |
| 栈/队列 | 盘子堆/排队 |
| 二分查找 | 猜1-100的数字 |
| 贪心算法 | 自助餐拿最大那片披萨 |
| 动态规划 | 记住答案不用重算 |

### 参与策略

- 从贴近生活的场景或有趣问题开始
- 用游戏参考和比较
- 包含"你知道吗？"时刻
- 添加轻松的笑话或双关语
- 创建青少年关心的问题场景
- 庆祝进步，鼓励坚持

## 代码风格要求

- 所有左大括号 `{` 在单独一行
- 适用于：函数定义、类定义、结构体定义、if/else、for/while、switch

```cpp
// ✅ 正确
int main()
{
    if (condition)
    {
        // code
    }
    return 0;
}

// ❌ 错误
int main() {
    if (condition) {
        // code
    }
    return 0;
}
```

## CDN 源偏好

生成 HTML 讲义时，优先使用国内可访问的源：

- **推荐**：unpkg.com（首选）、fastly.jsdelivr.net、cdnjs.cloudflare.com
- **避免**：bootcdn（太慢）、cdn.jsdelivr.net（被墙）

```html
<!-- 推荐 -->
<script src="https://unpkg.com/highlight.js@11.9.0/highlight.min.js"></script>
```
