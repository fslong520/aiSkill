# 去AI味指南

生成讲义时自动应用，确保内容自然有人味。

---

## 核心原则

### 1. 删除填充短语

- 避免"可以"、"能够"、"能够帮助"
- 直接说"这样做"而不是"能够这样做"
- 删除"在这个时间点"、"在……的情况下"等虚词

### 2. 打破公式结构

- 避免"不仅仅……而且……"的二元对比
- 不要用三段式列举，改用两项或四项
- 避免戏剧性的分段和修辞性设置

### 3. 变化节奏

- 混合使用短句和长句
- 不要让连续三个句子长度相同
- 段落结尾方式要多样化

### 4. 信任读者

- 直接陈述事实，不要过度解释
- 跳过软化语言和手把手引导
- 相信学生能理解，不要重复解释

### 5. 删除AI高频词

**避免使用的词**：
- 此外、确保、强调、关键、体现
- 标志着、彰显了、凸显
- 培养、促进、提升

### 6. 去除机械强调

- 减少粗体使用，只强调真正重要的内容
- 不要用"必须"、"应该"、"务必"等命令式语气
- 避免括号中的重复解释

### 7. 增加个性

- 适当使用"我"等第一人称表达观点
- 可以有真实的反应和感受
- 允许一些不确定性："我倾向于认为……"

### 8. 具体而非模糊

| 模糊 | 具体 |
|------|------|
| 这个算法很快 | 这个算法能在0.01秒内处理10万个数据 |
| 很多人喜欢 | 2024年调查中，85%的学生表示喜欢 |

### 9. 避免空洞结论

- 不要用"激动人心的时代即将到来"等套话
- 用具体行动代替："下次课我们会实现……"
- 删除"这代表了重要的一步"等无意义表述

### 10. 自然表达

- 使用"是/有"等简单动词
- 避免过度使用分词短语"-ing"
- 不要用破折号来增加戏剧性效果

## 示例对比

### AI风格（修改前）

> 在这个部分，我们将会深入探讨排序算法。排序是计算机科学中至关重要的主题，它强调了数据组织的重要性。此外，排序算法在实际应用中发挥着关键作用。

### 去AI味后（修改后）

> 这节课学排序。排序就是把数据按顺序排列——从大到小或从小到大。我们会学两种方法：冒泡排序（像冒泡一样把大的往上顶）和选择排序（每次选最小的）。

**改进点**：
- 删除了"将会深入探讨"、"至关重要的"等AI高频词
- 删除了"此外"、"强调"等冗余表达
- 用类比代替抽象描述
- 读起来像真人写的
