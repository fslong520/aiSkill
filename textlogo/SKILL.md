# 文字主体创意 Logo 生成器

## Overview

把用户输入的词/短语变成"文字即图形"的创意 Logo 方案 + AI 生图提示词。文字本身通过变形、连接、正负形、轮廓控制构成图形，而非"图标+文字"排版。

## Core Principles — 不可违背

1. 文字是 Logo 唯一主体
2. 图形由文字本身构成
3. 不在文字旁加独立图标
4. 不生成普通字体排版
5. 不生成插画/场景/照片
6. 文字必须清晰可读
7. 图形须与词义/行业/品牌气质相关
8. 默认黑白极简（用户指定颜色时适配彩色）+ 纯白/干净背景 + 居中构图
9. 输出须可直接用于 AI 生图

## When to Use

- 用户说"设计一个logo"、"做个字标"、"文字logo"
- 用户说"帮我生成logo的AI提示词"、"logo prompt"
- 用户给一个词/短语，要变成Logo创意
- 用户需要多个Logo方案方向
- 需要特定风格（国潮/科技/运动/餐饮/童趣）的Logo

## When NOT to Use

- 用户要的是插画、海报、真实场景画面（不是 Logo）
- 用户要的是图标设计而非文字为主体
- 用户明确要"图标+文字"组合而非纯字标
- 用户要的是 3D 渲染或写实风格

## Input Analysis Flow

收到用户输入后，按以下步骤思考：

1. **词义判断** — 输入是英文词/中文词/短语？含义是什么？
2. **最强视觉意象提取** — 词义能变成什么形状？（咖啡→杯子，山→山峰，火→火焰）
3. **Logo气质判断** — 适合什么品牌调性？（高端/运动/文创/科技/亲和）
4. **字形方法选择** — 用哪种图形化方法？（轮廓/嵌合/连笔/重组/正负形）
5. **生成输出** — 按输出格式组织内容

## Typographic Methods — 五种字形图形化方法

### 1. 轮廓法 (Silhouette Method)

整个文字形成完整物体轮廓。

| 输入 | 轮廓意象 |
|------|----------|
| Coffee | 咖啡杯 |
| Shoes | 运动鞋 |
| Mountain | 山峰 |
| Fish | 鱼 |
| 月亮 | 月牙 |

### 2. 嵌合法 (Embedding Method)

某个关键字母或汉字承担主要图形功能。

| 输入 | 嵌合方式 |
|------|----------|
| Ocean | O 形成海浪旋涡 |
| Flame | A 形成火苗 |
| 山峰 | "山"形成山体结构 |
| 竹影 | "竹"形成竹节 |

### 3. 连笔法 (Ligature Method)

字母/笔画自然连接形成统一图形。

| 输入 | 连接意象 |
|------|----------|
| Coffee | 字母上方形成蒸汽 |
| Dream | 字母形成云朵曲线 |
| 风 | 笔画形成气流 |
| 云 | 笔画形成云形 |

### 4. 结构重组法 (Structural Recomposition)

保持可读性基础上艺术化重构字型结构。
适合：科技品牌、运动品牌、文创品牌、国潮品牌、个人 IP

### 5. 正负形法 (Negative Space Method)

利用黑白留白在文字内部/外轮廓形成隐藏图形。

| 输入 | 负形意象 |
|------|----------|
| Cat | 字母负形中形成猫耳 |
| Bird | 字母负形中形成鸟翼 |
| 山海 | 笔画之间形成海浪 |
| 茶 | 字内留白形成茶杯 |

## Output Format

按以下结构输出：

### 1. 创意方向

一句话核心创意。

### 2. 设计说明

- 文字如何构成图形
- 哪些字母/汉字承担视觉功能
- 整体轮廓与词义的关系
- 整体品牌气质

### 3. 生图提示词

完整的 AI 生图提示词（中文/英文均可，直接可用）。
- 默认黑白极简；用户指定颜色时，颜色须附着于文字笔画而非独立色块
- 描述颜色与图形意象的关系（如"金色字母构成日轮""蓝色渐变如海浪"）

### 4. 负面提示词

需要避免的内容列表。

## Short Output Mode

用户说"只要提示词"、"简短"时，只输出：

```
## 生图提示词
...

## 负面提示词
...
```

## Multi-Plan Mode

用户要求多个方案时，每个方案独立输出完整结构：

```
## 方案一：{方案名称}

### 创意方向
### 设计说明
### 生图提示词
### 负面提示词
```

方案须在以下维度有明确差异：
- 图形轮廓
- 字形处理方法
- 品牌气质
- 视觉意象
- 构图方向

## Style Customization

### 高端品牌风
克制、精致、高级、留白、稳定、品牌感
配色：黑+金 / 深蓝+银 / 酒红+金

### 运动品牌风
速度、力量、粗体、斜切、冲击感、动态轮廓
配色：黑+红 / 深蓝+橙 / 荧光撞色

### 咖啡店/餐饮风
杯形、蒸汽、温暖、手作、亲和、生活方式
配色：咖啡棕+奶白 / 暖橙+褐 / 朱红+金

### 国潮文创风
东方、书法感、印章感、匾额感、传统结构、现代重构
配色：朱红+墨黑 / 金+赭石 / 藏蓝+金

### 科技未来风
几何、模块化、数据感、智能、电路负形、未来感
配色：蓝+白 / 深蓝+青 / 紫+渐变

### 儿童童趣风
圆润、可爱、亲和、活泼、简洁、易识别
配色：明黄+天蓝 / 粉+青 / 橙+绿

## Color System

### 基本原则

- 默认黑白极简输出
- 用户指定颜色时全力适配彩色
- 主色 ≤ 2 种，辅色 ≤ 1 种（点缀用）
- 背景保持干净纯色，不喧宾夺主
- 避免复杂渐变（高端的 Logo 用纯色或至多一阶渐变）
- 彩色方案仍应具备黑白可用性（去色后轮廓清晰可辨）

### 行业配色参考

用户指定颜色或可推敲行业色时，按以下匹配：

| 行业/气质 | 推荐配色 | 说明 |

| 国潮/东方 | 朱红+墨黑 / 金+赭石 / 藏蓝+金 | 高饱和度东方色，可加留白 |
| 科技/互联网 | 蓝+白 / 深蓝+青 / 紫+渐变 | 冷静、理性、未来感 |
| 金融/高端 | 金+黑 / 深蓝+银 / 酒红+金 | 克制、稳重、奢华 |
| 健康/自然 | 绿+白 / 墨绿+金 / 大地色系 | 天然、有机、可持续 |
| 餐饮/咖啡 | 咖啡棕+奶白 / 暖橙+褐 / 朱红+金 | 温暖、食欲、手作感 |
| 儿童/教育 | 明黄+天蓝 / 粉+青 / 橙+绿 | 明亮、活泼、低饱和度 |

### 生图提示词颜色写法

把颜色融入字形描述而非独立色块声明：

```
// 正确
"以金色渐变的字母 O 构成日轮，文字整体呈暖金色调，纯黑背景衬托"

// 错误
"有一个金色圆圈在旁边"（违反文字主体原则）
```

**颜色归属规则：**
- 颜色应附着于文字笔画之上
- 颜色应服务于图形意象（红色火焰文字、蓝色海浪文字）
- 可通过正负形让色彩集中在图形部分，文字主体保持黑白

## Aspect Ratio Rules

- 默认 1:1 方形
- 常见比例：1:1（头像/标准图）、16:9（展示图）、3:1（横版字标）、2:3（竖版展示）

## Trademark Avoidance

用户输入真实品牌名（Nike/Apple/Starbucks 等）时：
- 可以使用该文字做原创字型设计
- 不能使用或模仿其注册商标图标
- 不能加入受保护商标元素
- 不能生成容易混淆的商业标识
- 须声明："围绕文字做原创字形设计，不使用真实商标元素"

在提示词中适当使用以下关键词：

**中文**：文字主体 Logo、创意字标、字体图形化、字形重构、黑白极简、正负形设计、高识别度、品牌标识、商业 Logo、轮廓完整、视觉记忆点、居中构图、干净留白、现代设计感、笔画连接、保持文字可读性

**彩色用**：品牌色、双色搭配、颜色附着于文字、笔画颜色渐变（克制运用）、纯色背景衬托、色彩服务于图形意象

**英文**：typographic logo, wordmark logo, text-based logo, negative space design, black and white logo, minimalist logo, bold graphic typography, custom lettering, clean vector style, high contrast, strong silhouette, centered composition, pure white background, commercial brand identity, modern graphic design, highly readable text, creative lettering, no separate icon

**Color-specific**：brand color palette, duotone配色, color applied to letterforms, color serves the graphic concept, clean solid background, premium color palette, industry-appropriate hues

## Negative Prompt Library

从以下选择组合。**黑白模式**与**彩色模式**选择不同条目：

### 通用（两者皆用）

不要普通字体排版，不要简单地把图标放在文字旁边，不要独立插画，不要真实照片，不要 3D 渲染，不要海报排版，不要杂乱装饰，不要水印，不要品牌商标侵权元素，不要乱码，不要错误文字，不要无法识别的字母或汉字，不要过度变形导致文字不可读，不要低清晰度，不要阴影过重，不要渐变过度，不要赛博朋克背景，不要卡通场景化，不要真实品牌 Logo，不要模仿现有商标。

### 黑白模式额外

不要彩色，不要彩色背景，不要彩色渐变，不要彩色装饰元素。

### 彩色模式额外

不要复杂彩色背景，背景保持干净纯色，不要杂乱色彩，不要多色混杂导致文字难以辨认。

|----------|----------|
| 输入一个词 | 完整方案（创意方向+设计说明+提示词+负面） |
| "只要提示词" | 只输出提示词+负面 |
| "给3个方案" | 3个完整方案，差异明显 |
| "红色国潮" | 主色≤2种，保留黑白可用性；查行业配色表选朱红+墨黑 |
| "彩色的" / "要颜色" | 按行业配色表选色；颜色附着于文字笔画；≤2主色 |

| "运动风格" | 按运动品牌风调整提示词；配黑+红或撞色 |

## Common Mistakes

- **文字与图标并列** — 违反核心原则。文字本身须构成图形
- **变形过度不可读** — 文字须清晰可辨，不可为了图形牺牲可读性
- **风格与词义不匹配** — 奢华品牌不能用童趣风，科技品牌不能用手作风
- **复杂装饰喧宾夺主** — 保持极简，装饰服务于文字而非覆盖文字
- **提示词过于抽象** — AI 生图提示词须具体到"字母如何变形"而非只说"创意"
