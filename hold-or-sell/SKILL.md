# 持房决

## 技能之本

此技能之设，专为已购房而陷于浮亏者决疑也。其法有三：

一曰**问**。询用户购房之始末：何时买入、价几何、面积几许、首付几成、月供若干、今日市价几多。不备此数者，不可断也。

二曰**算**。以四铁律为经，以损益测算为纬，推演净资产之消长、负资产之远近、回本之期年。

三曰**断**。出三策：曰持、曰售、曰半仓。辅以硬止损线、软止损信号、减仓位、止盈线，并附风险管控五法与行动清单。

全程与用户对答，用白话文；所出文档，亦白话文。独此技能文本，用半文半白之体，取其简质也。

---

## 四铁律（决策之锚）

凡断房产之去留，必先明四铁律。此四者，提炼自日本失落三十载与美国次贷之祸，乃全球通行的房产投资不易之规。详见 [铁律.md](铁律.md)。

**铁律一：核心稀缺法则** — 唯供给刚性、需求不可替代之区位的优质产品，方能穿越周期。

**铁律二：杠杆致命法则** — 亏损之根源，非价格下跌，乃高杠杆放大下跌之不可承受性也。负资产（贷款余额大于市值）者，终极陷阱。

**铁律三：人口筛选法则** — 全国人口拐点既至，唯持续净流入之城市与板块，方有价值底线；人口流出之远郊，将现无底之跌。

**铁律四：租金锚定法则** — 当住宅租金回报率持续高于银行存款利率，资产金融属性逆转，长期投资者方大规模回流，此时方为真底。

凡有所断，必引铁律为据，不得凭空立论。

## 问：信息采集之法

与用户初晤，当循次询问以下诸项。缺一不可遽断，缺二以上者，令用户补齐再议。问时用白话，自然交谈，不必填表。

### 必问之项

| 序次 | 所问 | 格式/单位 | 用途 |
|------|------|----------|------|
| 一 | 买入总价 | 万元 | 计算成本基准 |
| 二 | 买入时间 | 年月 | 判断买入时周期位置 |
| 三 | 建筑面积 | 平方米 | 计算单价与可比性 |
| 四 | 首付比例 | %或万元 | 推算贷款额与杠杆倍数 |
| 五 | 月供金额 | 元/月 | 现金流压力测试 |
| 六 | 贷款年限 | 年 | 计算剩余贷款 |
| 七 | 城市与板块 | 文字 | 判断区位价值 |
| 八 | 楼盘名称 | 文字 | 检索产品力与当前市价 |
| 九 | 是否自住 | 是/否 | 决定是否计租金 |

### 可推而知者 / 可联网搜取者

以下诸项，无需用户提供，AI当自行推演或联网搜索：

- **当前二手市价**：据楼盘名称与城市，联网搜索链家、贝壳、房天下等平台之挂牌均价，取近3个月成交价或挂牌价之中位数。搜索不得者，方问用户。
- **贷款余额**：由首付比例推贷款额，由月供与贷款额反推利率，由已还月数推算剩余本金。
- **总市值**：由当前市价与面积相乘即得。

推算或搜索所得，当告用户方法与来源，请其核对。

### 选问之项（有则更精，无则不碍）

- 家庭月收入（判断月供占比）
- 现金储备（判断抗风险能力）
- 物业费（计入持有成本）
- 预计租金（计入以租养贷）

## 算：测算推演之法

信息既备，乃以 [模板.md](模板.md) 为骨架，依次为以下六算：

### 第一算：持仓损益表

制表如下，务必精确（白话文输出）：

| 项目 | 金额 | 说明 |
|------|------|------|
| 买入总价 | | |
| 当前市值 | | |
| 账面浮亏 | | 浮亏额与百分比 |
| 首付金额 | | |
| 贷款总额 | | |
| 月供 | | |
| 已还月数 | | |
| 已还总额 | | 本金+利息分别列示 |
| 剩余贷款 | | 推算值，标注推算依据 |
| 当前净资产 | | 市值减剩余贷款 |
| 本金亏损幅度 | | 净资产较首付之跌幅 |
| 若售净回收 | | 市值-剩余贷款-2%交易费 |
| 若售实际亏损 | | 首付-净回收 |

关键提示：必点明**杠杆放大效应**——房价表面跌幅与本金实际跌幅之差，引 [铁律二]。

### 第二算：卖 vs 持之辨

先列"不卖"之三理由，必对照铁律与标的实际：
1. 是否卖在周期底部（对照2026底部确认判断）
2. 标的是否具穿越周期基因（对照 [铁律一] 检区位与产品力）
3. 杠杆是否已在危险区（对照 [铁律二] 查是否触发负资产，月供是否可承受）

再列"可持"之三前提，缺一即建议卖出：
1. 现金流可持续（月供<家庭收入40%，备12个月月供）
2. 无更优替代（售房回收款无处可投）
3. 能受5年+锁定（心理与财务）

### 第三算：回本路径

三情景推演，从当前市价出发：

| 情景 | 年化涨幅 | 回本所需年数 | 概率 |
|------|---------|-------------|------|
| 保守 | 1-2% | 算出 | 30% |
| 中性 | 3-5% | 算出 | 50% |
| 乐观 | 6-8% | 算出 | 20% |

回本之定义：单价回到买入价，且覆盖2%交易成本。

另附7年后的三种情景下市值、净资产、总盈亏之推演。

### 第四算：止损止盈线

定四线：

| 线型 | 触发条件 | 操作 |
|------|---------|------|
| 硬止损 | 单价跌至**负资产临界点**（市值<贷款余额） | 即刻全售 |
| 软止损 | 城市人口净流出/产业停滞/板块成交枯竭 | 3-6月内售完 |
| 减仓线 | 回升至浮亏缩半之价位 | 售50% |
| 止盈线 | 买入价×115%且连续2年涨幅>8% | 分批清仓 |

每线皆须引铁律为据。硬止损对应 [铁律二]，软止损对应 [铁律三]。

### 第五算：风险管控五法

| 管控之法 | 操持之要 | 频次 |
|----------|---------|------|
| 月供压力测试 | 核月供占收入比，超40%即警 | 每季 |
| 价格跟踪 | 追本板块二手均价 | 每月 |
| 以租养贷 | 不自住则必租，算月租覆盖月供之比 | 一次 |
| 政策信号 | 盯供地量、限购限售之变、配套兑现 | 持续 |
| 替代融资 | 探抵押贷利率，低于按揭时可置换 | 每年 |

### 第六算：行动清单

按优先级列P0-P2行动项，每项附完成时限。P0为"即刻"，P1为"本周/本月"，P2为"持续/下月"。

## 断：出具结论之法

测算既毕，当以白话出具完整文档。文档结构以 [模板.md](模板.md) 为准，核心结论前置，后附六算详表。

最终建议，必以三句话收束：
1. 卖或不卖，一句话断之；
2. 设硬止损于某价，不可移也；
3. 需持若干年以为回本之备。

## 须恪守者

1. **凭据为先**：每断必引铁律或数据，不凭空臆断。
2. **白话为用**：与用户对答、出具文档，皆用白话，通俗可读。
3. **数据求确**：推算者必标注推算方法，请用户核实。
4. **不代决策**：所有分析为参考之用，不代用户决断。
5. **先问后算**：信息未备，不可强断。缺必问项二者以上，令补全再议。
6. **半文半白者**：唯此技能文本及铁律文档用半文半白，其余皆白话。

## 操持之序

1. 用户言及卖房、浮亏、持仓之事，即启动本技能。
2. 依"问"章采集十项必问信息，以自然白话交谈。
3. 信息既备，乃依"算"章推演六算，填入 [模板.md](模板.md)。
4. 文成，呈于用户，请其核对关键数据。
5. 若用户补正数据（如月供、利率等），即更新测算，不可牵就旧数。
