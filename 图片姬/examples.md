# 图片姬 - 使用示例集

## 基础示例

### 1. 技术概念图
**场景**：解释算法思维的重要性

**Prompt**：
```
Conceptual illustration showing the essence of algorithmic thinking. Central focus: human brain with illuminated neural pathways forming classic algorithm structures - binary trees, linked lists, and graph networks radiating outward. Surrounding elements: scattered puzzle pieces coming together, mathematical formulas floating in space, and abstract geometric shapes representing problem-solving patterns. Color palette: deep indigo (#4b0082) for the brain, vibrant gold (#ffd700) for neural connections, soft silver (#c0c0c0) for puzzle elements. Modern digital art style with glowing effects and particle systems. 16:9 widescreen format, high-definition rendering emphasizing intellectual illumination and systematic thinking.
```

### 2. 对比概念图
**场景**：传统编程 vs AI辅助编程

**Prompt**：
```
Side-by-side comparison illustration contrasting traditional coding with AI-assisted development. Left panel: lone programmer hunched over computer with multiple monitor setup, surrounded by crumpled papers and empty coffee cups, dim lighting suggesting struggle. Right panel: collaborative workspace with human and AI interface working together, clean organized desk, bright natural lighting, plants and comfortable seating. Connecting element: dotted line showing evolutionary progression from isolated work to collaborative innovation. Color scheme: left side uses muted grays and browns (#696969, #8b4513), right side features fresh blues and greens (#4682b4, #32cd32). Contemporary office illustration style with realistic textures and shadows. 2:1 horizontal layout, professional quality suitable for blog headers.
```

## 进阶示例

### 3. 流程演进图
**场景**：软件开发方法论演变

**Prompt**：
```
Evolutionary timeline of software development methodologies from waterfall to modern agile practices. Vertical flow layout: top shows traditional waterfall model with rigid sequential phases (requirements, design, implementation, testing, deployment) in industrial grays and blues. Middle section depicts transition period with hybrid approaches in transitional oranges and yellows. Bottom presents contemporary DevOps and continuous delivery in vibrant teals and purples. Each methodology represented by distinctive architectural metaphors: waterfall as concrete dam, agile as flexible bridge, DevOps as interconnected pipeline system. Background gradient shifts from cold steel blue to warm collaborative orange. Infographic design style with clear typography and symbolic icons. 9:16 portrait ratio, presentation-quality vector graphics ideal for educational content and technical documentation.
```

### 4. 数据可视化
**场景**：程序员技能需求变化趋势

**Prompt**：
```
Interactive-style dashboard visualization showing programming language popularity trends over the past decade. Main chart: line graph with multiple colored trajectories representing different languages - JavaScript (yellow), Python (blue), Java (red), Go (green), Rust (orange). Secondary elements: bar charts showing job market demand, pie charts for community adoption rates, and bubble charts for learning curve difficulty. Color palette: vibrant tech colors with dark charcoal (#2f2f2f) background for contrast. Modern UI design aesthetic with subtle gradients, drop shadows, and glass-morphism effects. Dashboard layout with clean grid system and intuitive data hierarchy. 16:9 format, crisp vector illustration mimicking real analytics dashboards suitable for technical blogs and industry reports.
```

Central element: large thought bubble above teacher's head containing visual representation of problem decomposition - complex problem breaking into smaller manageable pieces like puzzle solving. Learning aids: flowchart posters on walls, algorithm visualization screens, peer programming stations. Students depicted at different skill levels: beginner with guided exercises, intermediate with independent projects, advanced mentoring newcomers. **场景**：学习编程的心理历程

```
Emotional journey map illustrating the psychological experience of learning programming. Roller coaster visualization: starting point "Initial Excitement" with bright fireworks and celebration, climbing "Learning Curve Struggle" with storm clouds and lightning, reaching "Breakthrough Moment" with sunrise and achievement badges, cruising "Confidence Building" with smooth sailing and clear skies, arriving at "Mastery Satisfaction" with panoramic views and accomplishment trophies. Emotional indicators: frustration shown as tangled spaghetti code, joy represented by perfectly executed programs, pride depicted through completed project portfolios. Color psychology: warm encouragement tones during difficult phases, celebratory brights for success moments. Narrative illustration style telling a complete story arc. 21:9 cinematic format, emotionally resonant artwork suitable for motivational content and learning experience design.
```
