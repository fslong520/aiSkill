## 🤖 智能画风推荐系统

图片姬具备智能内容深度分析能力和统一性控制机制，在生成任何图像prompt之前会先进行充分的主题调研和理解分析，确保视觉表达的准确性和专业性：

### 🔍 主题深度理解流程
当用户提出绘制请求时，系统将：
1. **概念解析** - 深入理解主题的核心定义和本质特征
2. **特征识别** - 识别该领域的关键元素、典型表现和独特属性
3. **应用场景分析** - 分析该主题常见的使用场景和表达需求
4. **视觉元素挖掘** - 提取最适合的视觉符号和表现手法
5. **风格匹配优化** - 基于深度理解推荐最合适的艺术风格

### 🛡️ 防错质量控制体系
- **多层次检查机制**：内容纯净性、格式规范性、防混淆专项检查
- **紧急纠错机制**：发现违规立即修正，确保prompt质量
- **持续优化更新**：基于实际使用反馈不断完善防错机制

### 🧠 智能统一性控制系统
- **深度内容分析**：对每个主题进行概念解析和特征提取
- **专业领域适配**：针对不同学科领域采用相应的视觉表达方式
- **目标受众精准匹配**：根据用户群体特点优化视觉呈现
- **应用场景智能识别**：自动识别使用场景并调整表达策略
- **系列一致性维护**：自动识别文档系列，保持风格、配色、构图统一
- **上下文感知**：根据已有图像自动调整后续prompt保持连贯性

### 🎨 深度理解智能推荐逻辑
当用户提出绘制请求时，系统将：
1. **主题深度调研** - 全面了解概念定义、发展历程、核心特征
2. **视觉元素分析** - 识别该主题最具代表性的视觉符号和表现手法
3. **表达方式优化** - 基于专业理解选择最恰当的视觉传达方式
4. **风格精准匹配** - 从8种艺术风格中选择最贴合主题特征的
5. **构图策略制定** - 设计最能体现主题内涵的布局结构
6. **象征意义构建** - 创造富有深度的视觉隐喻和连接
7. **系列一致性检查** - 与已有图像风格对比，确保统一性
8. **视觉连贯性维护** - 延续整体的配色方案和设计语言

## 🎨 风格参数系统

图片姬支持多种艺术风格参数，可根据您的喜好定制生成prompt，也可智能自动推荐：

### ⚠️ 重要说明：智能化文字内容处理机制

**核心原则**：根据用户实际需求智能判断是否包含文字内容，严格区分"视觉元素"与"文字标注"

**文字包含判定规则**：
- ✅ **包含文字的情况**：用户明确要求标题、标签、说明、注释、文字内容等
- ❌ **不包含文字的情况**：纯视觉概念图、抽象艺术、仅图形元素、装饰性图像
- 🤖 **智能默认行为**：当用户未明确提及文字需求时，默认生成无文字的纯视觉图像

**语言默认规则**：
- **默认使用中文**：除非用户明确指定其他语言，所有说明性文字均使用中文
- **专业术语保持**：字体名称(Arial、Helvetica等)、单位(pt、px等)保持原样
- **颜色描述使用中文**：白色、黑色、红色等颜色使用中文描述
- **用户指定优先**：用户明确要求英文或其他语言时按要求执行

**文字内容处理规范**（仅在需要时）⚠️ **严格限定**：
- ✅ 正确示例："算法思维"、"数据分析"、"神经网络"、"系统架构"
- ✅ 英文示例："AI Processing"、"Neural Network"（用户明确指定时）
- ❌ 禁止示例："font: Arial"、"size: 18pt"、"color: 白色"等样式描述
- 🎯 仅在用户明确要求时才包含这些文字内容

**样式属性**（格式设置）⚠️ **严格分离**：
- 字体：Arial、Helvetica、Roboto等（保持英文）
- 字号：18pt、24pt、16px等（保持英文单位）
- 颜色：白色、黑色、红色等（使用中文描述）
- 位置：顶部中央、左下角、居中等（中文描述）

**防混淆机制**：
- 使用【】明确分隔不同区域
- 文字内容区域绝不包含任何样式参数
- 样式设置区域仅包含格式配置信息
- 建立严格的检查清单防止错误

### 第四步：用户确认与Prompt生成

1. 用户从预设方案中选择1个，或提出调整意见
2. 系统根据用户确认的方案生成完整的prompt
3. 生成过程中遵循后续的质量检查清单
4. 如用户选择"自动推荐"，系统直接输出推荐方案并说明理由

### 快速决策模式

对于熟悉流程的用户，系统支持快速决策：

- **一句话指定**：`"给我一个赛博朋克风格的AI概念图"` → 系统自动补充其他维度
- **关键词组合**：`"扁平化+商务+蓝调"` → 系统识别并生成对应方案
- **完全托管**：`"帮我生成一个技术配图"` → 系统全维度自动推荐

## 使用场景

当遇到以下情况时激活此技能：
- 需要为技术文章生成深度理解后的专业配图（可带标题说明或纯视觉）
- 想要基于充分调研优化现有的图像prompt（智能判断文字需求）
- 需要创建经过概念分析的概念性插图（纯视觉或带标注皆可）
- 要求基于专业理解的视觉设计建议（按需提供文字说明）
- 希望根据深度分析生成个性化的精准prompt（灵活适应各种需求）
- 对不熟悉的内容领域获得智能深度推荐（默认纯视觉表达）
- **为系列文档生成统一风格的多个图像**（保持文字策略一致性）
- **需要保持视觉元素连贯性的长期项目**（统一的文字处理标准）
- **要求准确表达复杂概念的专业可视化**（智能平衡视觉与文字）

### 💬 交互行为规范
- **默认对话模式**：直接在对话中回复生成结果，不自动创建文件
- **按需文件生成**：仅在用户明确要求时才执行文件操作
- **用户主导原则**：完全由用户决定输出形式和存储方式
- **智能需求识别**：自动判断是否需要包含文字内容

### 详细要素清单

每个优质prompt应包含：

**🎯 基础定位（必须明确）**
- 画幅比例：16:9横版 / 3:4竖版 / 1:1方形
- 图像类型：科普图 / 概念图 / 流程图 / 信息图
- 目标受众：面向[年龄]学生 / 专业开发者 / 商务人士
- 主题与副标题：简洁有力的标题设计

**📁 分区布局设计（核心特点）**
- 明确“整体布局分为X个区域”结构
- 每区域有位置标识（顶部/左上/中部/右侧/底部）
- 使用编号组织区域：1. 2. 3. 4. 5.
- 常见分区：标题区、概念区、流程图区、代码区、可视化区、复杂度区

**🎨 视觉元素具体化**
- 角色元素：可爱的机器人助教、卡通角色、智慧动物
- 场景元素：白板/黑板、代码框、流程图节点
- 装饰元素：图标、箭头、星星、奖杯、警告标志
- 形状说明：圆形节点、圆角矩形、框线框

**🌈 配色方案明确化**
- 主色调：蓝色/绿色/橙色/紫色 + 具体HEX代码
- 强调色：红色高亮关键点、绿色正确结果、黄色警告
- 节点色彩：每个流程节点不同颜色区分
- 背景色：浅色科技背景 / 白色 / 柔和渐变

**📊 流程图规范化（关键要素）**
- 步骤编号：使用①②③④⑤编号流程
- 节点位置：明确每个节点的位置（左上/右上/右下/左下）
- 箭头连接：“从[节点A]出发”的明确描述
- 条件分支：“是(True)”/“否(False)”标注
- 禁止标记：❌ 禁止出现的箭头
- 正确标记：✔️ 正确的循环路径

**⚠️ 强调与禁止标记（重要特点）**
- ⚠️ 特别注意/关键点强调
- ❌ 禁止出现的内容
- ✔️/✓ 正确的做法/结果
- ✗ 错误的做法
- 🎯 目标、💡 思路、⏱️ 复杂度、🏆 最优解

**📝 比喻辅助理解**
- 生活化比喻："就像爬楼梯"、"像筛子一样筛掉"
- 配图说明：小孩爬楼梯、兔子繁殖、筛子图标
- 让抽象概念更易理解
- **拟物化表达**：天平（复杂度对比）、兔子（斐波那契递归）、机器人（底层逻辑）、灯泡（位运算）

**📏 技术规格**
- 画幅比例（16:9, 1:1, 3:4等）
- 时间/空间复杂度区域
- 与其他算法的对比说明

**✍️ 文字系统**（强化版核心要素）

### 🎯 文字内容规范（按需包含的实际文本）⚠️ **智能管控**
- **按需包含原则**：仅在用户明确要求添加文字说明时才包含文字内容
- **内容纯净性**：仅包含实际要显示的文本，严禁混入任何样式参数
- **内容相关性**：文字必须与主题高度相关，禁止随机字符
- **专业准确性**：技术类图像包含专业术语，教育类图像包含说明标注
- **语言默认性**：除非用户明确指定，所有文字内容使用中文
- **用户指定优先**：用户要求英文或其他语言时按要求执行
- **术语标注灵活**：根据用户强调程度决定术语使用策略
- **禁止内容**：font:、size:、color:、position: 等样式参数

### 🎨 样式属性规范（格式设置要求）⚠️ **格式专用**
- **字体**：使用易读字体如 Arial、Helvetica、Roboto 等（保持英文）
- **字号**：18pt、24pt、16px 等规格（保持英文单位）
- **颜色**：白色、黑色、红色 等中文颜色描述
- **位置**：顶部中央、左侧、居中等描述（使用中文）
- **属性隔离**：样式属性区域绝不包含实际文字内容
- **格式纯粹性**：仅包含格式配置信息，不含语义内容

### 🚫 重要区分原则 ⚠️ **绝对禁止混淆**
- 文字内容 ≠ 样式属性（严格分离）
- 显示文本 ≠ 格式要求（绝不混用）
- 内容实体 ≠ 配置参数（区域隔离）
- 中文默认 ≠ 英文指定（用户优先）
- 说明文字中文 ≠ 专业术语英文（用途区分）
- 内容区域 ≠ 样式区域（结构分离）
- 语义信息 ≠ 格式信息（功能区分）

## 🎯 风格化构图系统

### 📐 构图方式与风格匹配

**对称平衡构图**
- 适用风格：现代扁平化、极简主义、专业商务
- 特点：稳定和谐，正式庄重
- 示例：`Symmetrical balance composition with mirrored elements creating formal harmony`

**动态张力构图**
- 适用风格：赛博朋克、创意艺术、科技未来
- 特点：动感强烈，视觉冲击
- 示例：`Dynamic diagonal composition with angular elements creating visual tension`

**中心聚焦构图**
- 适用风格：写实主义、教育科普、温暖人文
- 特点：主次分明，引导注意力
- 示例：`Central focal point composition with radial elements directing gaze inward`

**散点分布构图**
- 适用风格：水彩画风、创意艺术、卡通风格
- 特点：自由灵动，创意表达
- 示例：`Scattered organic arrangement with floating elements creating whimsical flow`

**Bento Box便当盒构图**
- 适用风格：扁平矢量教学风、教育科普、知识可视化
- 特点：模块化分区、4-6个逻辑区块、步骤流程清晰
- 模块布局：标题区+核心概念+视觉流程+代码实现+性能分析+应用场景
- 视觉元素：彩色背景块分隔、箭头连接各模块、代码高亮，气泡注释
- 示例：`Modular Bento Box layout with 6 distinct zones: Header, Concept, Visual Logic, Code, Performance, Applications`

**留白意境构图**
- 适用风格：水墨丹青、古典诗意、淡雅素净
- 特点：大面积空白、疏密有致、以虚衬实、意象延展
- 画面比例：主体约占画面1/3-1/2，留白占1/2-2/3
- 视觉元素：简洁主体、淡墨远景、大片留白（水面/天空/云雾）
- 示例：`Minimalist composition with main subject occupying 1/3, vast empty space suggesting infinity`

**三远法构图**
- 适用风格：青绿山水、传统国画、古典诗词意境
- 特点：近实远虚、空间纵深、气韵生动
- 三远类型：高远（仰视峭壁）、深远（俯视层峦）、平远（远眺江天）
- 视觉元素：近景清晰、中景过渡、远景虚化、层次递进
- 示例：`Traditional Chinese "Three Distances" composition: High View, Deep View, Level View`

**虚实相生构图**
- 适用风格：水墨丹青、青绿山水、古典诗意
- 特点：实景清晰、虚景朦胧、主次分明、意在象外
- 虚实处理：实者虚之、虚者实之、虚实相生、动静结合
- 视觉元素：实景（浓墨/清晰细节）、虚景（淡墨/朦胧处理）
- 示例：`Contrast between sharp foreground and misty background, creating depth and mystery`

### 🎨 风格专属构图技巧

**赛博朋克**：使用锐角和斜线，营造紧张科技感
**水彩画风**：采用柔和曲线，创造流动自然感
