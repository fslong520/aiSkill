Domain keywords: 配图, 插图, prompt生成, 图片生成, 视觉设计

Summary: 输入主题，AI自主推理风格/配色/构图，生成图像prompt或直接生成图片。

Strategy:
1. 深度理解主题（概念解析、特征识别）
2. 判断用户意图 → 智能推荐/用户选择模式
3. AI自主决策参数（受众/风格/氛围/构图/配色）
4. 输出2-3个方案供选择
5. 生成完整prompt → 质量检查

AVOID:
- AVOID 文字内容中混入样式参数（font:、size:）
- AVOID 不经用户确认就生成带文字的图
- AVOID 风格不统一，系列图要保持一致
- AVOID 使用"我有个朋友"等AI痕迹表达

---

## 双模式工作机制

| 模式 | 触发 | 流程 |
|------|------|------|
| 智能推荐 | 用户不确定风格 | 深度理解→自动推荐→生成prompt |
| 用户选择 | 用户有明确偏好 | 六维度调研→生成方案→确认 |

## 六维度参数

1. 目标受众
2. 艺术风格
3. 主题氛围
4. 构图偏好
5. 配色方案
6. 知识可视化结构（可选）

## 文字处理原则

| 情况 | 处理 |
|------|------|
| 用户明确要求文字 | 包含文字内容 |
| 用户未提及文字 | 默认纯视觉图像 |
| 语言未指定 | 默认中文 |

**严格分离**：文字内容 ≠ 样式属性

## Prompt模板

```
创建一张[比例]的[图像类型]，主题是"[主题]"，面向[受众]。

**整体布局分为[N]个区域:**
1. ([区域名] [位置]): [内容]
2. ...

**整体风格:** [风格]、[配色]
```

## 质量检查

- [ ] 文字区域无样式参数
- [ ] 构图描述清晰
- [ ] 配色方案明确
- [ ] 风格统一
