# 图片姬 - 结构化AI绘图提示词生成器

## 这个技能是做什么的？

**一句话**：输入主题描述，输出结构化的AI绘图Prompt。

**工作流程**：
```
用户输入主题 → 分析主体特征 → 决定风格方向 → 填充维度 → 输出结构化Prompt
```

**输出格式**：
- Markdown 结构化版（含完整维度描述）
- JSON 版（便于程序调用/二次编辑）
- 纯文本版（直接复制到绘图工具）
- 英文版（适用于 Midjourney/Stable Diffusion）

---

## 金字塔Prompt架构（总纲）

输出之组织，须按宏观→微观递进，层层约束，不可跳跃。六层架构：

```
第1层 空间与构图    → 比例、场景、色调板、负空间（定调）
第2层 主体与人物    → 年龄、种族、概貌（锚点）
第3层 微相与情绪    → 五官、微表情、眼神量化、克制美学（传神）
第4层 姿态与动势    → 站立/动态、手部细节、衣纹暗示（生姿）
第5层 服装与质感    → 款式、面料、版型哲学、正反约束（定性）
第6层 光影与镜头    → 光质、色温、焦段、景深、质量标签（定质）
```

每层须与上一层呼应，不可出现"第1层极简奶油风，第6层赛博朋克"之冲突。
强调跨维度一致性——所有层讲同一个故事。

## 思考维度框架（16 维度）

### 一、主体层（6 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **subject** | 主体是什么？一句话概括 | 角色、产品、场景、概念 |
| **age** | 年龄设定？ | 少女、青年、成熟、御姐、萝莉、老年 |
| **model** | 身体特征？ | 体型、肤色、比例、姿态 |
| **face** | 面部特征？见"微表情精度控制" | 五官轮廓、表情神态、眼神方向 |
| **pose** | 姿态动作？ | 站姿、坐姿、动态、交互动作 |
| **hair** | 发型特征？ | 颜色、长度、质感、造型 |

### 二、服装层（1 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **costume** | 服装特征？见"服装深度描述" | 款式、材质、颜色、配饰、版型哲学 |

### 三、风格层（2 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **style** | 整体艺术风格？ | 杂志时尚摄影、新海诚动漫、赛博朋克、水墨、极简 |
| **mood** | 情绪氛围？ | 慵懒宁静、冷酷、温暖、神秘、寂寥 |

### 四、环境层（3 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **environment** | 场景环境？ | 室内极简住宅、废墟、城市、抽象空间 |
| **lighting** | 光影效果？见"光质描写指南" | 漫射光、窗光、霓虹、逆光、柔光箱 |
| **composition** | 构图方式？见"镜头语言" | 画幅比例、视角、视觉锚点、纵深 |

### 五、镜头层（2 维度）← 新增

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **lens** | 镜头语言？见"镜头语言指南" | 焦段、光圈、景深、透视、电影感 |
| **quality_tags** | 质量标签？见"质量标签体系" | 渲染精度、平台特定词、风格定调词 |

### 六、控制层（2 维度）

| 维度 | 思考问题 | 示例方向 |
|------|---------|---------|
| **negative** | 避免什么？ | 常见错误、不需要的元素、风格冲突 |
| **aspect_ratio** | 画面比例？ | 由用途决定 |

## 层级化描述法则

每个维度的描述遵循三层结构：

```
维度名称
├── 基础描述（一句话定义核心）
├── 特征列表（3-5 个关键词或条目）
└── 细节补充（可选，具体参数）
```

关键：**每层都须考量正反双向约束**——不仅要写"是什么"，还要写"不是什么"。

---

## 美学深度框架（三层递进）

每张图像应追求三层递进的美学深度：

### 第一层：悦目层（感知层）
- 视觉冲击力：色彩对比、光影层次、构图张力
- 技术完成度：细节精度、渲染质量、风格一致性
- 评判标准：第一眼是否"好看"

### 第二层：悦意层（情感层）
- 情感共鸣：能否触发观者情绪（温暖/孤独/震撼/宁静）
- 叙事张力：画面是否在"讲故事"
- 评判标准：看完是否"有感觉"

### 第三层：悦神层（意味层）
- 深层表达：画面背后的思想、隐喻、文化内涵
- 开放性：留白与想象空间，观者可二次解读
- 评判标准：是否"值得回味"

### 执行建议
生成Prompt时，先确保悦目层达标，再追问：
- 这张图的情感内核是什么？（悦意层）
- 有没有更深层的表达可能？（悦神层）

## 微表情精度控制框架（face 维度强化）

人像prompt的核心竞争力在微表情。模糊的情绪词（"微笑""悲伤"）远不如精确的微表情描述。

### 六维微表情控制

| 控制维度 | 描述方法 | 示例 |
|---------|---------|------|
| **眼神方向** | 量化+锚点：直视度0~100% + 目光落点 | "眼睑微垂约1/5，不直视镜头，目光落于镜头前寸许" |
| **嘴部细节** | 开合度+舌尖+嘴角方向 | "唇自然微启，不刻意微笑，舌尖轻抵下齿" |
| **眉毛/眼睑** | 高度+弧度量化 | "眉自然松弛，眼睑下垂约五分之一" |
| **头部倾斜** | 方向+角度 | "头微左倾约10度" |
| **呼吸感** | 胸腔起伏/嘴唇状态暗示呼吸 | "呼吸轻柔，唇微启如欲言又止" |
| **克制美学** | 负向约束：不做什么 | "不刻意微笑，不直视镜头，神情安静而克制" |

### 眼神精度分级

| 级别 | 描述 | 效果 |
|------|------|------|
| 直视镜头 | 100%正视，目光相接 | 冲击、对峙、邀请 |
| 镜头附近 | 目光落于镜头前后，不直接相接 | 互动感+遐想空间 |
| 微垂（1/5~1/3） | 眼睑轻度下垂 | 慵懒、温柔、内敛 |
| 半垂（1/3~2/3） | 明显下垂 | 沉思、疏离、忧郁 |
| 侧视 | 目光转向画外 | 叙事感、猜疑、向往 |
| 闭目 | 完全闭合 | 沉浸、享受、冥想 |

### 执行要求

填充face维度时，必须考量至少3个微表情维度，不可只写"微笑/表情自然"了事。
克制美学优先——能用"不做什么"指导的，比"做什么"更有力。

---

## 服装深度描述框架（costume 维度强化）

服装不只款式+颜色。须覆盖四层：

```
第1层 款式与色彩    → 连衣裙、黑色、简约（外观）
第2层 面料与质感    → 轻薄外层、丝绒、棉麻（触觉）
第3层 版型哲学      → 不强调曲线、不过度贴身、宽松有度（约束）
第4层 配饰与点睛    → 黑色项圈、简约耳饰（细节）
```

### 版型哲学（关键）

这是大多数prompt忽略的维度。描述"不"比描述"是"更重要：

| 约束类型 | 示例 |
|---------|------|
| 体型约束 | "不强调身体曲线，不过度贴身" |
| 裸露约束 | "不露肤，高领长袖" |
| 结构约束 | "不收腰，直筒版型" |
| 风格约束 | "不花哨，无印花Logo" |

### 触觉词汇表

prompt中加入触觉词，AI对材质的渲染精度显著提升：

| 触觉词 | 对应面料 | 视觉暗示 |
|--------|---------|---------|
| 轻薄 | 雪纺、真丝 | 半透明、飘逸 |
| 厚重 | 羊毛、呢绒 | 垂坠感、压褶 |
| 挺括 | 棉麻、混纺 | 线条清晰、塑形 |
| 柔滑 | 缎面、丝绒 | 反光、丝滑 |
| 粗粝 | 针织、粗花呢 | 纹理感、温暖 |

## 光质描写指南（lighting 维度强化）

光影不分"亮"与"暗"，分"质"与"温"。

### 光质九宫格

```
             硬光（点光源）               柔光（大面积光源）
      ┌─────────────────────┬─────────────────────┐
  暖  │  暖硬：阳光/钨丝灯     │  暖柔：黄昏/暖白漫射     │
  色  │  戏剧感、强烈阴影      │  温馨、柔和、通透        │
      ├─────────────────────┼─────────────────────┤
  冷  │  冷硬：月光/无影灯     │  冷柔：阴天/柔光箱       │
  色  │  清冷、锐利、科技感    │  平静、均匀、纯净        │
      ├─────────────────────┼─────────────────────┤
  混  │  混硬：多色点光源      │  混柔：环境光+补光       │
  合  │  霓虹、舞台、冲突      │  电影感、层次丰富        │
      └─────────────────────┴─────────────────────┘
```

### 技术控制四要素

| 要素 | 描述 | 示例 |
|------|------|------|
| 光源色温 | 暖白/冷白/中性 | "暖白色室内漫射光" |
| 光线方向 | 顶光/侧光/背光/漫射 | "环境光均匀柔和，无定向主光" |
| 高光控制 | 过曝/细腻/高反差 | "高光细腻不过曝" |
| 特效 | Bloom/光晕/辉光/光束 | "轻微Bloom柔光效果" |

### 空气感（Atmosphere）

中文摄影美学的核心概念——"空气感充足"意谓：

- 空间中有介质感（薄雾/微尘/光路可见）
- 色调通透不闷
- 主体与背景之间有"距离感"的呼吸空间
- 不是"清晰"，而是"清透"

表达方式：用"空气感""通透感""呼吸感""空间感"等词直接写入prompt。

## 镜头语言指南（lens + composition 维度）

### 焦段选择

| 焦段 | 透视特性 | 适用场景 |
|------|---------|---------|
| 35mm | 小广角，略强调纵深 | 环境人像，叙事感 |
| 50mm | 标准视角，自然真实 | 半身人像，日常感 |
| **85mm** | **中长焦，浅景深，无畸变** | **人像黄金焦段，杂志摄影** |
| 135mm | 长焦，空间压缩感强 | 特写，面部细节 |
| 200mm+ | 超长焦，极浅景深 | 远摄，虚化极致 |

### 景深控制三档

| 景深 | 描述方式 | 效果 |
|------|---------|------|
| 浅 | "大光圈浅景深，背景自然虚化" | 突出主体、杂志风 |
| 中 | "中等光圈，前后景适度清晰" | 环境人像、叙事 |
| 深 | "小光圈全景清晰" | 环境展现、建筑 |

### 电影感三板斧

写prompt时加入以下三要素之一即可显著提升电影感：

1. **色彩叙事** — 限定色调板（"暖棕调配色"）
2. **画面呼吸** — 留有负空间（"留白充足"）
3. **纹理真实** — 强调材质（"真实皮肤纹理"）

### 平台特定质量词

| 平台 | 推荐质量词 |
|------|-----------|
| Midjourney | `--ar 9:16 --style raw --stylize 250` |
| SD/NovelAI | `masterpiece, best quality, ultra-detailed, 8K` |
| DALL-E 3 | 自然语言描述即可，不需要质量标签 |
| 通用 | `Ultra Realistic, Masterpiece, Best Quality, 8K, cinematic lighting` |

### 精度梯度

| 级别 | 关键词 |
|------|--------|
| 概念级 | concept art, sketch, rough |
| 标准级 | detailed, sharp focus, high detail |
| **专业级** | **Ultra Realistic, Masterpiece, Best Quality, 8K, cinematic** |
| 发烧级 | photorealistic, hyper-detailed, octane render, subsurface scattering |

### 风格定调词

放在prompt尾部作为风格锚点：`高级时尚杂志摄影，生活方式写真，电影级摄影，简洁自然`

## 正反双向约束原则

这是本技能最核心的方法论升级。每个prompt维度的描述须同时考虑：

### 正向指令（Positive）
描述"要什么"——具体、可感知、可渲染
> "暖白色漫射光，环境光均匀柔和"

### 负向约束（Negative-in-Place）
描述"不要什么"——嵌入主体描述中，而非只塞进negative prompt
> "不直视镜头，不刻意微笑，不强调身体曲线，不过度贴身"

### 为什么嵌入比--no有效？

`--no`或negative prompt是宽泛过滤，而主体描述中的"不…"是精确语义控制。
AI对主描述中的否定词敏感度远高于negative prompt。

### 执行要求

**每个维度在填充时，都必须自问：**
- 这个维度的正向描述完整了吗？
- 有什么AI容易"默认生成"但我想避免的？
- 能不能加一个"不…"来控制？

**face维度尤其需要负向约束**（不笑、不看、不夸张），其次是costume（不紧身、不暴露）。

## 执行流程

### 步骤 1：理解需求

向用户确认（如用户已提供足够信息则可跳过）：
- 生成什么主题？（角色/产品/场景/概念）
- 用途？（海报/壁纸/社交媒体/概念图）
- 特殊要求？（风格偏好、画面比例、是否需要文字）

### 步骤 2：分析主体

AI自主完成：
- 搜索/分析主体背景资料
- 提取标志性特征
- 确定世界观调性

### 步骤 3：决定风格

AI自主判断：
- 根据主题特性推荐最合适的风格
- 用户有明确偏好则优先遵循
- 系列图保持风格一致性

### 步骤 4：组织架构（参照金字塔架构）

按金字塔六层组织思路，从宏观到微观：
1. 定调（空间+比例）→ 2. 锚点（主体概貌）→ 3. 传神（微表情）
4. 生姿（姿态）→ 5. 定性（服装）→ 6. 定质（光+镜头+标签）

确保跨维度一致性——六层讲同一个故事。

### 步骤 5：逐个填充维度（参照正反双向约束）

按16维度框架逐一填充，每维度自问：
- 正向描述完整否？
- 负向约束充足否？（需要"不……"吗？）
- 细节具体否？（能用数字量化吗？）

按以下结构输出：

### 纯文本版（复制即用）

按金字塔架构组织，从宏观到微观：

## 文字处理原则

| 情况 | 处理 |
|------|------|
| 用户明确要求文字 | 包含文字内容，字体风格由AI根据主题推荐 |
| 用户未提及文字 | 默认纯视觉图像，不添加任何文字 |
| 语言未指定 | 默认中文 |

**严格分离**：文字内容 ≠ 样式参数（font:、size: 等避免混入画面描述）

## Principles

| 原则 | 说明 |
|------|------|
| AI自主决策 | 风格、配色、构图由AI根据主题特性自主决定 |
| 用户偏好优先 | 用户有明确要求时（风格/比例/年龄/配色），优先遵循 |
| 系列一致性 | 同一系列图锁定同一套风格关键词 |
| 灵活调整 | 维度可根据主题需要增减（如无人物则跳过age/face/pose/hair） |
| 正反双向约束 | 每个维度须同时考虑"要什么"和"不要什么" |
| 金字塔递进 | 从空间到镜头，层层约束，跨维度一致性 |
| 克制美学优先 | "不做什么"比"做什么"更能精确控制AI输出 |

## AVOID

- 文字内容中混入样式参数（font:、size:）
- 不经用户确认就生成带文字内容的图（除非用户明确要求）
- 同一系列插画风格飘移
- 使用"我有个朋友"、"众所周知"等AI痕迹表达
- 过度复杂化——一个清晰的画面描述胜过一堆华丽的术语堆砌
- 维度填充敷衍——每个维度须自问：精确否？量化否？正反约束否？
- 堆砌质量词忽略主体——"8K, masterpiece"不能替代精确的微表情和环境描述
