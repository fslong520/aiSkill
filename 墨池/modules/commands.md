## 命令处理流程

### `/墨池 索引`

**输出知识库所有知识点列表**

```
1. read_file knowledge_index.txt
2. AI 提取所有知识点名称和分类
3. 格式化输出
```

**输出格式**：

```markdown
# 知识库索引

共 10 个知识点

## 按分类

### 算法数据结构 (3)
- 差分（熟练，学习 3 次）
- 前缀和（理解，学习 2 次）
- 树状数组（了解，学习 1 次）

### 工具使用 (2)
- ClawHub技能发布（了解，学习 1 次）
- ...

### 信竞教学 (3)
- ...
```

### `/墨池 搜索 <关键词>`

**搜索匹配的知识点**

```
1. read_file knowledge_index.txt
2. AI 搜索：
   - 名称包含关键词
   - tags 包含关键词
   - summary 包含关键词
3. 输出匹配列表
```

**输出格式**：

```markdown
# 搜索结果："前缀"

找到 3 个知识点：

1. **前缀和** (算法数据结构)
   - 摘要：预处理技巧，O(n)预处理后可O(1)查询任意区间和
   - 掌握程度：理解
   - 学习次数：2

2. **差分** (算法数据结构)
   - 标签包含：前缀和
   - 摘要：前缀和的逆运算...

3. **树状数组** (算法数据结构)
   - 标签包含：前缀和
   - ...
```

### `/墨池 统计`

**学习统计报告**

```
1. read_file knowledge_index.txt
2. read_file learning_log.txt
3. read_file user_profile.txt
4. AI 统计分析：
   - 知识点总数
   - 各分类数量
   - 学习次数分布
   - 掌握程度分布
   - 近期学习记录
5. 输出报告
```

**输出格式**：

```markdown
# 学习统计报告

## 知识库概览

| 指标 | 数值 |
|------|------|
| 知识点总数 | 10 |
| 新鲜知识 | 5 |
| 老化知识 | 2 |
| 需验证知识 | 1 |

## 掌握程度分布

| 程度 | 数量 |
|------|------|
| 了解 | 4 |
| 理解 | 3 |
| 熟练 | 2 |
| 精通 | 1 |

## 近期学习记录

最近 7 天学习 5 次：
- 2026-04-14 Hermes Agent
- 2026-04-13 差分
- ...
```

### `/墨池 复习`

**智能复习推荐（基于艾宾浩斯遗忘曲线）**

```
1. read_file knowledge_index.txt
2. AI 分析：
   - mastery 较低的知识点（了解/理解）
   - 最近未复习的知识点
   - 根据遗忘曲线推荐复习时间
3. 输出推荐列表
```

**输出格式**：

```markdown
# 复习推荐

根据遗忘曲线，建议复习以下知识点：

## 今天复习

- **树状数组**（了解，5 天前学习）⏰ 已过遗忘周期

## 本周复习

- **前缀和**（理解，10 天前学习）⏰ 建议巩固

## 提前关注

- **差分**（熟练，15 天前学习）⏰ 状态良好，可延后复习
```

### `/墨池 推荐`

**个性化学习推荐（基于用户画像）**

```
1. read_file user_profile.txt
2. read_file knowledge_index.txt
3. AI 分析：
   - 用户身份推断（如：信奥教练）
   - 已学习知识点分布
   - 推荐相关但未学习的知识点
4. 输出推荐
```

**输出格式**：

```markdown
# 学习推荐

根据您的画像（信奥竞赛教练），推荐学习：

## 相关扩展

- **线段树**（与已学"差分"相关）
- **树链剖分**（与已学"树状数组"相关）

## 进阶挑战

- **可持久化线段树**（适合教练级别）
- **树套树**（竞赛高级内容）
```

### `/墨池 导出`

**导出知识库为 Markdown 文件**

```
1. read_file knowledge_index.txt
2. read_file learning_log.txt
3. read_file user_profile.txt
4. AI 整合为完整 Markdown 文档
5. write_file 保存到 ~/桌面/copaw/墨池导出_日期.md
```
