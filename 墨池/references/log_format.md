# 学习日志格式规范

> 文件路径：`db/learning_log.txt`

## 格式说明

简单行格式，用 `|` 分隔：

```
YYYY-MM-DD HH:MM | 知识点名称 | 事件类型 | 触发来源
```

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 时间 | 事件发生时间 | `2026-03-29 01:28` |
| 知识点 | 涉及的知识点名称 | `差分` |
| 事件类型 | 新增/复习/验证通过/存档/删除 | `新增` |
| 触发来源 | 对话学习/用户提问/过期验证 | `对话学习` |

## 事件类型枚举

| 事件类型 | 说明 |
|---------|------|
| `新增` | 新知识点创建 |
| `复习` | 已有知识点再次学习 |
| `验证通过` | 过期验证成功 |
| `验证失败` | 过期验证失败 |
| `存档` | 标记为存档 |
| `删除` | 存档过期删除 |
| `标记永不过期` | 用户标记为基础理论 |

## 示例

```
2026-03-29 01:28 | 信息学竞赛数学学习路径 | 新增 | 对话学习
2026-03-29 01:30 | 差分 | 复习 | 用户提问
2026-03-29 02:15 | 快速排序 | 新增 | 算法讲解
2026-04-14 10:00 | ClawHub技能发布 | 验证通过 | 过期验证
2026-04-15 09:00 | 测试知识点 | 删除 | 存档过期
```
