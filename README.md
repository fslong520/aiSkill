## 📋 项目概览

这是一个精心打造的AI技能集合，包含了8个针对不同场景优化的专业技能模块。每个技能都经过深度定制，旨在解决特定领域的工作痛点，提升日常工作效率。

### 🎯 核心理念
- **专业化分工**：每个技能专注解决特定类型的问题
- **开箱即用**：无需复杂配置，直接调用即可使用
- **持续进化**：基于用户反馈不断优化和迭代
- **本土化适配**：深度适配中文使用场景和表达习惯

## 🚀 技能目录

### 1. 🧠 [Humanizer](./Humanizer/SKILL.md) - AI写作痕迹去除专家
**核心功能**：识别并去除AI生成文本的人工痕迹，让内容更自然流畅

<details>
<summary>查看详情</summary>

- 🎯 **双重模式**：评估模式（仅分析）+ 改写模式（分析+优化）
- 📊 **24种AI痕迹识别**：过度强调、模糊归因、填充词等
- 🔧 **文件处理**：支持直接修改文件内容
- 📈 **质量评分**：7维度量化评估文本自然度
- 🎨 **个性化优化**：根据内容类型调整优化策略

**适用场景**：
- 学术论文润色
- 商业文案优化
- 技术文档改进
- 社交媒体内容优化
</details>

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Node.js 16+ (部分技能需要)
- 相关API密钥（如微信、图像生成等）

### 安装依赖
```bash
# 进入各技能目录安装依赖
cd news && pip install -r requirements.txt
cd ../stocktourch && pip install -r requirements.txt
# 其他技能根据需要安装相应依赖
```

### 基本使用
每个技能都有独立的使用说明，详见各目录下的SKILL.md文件。

## 📚 使用指南

### 选择合适的技能
根据你的具体需求选择对应的技能：

| 需求场景 | 推荐技能 |
|---------|---------|
| 内容创作 | 公众号写手、财经分析师 |
| 文档处理 | md2wechat、Humanizer |
| 数据分析 | news、stocktourch |
| 图像设计 | 图片姬 |
| 编程教育 | 搬题姬 |

### 最佳实践
1. **组合使用**：多个技能可以协同工作，如先用公众号写手创作，再用md2wechat发布
2. **参数调优**：根据具体场景调整各技能的参数设置
3. **反馈优化**：积极提供使用反馈，帮助技能持续改进
