## Step 1：创建讲义/PPT

使用模板 [assets/讲义ppt模板.md](../assets/讲义ppt模板.md)

### Marp PPT 标准格式

```markdown
---
marp: true
size: 16:9
theme: zg_green
paginate: true
header: "**C++ 超神之路 · 模块名**"
footer: "**智国学堂** [章节 1](#3) [章节 2](#15) [作业](#40)"
transition: dissolve
---

<!-- _class: cover_e -->
<!-- _header: "" -->
<!-- _footer: "![h:140](../../imgs/zhiguo.png)" -->
<!-- _paginate: "" -->

# 标题
```

### 讲义结构

| 部分 | 时间 | 内容 |
|------|------|------|
| 回顾 | 5 分钟 | 上节课回顾，含互动问题 |
| 导入 | 10 分钟 | 新主题介绍，含互动问题 |
| 模块 1 | 25-30 分钟 | 主要内容讲解 |
| 模块 2 | 25-30 分钟 | 进阶内容 |
| 重点词汇和作业 | 10 分钟 | 总结 + 作业布置 |
| 作业答案 | - | 参考答案 |

## Step 3：编写代码示例

### OI 代码规范

- 左大括号 `{` 在单独一行
- 变量名简短（不超过 5 字符）
- 优先使用全局变量
- **不提供完整代码**，只给关键步骤或留空让学生填

### 代码质量要求

- 详细注释（每行关键代码都有注释）
- 时间/空间复杂度分析
- 易错点提醒
- 变量名 OI 风格

## Step 4：设计分层作业

| 层级 | 题量 | 难度 | 目标 |
|------|------|------|------|
| 基础 | 3 道 | 课堂知识直接应用 | 巩固基本概念 |
| 提高 | 3 道 | 需要综合运用 | 培养解题思维 |
| 挑战 | 2 道 | 超出课堂范围 | 冲刺竞赛 |

**注意**：作业题不与课堂例题重复。

## 语言风格

### 目标受众

14 岁中学生，调整内容匹配认知发展和兴趣。

### 风格要求

| 要求 | 说明 |
|------|------|
| 机智幽默 | 用引人入胜、轻松的语言 |
| 适合年龄 | 不幼稚，也不过于学术化 |
| 贴近生活 | 游戏、社交媒体、日常生活 |
| 清晰但不简化 | 用精确术语，但可理解地解释 |
| 对话式语调 | 像和聪明青少年交谈 |

### 常用类比

| 概念 | 类比 |
|------|------|
| 变量 | 贴标签的盒子 |
| 函数 | 有输入输出的专用工具 |
| 循环 | 游戏中的重复动作 |
| 数组 | 编号的储物柜行 |
| 指针 | 信封上的地址 |
| 递归 | 俄罗斯套娃 |
| 栈 | 盘子堆（后进先出） |
| 队列 | 售票处排队（先进先出） |
| 二分查找 | 猜 1-100 数字，每次减半 |
| 贪心算法 | 自助餐上总拿最大的那片披萨 |
| 动态规划 | 记住数学题答案，不重新计算 |

## 去 AI 味规范

详细原则和示例参见 → [去 AI 味指南](./去AI味指南.md)

**开发阶段必须逐项检查**：
1. 删除填充短语
2. 打破公式结构
3. 变化节奏
4. 信任读者
5. 删除 AI 高频词
6. 去除机械强调
7. 增加个性
8. 具体而非模糊
9. 避免空洞结论
10. 自然表达

生成 HTML 讲义时，CDN 链接优先使用国内可访问的源：

| 优先级 | 源 | 示例 |
|--------|-----|------|
| 首选 | unpkg.com | `https://unpkg.com/highlight.js@11.9.0/` |
| 备选 | fastly.jsdelivr.net | `https://fastly.jsdelivr.net/npm/` |
| 备选 | cdnjs.cloudflare.com | `https://cdnjs.cloudflare.com/ajax/libs/` |
| 避免 | cdn.jsdelivr.net | 国内访问不了 |
| 避免 | bootcdn | 太慢 |

## 讲义文件处理规则

更新讲义数据库时（`update_item.py`）：

1. **HTML 优先**：同时存在 `.html` 和 `.md`，只添加 HTML
2. **MD 过滤**：`.md` 文件必须以 `_ppt` 结尾才会被添加
   - `循环结构_ppt.md` ✅
   - `循环结构.md` ❌
3. **支持类型**：`.pdf`、`.html`、`.md`（需符合上述规则）
