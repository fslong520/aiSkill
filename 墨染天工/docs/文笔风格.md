# 文笔风格指南

整体定位：**"现代白话 + 历史质感 + 细节沉浸"**

参考风格：《明朝那些事儿》的易读性 + 马伯庸的细节质感 + 《三体》的技术震撼

---

## 一、叙述语言：文白交融

**原则**：白话叙事为主，关键描写带古典韵味

| 风格 | 示例 | 评价 |
|------|------|------|
| 过于文言 | "汴河之上，舳舻相接，帆樯如林，商贾云集，市井喧嚣，昼夜不息。" | 中小学生阅读困难 |
| 过于白话 | "汴河上船很多，人来人往特别热闹。" | 缺乏历史质感 |
| **文白交融（推荐）** | "汴河水运繁忙，大小船只首尾相接，从码头一直排到河心。商贩们扯着嗓子叫卖，船工的号子声此起彼伏。" | 易读且有画面感 |

---

## 二、对话风格：因人而异

**古人说话**：半文半白，符合身份，有宋代口语特色
**主角说话**：现代白话，但要"包装"，避免暴露穿越者身份

```
【场景】主角想表达"需要实验验证"

❌ 主角："我们需要做对照实验"（太现代）
❌ 主角："此事当以实事求之"（太文言）

✅ 主角："这事儿光想没用，得试过才知道。"
✅ 古人（书生）："兄台所言甚是，凡事当以实证之。"
```

---

## 三、描写风格：细节沉浸

**原则**：用具体细节展现古人智慧，而非抽象概括

| 写法 | 示例 | 效果 |
|------|------|------|
| 抽象概括 | "水运仪象台很精密，令人叹为观止。" | 读者无感 |
| **细节沉浸** | "他凑近了看，那些齿轮竟比指甲盖还小，密密麻麻咬合在一起。水流推动叶轮，叶轮带动齿轮，齿轮牵引着浑仪缓缓转动——一刻不停，分毫不差。" | 读者震撼 |

---

## 四、技术描写：硬核+易懂

**原则**：技术要有逻辑（满足历史爱好者），但要易懂（照顾中小学生）

**技巧**：
1. 用类比解释原理
2. 用主角的困惑引导读者理解
3. 关键原理讲清楚，细节可略

```
【描写算筹计算】

主角看呆了。

桌上那些小竹片，在博士手里像是活了一样。红的代表正数，黑的代表负数，摆成几列，手指一拨，就得出了答案。

"这是……解方程？"

"天元术。"博士头也不抬，"你要解的这道题，用《九章算术》里的方程术，要算小半个时辰。用天元术，一盏茶功夫。"

主角心里翻起惊涛骇浪。这哪里是"古代数学"，这分明是——代数。
```

---

## 五、节奏把控：中速稳健

| 场景类型 | 节奏 | 篇幅 |
|---------|------|------|
| **日常探索** | 慢，铺陈细节 | 可长 |
| **技术发现** | 中，详细描写智慧点 | 适中 |
| **危机冲突** | 快，紧凑有力 | 简短 |
| **情感转折** | 慢，沉淀情绪 | 适中 |

---

## 语言禁忌

| 忌讳 | 示例 | 修改建议 |
|------|------|---------|
| 现代术语直接使用 | "这个效率太低了" | "这样太费工夫了" |
| 网络用语 | "这波操作666" | "这手段当真高明" |
| 过于黑暗沉重 | 详细描写酷刑、屠杀 | 淡化处理，侧面描写 |
| 说教感 | "我们要学习古人的智慧……" | 用故事让读者自己感受 |
| 历史硬伤 | 宋代出现玉米、土豆 | 严格考据，不符者不用 |

---

## 特殊处理

### 名词注解

首次出现专业名词时，用括号或正文自然解释：

```
✅ "他走进翰林天文院，一眼就看到了那座水运仪象台——那是一座三层楼高的天文钟，既能观测天象，又能报时。"

✅ "博士用的是天元术，一种用符号代未知数的方法——在西方，这要等到几百年后才出现。"
```

### 知识点嵌入

每章嵌入1-2个知识点，让读者"顺便学到"：

```
【第X章片段】

主角跟着老工匠走进工坊，里面热浪滚滚。

"这是炒钢。"老工匠指着炉子说，"生铁太脆，熟铁太软。把生铁炒一炒，碳就少了，出来的钢正好。"

主角愣住了。炒钢？这不是——

"这法子，咱们用了快一千年了。"老工匠很平淡。

主角心里却在咆哮：一千年？西方要到18世纪才有类似的工艺！

（知识点：中国古代炒钢技术，领先西方近千年）
```
