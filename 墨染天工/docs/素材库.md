# 素材库

## 古代科技成就（可引用）

### 数学
- **天元术**：元代李冶，用符号代未知数，相当于代数学
- **四元术**：元代朱世杰，可解四元高次方程组
- **勾股定理**：《周髀算经》，比毕达哥拉斯早500年
- **圆周率**：祖冲之算到小数点后7位，领先世界近千年

### 天文
- **水运仪象台**：苏颂，世界最早的天文钟
- **浑天仪**：张衡，自动运转的天球仪
- **圭表测影**：精确测定节气
- **授时历**：郭守敬，精度与现代公历相当

### 机械
- **指南车**：机械齿轮实现，非磁铁
- **记里鼓车**：机械计程器
- **地动仪**：张衡，世界最早地震仪
- **水排**：水力鼓风，冶金用

### 冶金
- **炒钢法**：西汉，领先西方近千年
- **灌钢法**：生熟铁合炼，优质钢材
- **铸铁技术**：比西方早1500年

### 医学
- **针灸**：经络理论，至今有效
- **麻沸散**：华佗，世界最早麻醉剂
- **人痘接种**：宋代，比牛痘早几百年

### 水利
- **都江堰**：李冰，自动分水排沙
- **郑国渠**：灌溉关中平原
- **大运河**：隋代，世界最长人工河

## 宋代特色

### 城市生活
- **汴京**：人口超百万，世界最大城市
- **夜市**：不宵禁，商业繁荣
- **瓦子**：娱乐场所，说书、杂技、戏曲
- **茶坊酒肆**：遍布街巷

### 科技水平
- **活字印刷**：毕昇
- **指南针**：用于航海
- **火药武器**：突火枪、霹雳炮
- **水力机械**：广泛使用

### 社会风貌
- **士大夫政治**：文官治国
- **科举制度**：寒门可入仕
- **商业发达**：交子（纸币）流通
- **文化繁荣**：宋词、山水画

### 口语习惯
- 称呼：兄台、小可、足下
- 谦辞：在下、鄙人
- 敬辞：阁下、尊驾
- 语气：甚是、当真、岂非

## 使用方式

1. 每章嵌入1-2个知识点
2. 用具体细节展现智慧，不用抽象概括
3. 通过主角的震撼传递读者的震撼
4. 避免说教，让读者自己感受
